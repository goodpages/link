1.军婚宠文创作指导

人设：一般这类型的文都为强强联合
（1） 女主：御姐风范，长相或可爱或妖娆，各类型皆可，但最主要的是心性坚强，有做女强人的信念，在各领域发光发热，有吸引各男主男配的闪光点
（2） 男主：军人身份，军衔职务很高，多为特种大队大队长中队长抑或中将少将，号令军政商三界，性格矛盾，有反差（比如表面温润实则残酷；表面风流实则纯情；表面冷酷实则温柔，温柔只针对女主……）
（3） 女配：因为多为宠文，个人阅读喜好也不喜欢男主曾经有爱的女人，偏向从一而终，所以女配一般是自己倒贴为主，但即使这样，也不能全部都为炮灰，需要一两个贯穿全文坏到极致的女主，从而达到离间男女主感情，制造坎坷的作用
（4） 男配：和男主一样，不一定要是军人身份且最好不为军人身份，和女主有比较多接触的机会，比如，女主从商，男配在商界呼风唤雨，男配的本领以及设定不能弱势，甚至和男主不相上下，是男主最重要的情敌，贯穿全文，也是制造坎坷作用，但此人不为炮灰，结局可以是安排好姻缘可以是默默陪伴女主

家世：依旧是强强联合
（1） 男主：军政世家，红N代，在国内是横着走的类型。两个方向：男主与家里关系好，家里喜欢女主，不构成两人间的阻碍，甚至在两人感情受挫时会撮合男女主；男主和家人相处不好，家里也不喜女主，从而导致各种阻碍横生，这种情况，男主势必是宁愿脱离家族也维护女主的
（2） 女主：三种可能。①女主家庭也是军政家庭，与男主家里关系融洽，双方履行婚约，男主陪着女主一起打天下，此类文温馨基调为主；②女主家庭没落，受尽欺辱，女主变强，使家族越来越强大，受到尊重，主要在于女主驰骋在自己的圈子，男主默默予以支持帮助，此类励志女强为主；③孤儿，其实是军政家族抛弃的子嗣，逐步自己在自己的领域慢慢变强，最终认祖归宗的故事，家族可以是欢迎女主，可以是不认同女主，此类励志女强为主。
（3） 男主：军政商无所不能
（4） 女主：设定在商界、军界、娱乐圈、政界某一领域逐步提升自己，若是驾驭不了，选择其一，能够驾驭的话，可以多项选择

情节设定（一般为励志女强）
（1） 男女主角的情感发展以及女主逐步变强这两条是主线，也是不能分割的，要衔接在一起，连贯的贯穿于全文
（2） 开篇节奏要快，不能拖沓慢热，没必要的人物情节全部去除，必要的背景几句话带过，切忌太多叙述性的内容，最好用事件直接阐述想要表达的情感，开篇稍带暧昧，整体尺度不要过大，男主开篇前几章就能够出场，开篇3w情节以男女主角情感碰撞为主，但不能水，需要代入相关支线伏笔

写作技巧
（1） 书名简介章节名迎合市场需求，稍带暧昧，能够吸引点击，即使是标题党，作品最需要的就是能够有让人阅读的欲望，否则正文内容再好，也很难受到关注
（2） 1-20章1200字+，21章开始3200字+，每个章节卡点精准，能够有让人继续阅读下一章的欲望，尤其是20章，因为无线21章开始计费，若20章读者难以沉迷，21章是断不可能付费阅读的了
（3） 不要过于注重华丽的辞藻，无线文通俗易懂就可以，情节比任何都重要，所以要在情节方面多下功夫，最快速的方法就是要写这一类文的时候，去把无线榜单上关于这类型的文全部浏览阅读一遍，掌握此类文的套路以及基本规律，尤其在以上所说的几点上尤其要注意，看看别人是如何写的

《军婚撩人》
人设：
女主为小警花，受过训练本身技能不弱，后被男主带到军队，从而成为男主麾下的女特种兵，因为男主是军人，军人服从命令是天职，男女主在一起的时间就是个问题，这篇文把女主带到军队，便解决了这种问题。
男主为皇朝太子爷，表面纨绔实则冷酷危险，这就是人物设定的矛盾点。
女配是男主曾经爱的女人，死去七年却又出现，并成为男主敌对势力的诱饵
男配一为男主敌对势力的头，长相漂亮，能力和男主不相上下；男配二为女主曾经爱过的男人，两个男配都能对男女主角的情感动向起到一定的作用

家世：男主的爷爷对男主为了女主做出种种离谱事件不满，所以是反对的，男主一直是维护着女主，以女主为第一考量
情节设定：
主要的场景在军队，全篇情节围绕男女情感展开，暧昧适中，全文基调轻松，语句诙谐，和现实军队的气场比较合，军人说话以及各种表达方式和一般的文还是很不同的，这篇文算是把握比较不错的
这篇文其实暧昧向为主，虽然尺度不大，但大部分打的擦边仗，书名和内容都是，男主的设定就是碰到女主就欲罢不能的类型，所以这种暧昧戏比较多，情感的代入感比较强
欠缺的部分就是暧昧向为主，情节看点不够，这也是这篇文没有《军婚，染上惹火甜妻》受欢迎的主要原因，这篇文主线就是男女情感为主，男女主主要的场地在军队，而且已经把女主塑造成了特种兵，却没有继续延展开来，没有让女主在这一身份上发光发热，所以导致看点不够，暧昧过多也就腻味了。
