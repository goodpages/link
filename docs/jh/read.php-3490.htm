2012，不可怠慢企业级移动应用

当今世界正在成为一个移动化的社会。无论是在办公室，还是地铁上，都能看到盯着手机或平板电脑，不停滑动手指点击按键的人，几乎工作和生活的所有场景都在被智能手机和平板电脑这些新一代移动终端所占据。汹涌而来的移动互联浪潮，不仅让个人消费级应用市场炙手可热，也促进了企业级移动应用由点到面的不断蔓延。可以预见，2012年，移动应用将会出现在更多的企业中，移动应用也将成为企业战略投资的主要领域之一。

　　驱动因素的升级

　　企业级移动应用不是一种全新的趋势。许多行业和企业在几年前就开始试水。快消品行业就是如此。由于业务覆盖范围广泛分散，为了实现对终端的有效管理，两三年前快消品企业就为基层业务人员配备移动终端，通过移动终端获取每天销售数量和工作日志等信息，以便更好地进行人员和终端市场的管理。烟草行业也很早就开始使用手机查询卷烟营销结果，利用GPS/GIS定位监控物流配送车辆。然而，2010年以来，企业级移动应用又有了新的突破和增长。

　　之所以出现这种突破性增长，是因为驱动移动应用的因素正在升级。

　　第一，3G技术被广泛采用，互联网可以随时随地连接，为信息移动化提供了技术基础。

　　第二，以苹果iphone和ipad为代表的智能手机和平板电脑等终端产品越来越普及，培育了人们对移动终端的使用习惯。这种习惯逐渐从个人消费应用延伸到企业级的业务管理应用。移动应用不受时间、地点的限制，员工不必被束缚在办公桌之前，可以随时随地通过移动终端展开工作，从而更快速地响应企业的业务要求和客户的需求，工作效率得以大大提升。另外，借助移动终端，企业也可以尽快掌握信息，加快各流程环节的信息反馈，保障快速决策，提高管理效率。

　　第三，云计算概念和技术的日渐成熟，则将从根本上清除移动应用的障碍。云计算强调分布计算，将任务处理方式由传统的桌面核心转变为网络核心。因此，移动终端相较于PC在传统环境下的种种短板在云计算环境下都将不复存在。

　　选好切入点

　　越来越多的企业开始认识到移动应用带来的价值，CIO们对移动应用也表示出了浓厚的兴趣。在考虑信息化建设时，开始拨出专门的预算用于实施移动应用。如果说之前的移动应用是小打小闹的话，那么从去年开始则与业务和管理紧密相关。

　　企业应用从PC向移动终端的延伸要考虑数据和各层级业务的问题，部署时要充分考虑移动平台的可扩展性和融合性。因此，找准移动应用的切入点是保证应用成功的关键。

　　移动办公和流程审批是一些企业选择的切入点。因为这个环节最容易得到业务部门和领导层的支持，移动化应用能够很明显地提高工作效率。移动BI也是较容易得到领导层认可的应用。领导层可以随时随地通过移动BI了解企业运营情况，为及时决策提供条件。

　　北京烟草今年就首先研发了新的移动办公系统。这个系统是按照云计算的技术思路搭建的，它不是通过登录网络进入系统，而是通过桌面虚拟化，将桌面上的所有应用都拖拽到平板电脑上面。办公、流程审批、财务、监督监管的移动应用都通过虚拟化的方式得以实现。这种移动应用与操作系统和终端pad无关，无须针对不同的终端和操作系统开发不同的移动版本，在安卓、苹果和windows系统上都可以顺畅应用。

　　销售渠道的移动化管理是另一些企业的切入点。农夫山泉就率先实现了销售终端的移动化。农夫山泉特别需要解决如何与企业信息网络之外的下游客户保持互通的问题。为此，企业开发了移动商务系统，为终端销售人员配备移动终端，销售人员可以第一时间将各种信息通过手机短信传送到后台。其中GPRS功能也很好地实现了对一线业务人员销售行为的管理。

　　另外，一些企业甚至利用移动技术实现了业务的非现场即时管理。天津港就是一例。它们从RFID物联网技术切入，对RFID技术进行了大量研发，建设了智能化无人闸口，提升了集装箱码头的作业效率。

　　业务管理正在多方位、深层次移动化

　　随着外部驱动因素的升级以及内部业务需求的涌现，在企业中移动应用正在由点向面、向纵深方向扩展。本期专题中采访到的企业就正在将移动应用进行深入部署。北京烟草陆续在专卖打假、移动终端订货、物流等各个业务环节上做了规划和实践尝试。天津港也将结合GPS、RFID技术，打造全程物流跟踪，深化物联网应用。

　　这些业务管理层面的更深层次应用，正是架构在已有的移动应用基础上的。实际上，就像一位长期关注移动互联网的专家所说，未来企业移动应用将无处不在，无论是企业经营、内部管理和战略决策，还是外部对生态链的打造，都将会产生移动需求和应用。当然，这样的应用是建立在点状应用基础上逐步深化的。

　　所有的迹象都表明，企业点状移动应用正在形成面，因此，2012年，企业的移动应用将不能被怠慢。
