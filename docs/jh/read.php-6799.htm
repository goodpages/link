第1页独立游戏生存现状:在巨头狭缝中侧翼突围

　　在中国独立游戏的发展历程中，2015是不同寻常的一年。巨头集中发力抬高研发、发行门槛，行业寡头化竞争趋势基本形成。在这种左右夹击的狭缝中，竟有数款独立游戏成功实现侧翼突围。

　　一款名叫《球球大作战》的独立游戏，在不到两个月时间里，靠口碑传播实现百万用户增长、DAU(日活跃用户)逾30万，获苹果APP Store全球推荐。无独有偶，另一款名为《鲤》的小众游戏也迅速积攒口碑，并跻身旧金山GDC IMGA最受期待游戏作品行列。在竞争已白热化的手游市场，这些独立游戏横空杀出，有如酷热夏天中不期而至的一股凉风。它们带给行业的除了沁人心脾，还有值得思考的深层原因。

　　独立游戏的“前世今生”

　　所谓独立游戏，是相对于商业游戏制作而存在的另一种游戏制作行为。这与独立电影与商业电影的概念非常相近。在没有商业资金的影响或者不以商业发行为目的，独立完成制作，都可认为是独立游戏制作行为。

　　在国内，大家普遍认为自筹资金，拥有较少的人才和宣传等资源的团队所制作的游戏，称为独立游戏。而在国外，独立游戏则被认定为在没有商业资金的影响下，或者不以商业发行为目的独立完成的游戏制作。但是如今，独立游戏制作和商业游戏制作之间的界限，也随着游戏产业的发展，一天天变得模糊起来。

　　中国第一位独立制作者已无从可考。在互联网普及之前，即使成功完成游戏作品，也缺乏途径将之传播给大众。许多独立游戏制作者淡漠在人们的视野中。随着门户网站开始兴起，开始了大众网络时代，独立游戏才得以广泛传播。

　　2015年，随着一批国产独立游戏脱引而出，登上世界舞台。独立游戏的生存现状，开始备受瞩目。作者调查发现，一款独立游戏的脱颖而出，离不开以下几点深层因素。

　　第一：团队小而灵活、唯快不破

　　《球球大作战》从诞生到发展，充分见证了“一颗球的简单胜利法则”。小团队、小制作，比大公司有着更快、更灵活的反应速度，这是所有独立游戏与大公司相比所具备的独特优势。该游戏第一个版本，像所有成功的互联网产品一样，非常聚焦非常轻，仅靠4个人团队用10天时间打造;但在第一版推向市场后，通过玩家反馈，该团队日夜赶工，不断迭代、不断修正，持续完善玩法与版本。曾有一度，《球球大作战》在一个月内迭代了4个版本。扁平化的团队工作模式，使得这类独立游戏有了更快的反应速度、更灵活的空间、更贴近用户的需求，这是当下的游戏巨头无法实现的优势。

　　第二：快速上手的低门槛玩法

　　从《黏黏世界》到《FlappyBird》，再到今年的《球球大作战》、《鲤》，近年来火爆的独立游戏都有一个共性，操作简单、老少兼宜、能够快速上手。《FlappyBird》只需要不停的戳屏幕，《黏黏世界》只要把这些黏豆包组合在一起就可以通关，《球球大作战》只要不断吃下去就能赢得比赛。没有门槛的玩法，能让独立游戏迅速聚集起一批基础玩家。然而近年来，游戏巨头将重心放在开发重度手游上。2014年下半年以来，随着传统端游厂商加大手游推出力度，中重度手游开始充斥市场。重度游戏的到来提高了用户门槛，但这无形中给了独立游戏开发者留出了生存机会，形成新的蓝海。善于抓住机会，有独特创新玩法的独立团队，往往能利用简单易操作的规则切入产品，实现快速吸引用户目的。

　　第三：简单制作的背后有深意

　　复杂的规则，往往会限制受众的想象空间;规则简单，反而能给用户更多自由发挥的余地。有玩家曾这么评价过《球球大作战》，“一万个人眼中有一万个哈姆雷特，一个球球里居然会有无数种死法。”而知名艺人李晨也曾早早在微博中，用一连串的叹号，预言这款游戏的前景：“这是一款马上就要火的游戏!简单至极!简单到令人发指!却引人入胜!发人深思!小球靠吞噬变大!小时移动迅速，大时行动缓慢!可通过拆分个体寻求快速!也可通过合并完成强大!如贪欲太重会被瞬秒!天地万物不离其中!竟被一个游戏解释清楚!”

　　第四：不急于赚钱催生口碑传播

　　在游戏行业中，大部分的独立开发者都会面临着一样的问题：没有钱，我该如何推广游戏？特别是2015年，手游市场表现出的都是砸钱拼市场的死磕态势，市场发行、推广规模迅速水涨船高。在渠道集中、产品集中、IP集中的如今，独立游戏的生存空间似乎越来越小。

　　《球球大作战》、《鲤》的爆红，给了独立制作人希望。这些优秀的独立游戏前期都是靠口口相传累积起的用户，通过玩家的二次传播达到几何级的传播，不依靠渠道自然吸引百万、甚至千万的用户。这听起来是意料之外，其实也在情理之中。以《鲤》为例，这款独立大作已经入围了2015年旧金山GDC IMGA最受期待游戏的作品行列，是国内唯一一款入围的游戏，并且还获得了苹果官方的商店首页推荐通知，说明该作还未经正式推广便获得了国内外的高度关注。

　　有游戏业者指出，独立游戏在制作过程中，不受商业化因素干扰，充分聚焦玩法、抓住游戏本质，是它们获得玩家青睐、催生口碑的重要原因。在游戏制作中，更多体现制作人的独立思考，传递个性化创意的过程，回归游戏本质。这种游戏的趣味性是灵动的，精神是独立的，玩法是极致的。像所有成功的互联网产品一样，在初期它们可能很小众的，但由于品质出众，后续生命力、口碑传播能力极强。

　　“好的游戏真的会走路。”有独立游戏开发者表示，“是金子总会发光的，走心的产品还是会被玩家认可的。”资深玩家Nic Chen则指出，在同质化、重度化、高付费化的手游市场中，独辟蹊径的独立游戏有如一股清风来袭，有摧枯拉朽威力。

　　独立游戏还能走多远？

　　作者常能够想象这样一副画面：一个形象邋遢的程序员，啃着方便面，目光呆滞的盯着苍白的屏幕。这是独立游戏制作人的一方缩影。有独立游戏人表示，做独立游戏的，绝大多数都是“普通青年”或“二逼青年”，有大学辍学者，有无业游民，有从游戏公司辞职自己创业的，“我们更平民，更屌丝，但也更真诚”。

　　上述业者坦言，他投身独立游戏制作人的初衷非常简单，因为喜欢游戏所以想做一款自己爱玩的游戏，带着诚意制作。“如果这个游戏能够带给我快乐，那我相信在某个地方，还会有玩家也因这款游戏得到快乐。”很多独立游戏人一开始都是玩家，也都曾为了非常个人化的目的开始做游戏。过往的游戏经历，就像数亿年前的树木一样，在潜入地层之后，在他们心中蕴酿着创造力，等待有朝一日能够井喷而出。

　　比起巨头们的大制作大推广，独立游戏正是凭借其“不一样”的独立风格赢得众多粉丝。独立游戏回归到内容本质，以独具匠心的创意、简约而不简单的画面、别出心裁的玩法为特点，带给玩家想象之外的无限惊喜。

　　对独立游戏人来说，聚焦内容、创意创新，是唯一出路，也是在时下复杂的手游时局中脱颖而出、侧翼突围的小秘道。
