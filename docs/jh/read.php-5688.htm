把360、美图秀秀等模式复制到国外 半年内用户达数百万

renjihe1988 , 13:50 , 收集 , 评论(0) , 引用(0) , 阅读(3) , Via 本站原创 大 | 中 | 小
引用功能被关闭了。
当我们在11月份介绍“四叶新媒体”时，有读者向我们指出，一个叫“Toolwiz”的团队通过将国内360、美图秀秀等产品模式复制到国外，做了好几百万的用户。于是出于职业好奇心，我联系到了这个新加坡的3人团队，并对他们进行了一个简单的远程采访。
 
团队
 
团队名称叫 Toolwiz （中文名魔法工具或者兔卫士），于去年9月份在新加坡成立，主要开发 Windows 平台上的免费软件。最开始团队只有3个人，团队负责人 James 之前多年在欧洲的安全公司开发安全软件，擅长于内核应用开发；另外两名成员是一对双胞胎，患有天生的血友病，也因身体的原因没怎么上过学，在家自学成才，擅长于图形图像领域开发。
 
成绩
 
Toolwiz Care 的3人团队在近半年时间发布了12款 Windows 平台上的软件产品，用他们自己的话说，其核心产品一个类似国内的360安全卫士，另一个类似美图秀秀，他们称为“Toolwiz Care”和“Toolwiz Pretty Photo”。其中 Toolwiz Care 在国外的下载量已达数百万。目前他们所有的产品中，有20%的用户来自俄罗斯，10%来自南美，10%来自印尼和越南。美国与法德意西各在30%左右，而来自中国的用户只有1%。如果仅算 Toolwiz Care 这款产品，50%的用户来自美国。
 
模式
 
他们所有产品均采取纯免费模式（没有广告、没有增值收费服务、无捆绑），这种免费模式在以收费为主导的欧美一些市场引发了一些争议，不过在他们长时间的和用户接触后，用户也慢慢接受了这种模式。团队告诉我说，在这个过程中，如果收到用户邮件抱怨某个功能，他们会在3天内给出一个新版本。而在东南亚、南美以及东欧国家，免费文化则很容易接受。
 
在这些争议中，Toolwiz 集结了来自世界各地的很多志愿者，免费为软件翻译到不同的语言，推广到不同的国家。一些志愿者正式加入团队，使得现在的核心成员已有10多人，加上志愿者则有40多人，这些志愿者分布在世界各地，形成一种他们称为“自服务”的模式。这些遍布全球的志愿者，为他们提供了各种信息和素材，也搜集了很多用户反馈。
 
由于软件都是纯免费的，因此一开始他们就做好了长期没有收入的准备，最早的启动资金是他们拼凑出来的，主要用于购买服务器空间和为用户提供基础性服务所需的一些东西，一开始基本上整个团队是零报酬在工作。但是现在，他们已经获得了来自新加坡政府的资金支持。团队告诉我说，事实上，现在找他们的投资人或者捐赠者都非常多，但是都被他们拒绝了，因为他们说只想做点有意义的事情。
 
为何选择 Windows 平台，而不是 iOS 或者 Android 平台
 
目前的很多开发者都将目光放在了苹果的 iOS 以及 Google 的 Android 等移动互联网平台，而 Toolwiz 则选择了传统的 Windows PC平台。James 说起因主要是他在欧洲工作时感受到欧洲安全领域的“步伐沉重”——忽略用户的需求，开发响应速度慢，以赢利为核心等，他想做一些不同的东西。
 
但经过研究，他们发现，在互联网的整体发展趋势上，国内（中国大陆）目前和美国基本上是同步的。国内在移动互联网上几乎不逊于美国的发展，但是在印尼等东南亚国家，很多地方还在普及功能机，而印度很多地方还在购买第一台PC，因此他们认为 Windows 领域其实仍有很多机会。而他们最大的优势在于是在新加坡做，新加坡辐射东南亚，在欧美也都有很好的口碑。
 
面对挑战：贴近用户，崇尚口碑，每周一版，邮件必回，小步快跑
 
在成立 Toolwiz 团队之前，他们对欧美的个人软件市场做了大量分析，认为系统清理优化加速和图像处理领域有很大的需求。特别是系统清理优化加速这个领域，市场有大量的Scareware（恐吓软件），这种软件在用户的计算机上扫描出一堆的虚假问题，然后诱导用户去购买。还有很多的虚假免费软件，他们把一些功能免费，然后号称是免费软件，同时又锁住一些核心功能，用免费软件的名义来哺市场，最后让用户为这些封锁的功能进行升级买单。
 
Toolwiz 则采取“产品免费+服务免费”完全免费的模式，一开始很多欧美用户不习惯，还有很多带着疑虑，甚至有些带着抵触情绪，因为对他们依靠什么生存表示怀疑，并且认为以后会慢慢收费，不过 Toolwiz 团队慢慢打消了这些疑虑。
 
团队告诉我说，从成立之日起，James 就坚持几个原则：贴近用户，崇尚口碑，每周一版，邮件必回，小步快跑。在每天的开发时间之外，James 会坚持回复每天来自世界各地的用户邮件，并且将用户的合理化建议迅速体现在新的版本中，也就是我上面说的3天内给出一个新版本。这个我自己深有体会，在我发邮件给 James 的当天，他就回复了我邮件。
 
技术才能长期立足
 
除了提供好的服务外，Toolwiz 团队认为只有技术才能长期在市场立足，在 Windows 系列软件中，他们在技术上做了一些工作：
 
时间静止功能让系统垃圾都在系统重新启动后自动消失
一般来说，Windows 用户认为系统使用时间长了就会有垃圾，进而需要清理。Toolwiz 的 Toolwiz Care 产品为此提供了“时间静止”功能，让用户的系统随时可以被固化下来，让系统垃圾都在系统重启后自动消失，这样用户就能天天用新系统，免去了清理垃圾的麻烦。
 
把系统休眠状态固化
Windows 的系统休眠（Hibernation）功能可以让用户从休眠状态迅速开机，Toolwiz 团队将系统固化功能延伸到休眠状态，把这个休眠状态固化下来，这让每台电脑都能在20秒启动。现在 Toolwiz 团队已经把这个产品（Toolwiz BHORM）开源，同时他们透露，这个产品已经被美国的机顶盒公司看中，被使用在数万台机顶盒中。
 
借助旁边电脑进行修复
一般来说，当我们的电脑中毒或者出问题时，基本都依靠自己的电脑杀毒软件来杀毒或者修复。Toolwiz 则对这个用户习惯进行了大胆的扩展，开发了 Toolwiz Remote Back 产品，当一台电脑受伤时，可以利用它旁边的电脑帮助这台电脑进行修复。团队告诉我说，目前这个功能已经在和包括DELL在内的一些公司进行合作谈判，未来将实现远程的系统救助和修复。
 
除此之外，当 Facebook 在今年收购 Instagram 时，他们很快在2个月的时间内开发出了类似美图秀秀的新产品Toolwiz Pretty Photo，同时为这款产品设计了近百个滤镜以及磨皮祛痘、皮肤增白、瘦身美体、超级拼图、智能补光等功能。
 
不过正如我在文章前面提到，国内有读者认为，Toolwiz 的这些产品和模式，主要是借鉴复制国内360和美图秀秀等产品，再通过在国外的推广而获得目前的成绩。但是不管怎样，我相信他们的一些做法和推广方式值得参考。
 
关于未来
 
谈到未来，Toolwiz 团队仍然希望能以非商业化的心来做产品，他们说目前的钱已经够他们的基本生活，不打算去拿投资人的钱。而圣诞节后，会发布几款 iOS 上的产品，这些产品将把公益结合到 LBS 中，他们认为移动互联网除了交友等功能外，还有更多和慈善、公益结合的机会，他们也希望通过这种方式践行自己的梦想。
 
在我们的聊天式采访过程中，他们一直强调，新加坡是一个义工非常普及的地方。他们也打算在目前的这条道路上一直走下去，不商业。
