李开复：5年内人工智能将长足发展 技术创业的春天快来了

未来的人工智能会欺骗人类吗?

“我本来是想回答‘不会’，但我的机器告诉我，应该回答‘会’。”在创投圈天使晚宴开始前，接受记者采访的创新工场CEO李开复开了这样一个玩笑。

“我觉得想太多了。科幻片看太多的同学们可以不用担心(机器人欺骗人类)，但需要担心的是，机器可能真的会取代我们的工作。”李开复说。

他预计，未来5年左右人工智能会有长足的应用爆发出来，技术领域创业者春天马上要来了，这是乐观的一面。不过，如果认为人工智能很快会成为一个平台“百花齐放”，其实为时尚早。

除了火热的人工智能外，李开复同时也对VR(虚拟现实)、O2O、网红经济等表达了自己的看法。

人工智能的真正威胁是什么？

为什么人工智能还没有到成为平台、百花齐放的时候?

李开复给出了自己的理由——虽然人工智能已经验证可以达到一定的高度，但是做一个应用仍然非常花时间。“要把这些算法应用到一个新的领域是需要巨大的数据量和计算量，还需要真的懂深度学习的人来深度参与项目，然后再花一两年的时间，才能做出一个潜在的解决方案。”

李开复举了个例子，自己曾在1988年发表了关于统计学方法做出的语音识别的博士论文，当时自以为语音识别差不多要突破了，但事实上一直到今天才开始有大部分人使用语音识别，“所以有些技术还是会比较长久的”。

他把人工智能划分为几层。一层是封闭领域，围棋就是典型的例子，在这个领域机器打败人类是必然的事情，而且人类永远不会再有机会了;另一层是开放式的领域，机器可以靠数据统计达到一定的效果，比如今天的语音识别、感知还有人脸识别等，最终机器会做得越来越好。

但在上述两种情况里，机器仍旧是冷冰冰的技术，即使它打败了世界冠军也没有感到高兴。在这两层之外，人类有一些更深层次的理解，例如讲双关语里的幽默，或是对一个人明褒暗讽、暗箭伤人，现在机器理解起来还是相当困难。

“还有一个问题是机器即便能猜到你会说什么，知道这句话代表了你要执行什么，但是还不知道是为什么。”李开复说，当你一旦进入了这种人的右脑思维，人的幽默、爱情、友情等等这些，机器还搞不懂，而它要花多长时间才能做到这点，目前还无法预测。

“现在看来，这些计算机都是非常听话的人类的忠诚仆人，没有自我意识，没有太多的野心，科幻片看多的不用担心。”李开复认为，太早担心还没有必要。

但人们需要担心的是，五年到十年之内，这些机器可能会取代人类的工作 。李开复认为，很快会有类似谷歌AlphaGo这样的人工智能来辅助人类或者取代人类的某一些专业功能，这些功能也许有关会计，也许是记者，或是互联网经济、房地产等等，这些领域慢慢都会被这一类的聪明的机器取代，不是一两年之内，也许在三五年之内就会取代。

“我觉得首先人类要度过的下一关，是大量的工作被机器取代之后，但是机器可以产生财富，那么这一批人会不会很不开心，或者做出一些不好的事情?未来怎么样用他们的脑子产生更多的创意，这些都是未知数，但是这个问题可能是5到10年会发生。但未来机器人用诡计把人类灭亡了，或者征服了，我觉得恐怕是下一个灾难了，先一个一个考虑吧!“李开复说。

技术创业者的春天

尽管2016年仍被不少人视作资本寒冬，但伴随着AlphaGo引发的人工智能热，李开复预计，至少在互联网技术领域，创业者的春天是可能存在的，“也许后半年有可能看到。”

他认为，在人工智能领域，中美之间的差距在三年左右。现在做深度学习的工程师在美国有可能一年拿到两百万美元以上年收入，这足以证明人才的稀缺性。相信清华北大的同学听到这句话，很多人会跳进去学这个技术，过两三年有成千上万的技术人才，而且未来人工智能一定会朝平台化方向发展。”李开复说。

事实上，不久前Facebook创始人扎克伯格在和马云的对话中也提到，他对中国特别乐观的一点就是，中国非常关注理工科教育。“未来可能在全球就业方面，更需要与技术相关的工作。”

和人工智能领域面临的情况类似，对于火热的VR(虚拟现实)、AR(增强现实)，李开复引用了比尔盖茨在科技领域的名言，“人们总是高估一两年内将要发生的变革,却低估未来10年将要发生的变革。”

李开复认为，如果再过五年，VR和AR很可能会颠覆人类绝大多数的应用场景，特别是会在娱乐、视频、游戏等现在是3D体验的领域进行颠覆，而AR则是教育、培训等，但就目前来看，短期的普及率都达不到这种程度，还无法解决用户眩晕、沉浸感等体验问题。

李开复也坦言，对于人工智能和VR等智能硬件领域，创新工场始终保持“长期乐观，短期谨慎，但还是投资”的态度。“巨大的人工智能革命还有一段时间，但是现在对用户有价值的公司已可以投资。”

下一个风口

移动互联网的下一个风口是什么?

李开复告诉记者，除了关注人工智能和智能硬件外，创新工场还专注在文化娱乐、互联网教育、O2O 和B2B等几大领域。“这些领域有些已经成为风口一段时间，有些才刚刚快要起来。”

其中，在文化娱乐领域，包括95后内容的普及化、从电视转向互联网产生的机遇和手游、内容IP等方面。此前，创新工场在这一领域已经投资了暴走漫画、有妖气、米未传媒、SNH48等。

一位创新工场方面人士对本报记者独家透露，创新工场将很快设立一支上亿规模、专注在文化娱乐这一垂直领域的基金。

值得一提的是，在文化娱乐领域，2016年第一网红“Papi酱”刚刚获得徐小平、罗振宇等在内的1200万投资。对于网红经济，李开复认为，更多的年轻用户尤其95后用户，正在把更多的时间花在互联网上，因此关注这种95后的内容是一个特别巨大的趋势，而针对某一个网红是否能做好，“还是看她的内容是不是有强大的吸引力和深度、广度。”

而在互联网教育方面，李开复认为，无论是美国式的相对更大颠覆式的互联网教育，或者是中国式的针对应试教育、出国类考试等都是短期的机会，“但是未来我觉得不仅仅是只有这样的领域，未来还有更多的机会在教育上，经过互联网的各种工具能够给用户很大的帮助。”

在融资危机和创业泡沫尤其明显的O2O领域，李开复表示，自己对于O2O大方向依然看好，因为中国的城市人口结构以及移动支付能力，正在形成一种全球领跑的生活方式，就是让衣食住行吃喝玩乐都可以来到用户身边。

在他看来，出现问题的O2O公司，问题不在O2O，而是公司本身是不是有创造足够大的价值，是借烧钱留存短期用户还是真有足够的黏性吸引用户多次购买?这些才是真正的问题所在。

“过去可能因为有些VC可能投的钱太多，有些创业者把故事和泡沫讲得太大了，没有给用户产生真的价值，个别的案子确实是有一些泡沫的发生，这都是正常市场波动。”李开复说，资本寒冬如同四季的变化，对于创业者和投资人来说每个季节都应该有不一样的打法。

“夏天就帮助我们的创业者去融资，到了冬天帮他节流省钱甚至裁员，专注营业线活下来，冬天还可能以更低的价钱投一些新的创业公司。对创业者来说也不用担心，我觉得冬天愿意创业才是真正的创业者，哪些只是靠夏天的来临，靠着泡沫融一大笔钱的创业者，冬天一来，正好是他们退潮的时候。”

“而且冬天来了，春天也不会太远 。”李开复说。
