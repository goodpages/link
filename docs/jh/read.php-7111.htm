高富帅请掩面绕道，您不需要。

女生请留步细看，了解对手，才能战而胜之。


你脸皮够厚，嘴巴够甜，却换不来女生回眸一笑？

你弯腰垂头，献媚堆笑，却只等来一句“你去死吧”？

你使劲浑身解数，却仍然单身，看着别人过节？

同学，不是你命苦，不是她高冷，而是你不懂方法，不懂女孩的语言模式和心理状态啊！

卡卡也不懂……但有两本书，现成的干货，卡卡分享给单身同学，祝华丽转身，早日脱单。

1.

首先，请看下面两组对话。

男：周末打算怎么过？

女：在家宅着。

男：这么好的天气应该出门呀，我们去看电影哦，听说《美人鱼》不错……

女：我去洗澡了。

男：终于下班了，你在做什么？

女：感冒了，在家休息。

男：这两天降温，要多穿衣服，多喝温水，好好休息。我同事也有不少感冒的，我没事就在单位喝板蓝根。

女：没事的，谢谢关心。

稍微有一点情商的人都能体会到，“谢谢关心”、“我睡了”、“我去洗澡了”这些词是女生想结束对话时的惯用词。对男生而言，这些词就意味着你搭讪失败甚至追女失败。

聊天都不在一个频道上，还怎么进一步亲密接触？

GAME OVER。

2.

正确的搭讪方式是不能简单地靠直觉，而是要了解女生的心理和语言特点。

从心理上来看，女生比男生更缺乏安全感，卡卡在《上海女孩提分手 差的不是房子是安全感》这篇小文中简单地解释过，不重复了。

心理安全感的差别决定女生更习惯于保持目前状况的稳定，保持眼前的和谐，因为变化意味着某种冒险。女生也从聊天中收集信息，但不是为了下一步行动，所以没有明确的目标性。

男生则不同，他们总是征服感爆棚，采集信息，然后进攻或防御。他们只关心跟目标相关的事物和状态。

所以，适应女性思维的语言模式关注具体细节：

表达感受、描述问题、关注过去、对照眼前、罗列现象、就事论事、活在当下、以自我为标准。

适应男性思维的语言模式则抽象概括：

做出评价、解决问题、探求动机、预测未来、建立因果、推理总结、防御否定、目的优先、结果导向，以客观为标准。

3.

日常生活中你会发现，愿意聊天的人总喜欢把话题说得更具体，而不想聊天的人则会顺着话题的方向直接概括出结论，“我已经都知道，你不用再多说。” 我们把概括化的语言模式称之为上堆，把具体化的语言模式称之为下切。

上堆就像在对话中升起一个防护罩，拉开了你和对方之间的距离，而下切则能传达出更友好更信任的态度。

男性思维决定男性下意识的语言模式天然是上堆的（抽象概括），自然和女生（关注细节）不在一个频道上，不作出改变的话没法聊天。

下面是两个失败案例。

男：认识一下可以吗……

女：为什么呀？

男：因为我们很有缘（抽象概括的男性思维）或者，因为你很特别（做出评价的男性思维）

聊天失败！谁和你有缘？

男：加个微信可以吗？

女：我和你不熟耶……

男：人不都是从不认识到认识的吗？（防御否定的男性思维）聊天失败！

为啥要认识你？朋友圈卖东西的不差你一个了。

下面是据说能成功的案例……

男：认识一下可以吗……

女：为什么呀？

男：刚才在电梯旁看见你（活在当下的女性思维），突然觉得如果不过来跟你说句话的话，我今天一定会后悔的。（表达感受的女性思维）

男：加个微信可以吗？

女：我和你不熟耶……

男：我也觉得有点儿唐突，但实在很想认识你，找不到其他办法了（表达感受的女性思维）

只是例子，实践时被打脸，卡卡概不负责。

再举一个例子，看起来更靠谱些。

妻子：昨晚干什么去了？回来这么晚？

丈夫：跟朋友吃饭去了。（抽象概括的男性思维，如果用女性思维，则会把朋友的信息讲述的更具体一些：“我跟部门的xx、xxx一起去吃的饭”）

妻子：真的吗？

丈夫：骗你我天打五雷轰。（预测未来的男性思维，如果是女性思维，则会“关注过去”：“真的，我们昨晚在xxx地吃了xx点”）

妻子：还撒谎，我昨晚在xxx就没见到你！

丈夫：你就这么不信任我！（探究动机、防御否定的男性思维，如果是女性思维则会进一步暂时具体细节、叙述详细经过：“亲爱的，xxx、最近在xx开了家分店，你去的是哪家呢”）

PS：一个被冤枉的好人由于极度紧张会也会导致男性思维，一个有经验的说谎者因为训练有素却可能会使用女性思维，所以并不能单凭语言模式就去断定真伪，语言模式只是我们了解说话者心理的一个途径。

4.

在男女搭讪、聊天中，男性思维不是不能用，但如果使用的话，最好要具备鲜明的观点或者独特的视角，人云亦云的道理或者显而易见的结论不讲也罢，会让女生感到乏味。

比如在找女孩要电话的时候，女孩问：“为什么要给你电话呀？”即使同样做上堆式男性思维的回答，——“因为我不是坏人”就比较俗套，——“因为你不是坏人”则比较有趣。

荐书《魔鬼搭讪学》、《魔鬼约会学》。
