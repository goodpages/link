为独立游戏开发融资

每每谈及游戏开发，人们很少涉及各种融资渠道。如今大多数开发者喜欢用Kickstarter众筹网站，然而条条大路通罗马，还有其他融资方法值得推荐。

这篇文章列出了一些值得考虑的融资渠道以及法律方面的注意事项。

自主融资

“自主融资（Bootstrapping）”就是用自己的存款或信用卡贷款为游戏开发融资，这完全可行——但相比之下，用别人的钱来投资不是更划算吗？不但风险大大降低，还能让开发者全心全意开发自己的产品。那么如何用别人的钱来投资呢？以下方法值得一试。

亲友贷款

亲友是最靠谱的资金来源，因为他们充分信任你，并乐意倾囊相助。除了应付利息之外，他们一般不怎么在意利润，所以不会牵扯到证券法。但其他融资渠道就不一样了，稍后会谈到。

请亲友投资的另外一个好处是：万一项目失败了，他们多多少少会更宽容一些。而银行和众筹跟投人就没那么好说话了。对急于寻找启动资金的开发者来说，这一渠道是最靠谱的，没有之一。


商业贷款

正儿八经的商业贷款适用于有限责任公司（LLC）或企业（Corporation），但前提是公司必须有可追踪的贷款记录，否则就得以个人名义进行担保，用个人信用换取贷款。

要是实在没别的法子，试试商业贷款吧，至少拿来做启动资金也不错。

定金投资

诸多创新型公司常用“定金（Front money）”来解决燃眉之急。这种渠道通常是投资者为一个新项目预付一定数额的定金，以换取股权或特许使用权。投入的这部分种子基金（seed money）主要作为启动资金，直到该项目融到所需数额的资金。定金融资容易触及证券法，建议咨询一下律师的意见。

欲了解更多信息，请参阅律师Gordon Firemark的这篇文章。

独立游戏基金

Indie Fund和索尼的Pub Fund属于独立游戏基金，跟种子基金投资相似，一般是投资者给开发者提供一定数额的资金，换取游戏发行之后的特许使用权。

捐赠式众筹

如今，捐赠式众筹恐怕是游戏开发者最青睐的一种融资方式了，因Kickstarter、IndieGoGo等众筹平台广为人知。这种融资方式严格意义上不算是投资，也不会触及证券法。

但这并不意味着免除了法律责任。开发者须跟投资者达成协议、签订合同，才能拿到对方承诺提供的资金。基于这一点，就有必要在签订合同之前对项目进行合理规划。


股本融资

在游戏开发行业，有一种融资方式并不招人待见，但却颇受创业家们的欢迎，那就是股本融资。这种方式相当于卖掉公司的股权来换取创业资金。

这种融资方式的确会触及州及联邦证券法。公司的首次公开募股（IPO），即公开销售公司股份，就是股权融资的一种形式。其特点是周期长，成本高，过程往往需要诸多律师的参与以及大量披露公司财务状况的文件。虽然繁琐，但融资潜力巨大。

考虑到公开发售（public offering）的法定要求往往有例外情况发生，所以适用于仅仅针对少量投资者或大量具有特定资质的投资者的融资，这样一来可避开成本高昂的SEC（证券交易委员会）注册手续。如果已经筹到了种子基金，计划迈出下一步，股本融资是个不错的选择。

股权众筹

美国2012年4月通过《创业企业扶助法（JOBS Act）》之后，众筹行业就翘首企盼所谓的“股权众筹”能够在法律上落实，好让公司借助Kickstarter等众筹平台来发售股权，而不仅仅是出售游戏的特许使用权等。这样的众筹听上去挺适合开发者的。

不过这么多年过去了，股权众筹还是没着落。尽管Kickstarter已免去了很多手续上的繁文缛节，但我还是不确定这条路游戏开发者走不走得通。对那些开发了一部分游戏，还想继续筹资的人来说，股权众筹正合适。

结论

希望这篇文章对开发者有所帮助。若有任何问题，或者在寻找合适的融资方式，还是咨询一下律师的意见吧。我写的两本关于游戏开发法律事宜的电子书，包括《合理使用（Fair Use）》也许有用。另外，我还成立了一个名为indieGenerator的网站，帮助独立游戏开发者在签订游戏合同时尽量降低律师费开销。
