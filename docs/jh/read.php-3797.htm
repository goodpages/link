“悠嘻猴”创造者：创业有时靠时机

从游戏公司到动漫公司，这家公司摸索出一个新商业模式。

　　有人说创业这事儿要靠“时机”，好的商业模式要比别人走快半步，而不是走快一两步。其实事情本身没问题，只是由于市场不成熟，导致策略错误。2005年时的上海玖峰数码科技有限公司(以下简称“玖峰”)就面临相当危险的局面。

　　张京鹏刚刚空降到玖峰时就赶上这里严重的入不敷出，这是疯狂砸钱砸出的后遗症，整个公司快40个技术研发人员，分成六组，“一意孤行”地开发了近40款手机游戏。要知道，做无线互联网行业在某些程度上需要“靠天吃饭”，2003年就开始开发手机游戏，还想以量取胜，而产品惟一的输出渠道就是手机“百宝箱”。那些热门游戏的成功点连开发者自己都说不清。当然这是后话。

　　公司处于穷途末路的临界点，出资人萌生退意，合伙人之一郁峰很不甘心，便盘下其他人的股份。他和张京鹏想继续做游戏，但是活命要紧，只得把多数员工“劝退”，只留了5位：一个程序员和两个美术设计足以做个网站，还有两个文案可以做营销策划，顺便跑跑市场。

　　张京鹏和郁峰决定去找点有现金流的事来做，为了五万块钱的利润，去ChinaJoy(微博)帮人家布展，在上海的炎夏中到处求购道具。大家腾出空做了个网站“2UC.com”，取名“爱游戏”，包括网游资讯、论坛和游戏工会，内容并无特别之处。张京鹏之前曾经开发过卡通风格游戏，就效仿新浪的大眼睛、腾讯的小企鹅，请美术师顺手画了只Q版小猴作为“爱游戏”的吉祥物。

　　这小猴大名为“游戏猴”，听起来有点生硬，就按照谐音UC演变为“悠嘻猴”。开始只是当做论坛的头像，后来又画了些作为论坛的表情图。几天后很多玩家就来反馈，小猴挺可爱，可否推出些以小猴为背景的桌面？大家又顺手画了几张挂出来。结果郁峰发现其他网站和论坛竟然有人在借用悠嘻猴的形象。难道这个小猴乘上了互联网病毒式传播的快车？

　　玖峰处于财务紧张状态，也许这是上天为他们打开的一扇窗。仅仅是和几位腾讯公司的负责人在网上聊了聊，对方就表示，希望“悠嘻猴”以表情形式进驻QQ，来调动网友们的情绪。很快，他们就与腾讯签下战略协议，成了腾讯的第一个“表情提供商”，接着还和MSN签下合作。玖峰的第一批悠嘻猴表情做了10个，在QQ和MSN传播极快，QQ干脆提供了一个表情包让用户下载，之前表情并无打包下载模式的MSN还专门为悠嘻猴推出了表情包。

　　在极短的时间内，悠嘻猴变成了一个“表情公司”。最初画悠嘻猴只是因为好玩，彼此做做鬼脸引发灵感；后来在腾讯、MSN的要求下，要定期交上“创作”，大家每天的任务就是苦思冥想，如何用小猴的表情和肢体语言把不同情绪表达出来。这是个没人做过的事，团队很兴奋。

　　悠嘻猴的形象也逐渐变化，从眼睛是长方形，进化为眼睛眯成一条线的“线眼”，因为张京鹏觉得日式水汪汪大眼睛和“豆眼”都不如“线眼”更搞笑，小猴的脑袋也越来越大。郁峰和张京鹏想过，从商业角度，所有生意都要看供需，以前大家爱在QQ上发个笑脸表示“笑”，但要表现“坏笑”该怎么办？用一个小猴以表情、肢体语言更容易理解。“表情”是个需求点，挡也挡不住。

　　不过面前最大的问题是，这个团队还没有找到自己的盈利方式。与腾讯、MSN的合作都不涉及费用，之后腾讯开发了播放一段Flash的“魔法表情”，找到人专门根据玖峰的创作制作Flash版本并针对会员收费，可玖峰在此之中还是分文不取。大家感觉应该继续走“表情”的路线，相信只要有人喜欢这只小猴，总会有商机。和腾讯签约后要在几个月中赶制出200多个表情图，团队通宵达旦进行创作，非但没有人抱怨，还有位担任美工的小女生在MSN签名中为悠嘻猴写道：“宝贝，你会成名”。这种期许让郁峰和张京鹏特别感动。

　　即使是为了同事们，也该尽快考虑收入了。他们以悠嘻猴为主角做了一些Flash动画、壁纸等，并决定做一款“落地”产品。2006年《魔兽世界》正火爆，他们设计了一款悠嘻猴摆件，上面是小猴Cosplay牛头人的形象，到ChinaJoy上去发放。结果3平方米的标展旁边排了长队，从免费发到搞点小游戏向优胜者发放，再到后来很多人对着盒子上的标价“66元”直接掏钱购买。悠嘻猴摆件卖了1000多盒，成了那届展会最火的产品。

　　那次经历让郁峰和张京鹏感悟到，其实悠嘻猴是个动漫形象，只是在网络上诞生而已。但他们还是希望来个不按牌理出牌：不去设定针对年龄层的定位，惟一的硬指标就是“可爱”，这使悠嘻猴无法针对某个年龄层主推高利润产品，却能将男女老少“一网打尽”；故意不给悠嘻猴设定性格特点，把悠嘻猴炮制为一个Cosplay演员，今天演《魔兽世界》的牛头人，明天演《大航海时代》的海盗，像捧演员一样把悠嘻猴捧为卡通界的“刘德华”。

　　紧接着，他们又想到配合电影《恋爱上上签》推出了悠嘻猴版的情侣手机挂坠。这是郁峰在火车上想到的创意，两个小猴拼在一起就是个桃心形状，里面还有“签”，当然都是上上签。第一批卖掉大约2.5万套，断货后在淘宝上一盒可以炒到108元。

　　这又引出了他们的设想，两只小猴可以出双入对，作为一对小情侣分别叫“悠悠”和“嘻嘻”，凡是成双成对的产品容易将价格提升，销量还可以翻倍；从Hello Kitty多年大热可以看出，女孩的钱比较好赚，而成双成对的产品是女孩看上后男孩付钱；再往后延伸，悠嘻猴没准还可以涉及到结婚礼品。他们还希望在爱情之外打友情牌，为悠嘻猴设计出一帮朋友。没有年龄定位、Cosplay演员、成双成对出现……悠嘻猴的特点终于丰满起来，无意中与多数卡通形象形成了差异性。

　　此后玖峰的道路果然平坦起来，越来越多人喜欢悠嘻猴，商机也找上门。2007年他们一口气授权了大概300多种产品；完美时空、久游网等游戏企业也找到他们，将形象授权到《梦幻诸仙》、《劲舞团》等游戏中去，这部分授权的报酬不菲；张京鹏又重操旧业，2006年底开发的网页游戏《乱舞春秋》以悠嘻猴为主角，在第二年开始公测，无意中成了最早的网页游戏之一；郁峰则开动脑筋思考一些周边产品，例如“三国杀”系列，观音菩萨、唐僧等角色全由悠嘻猴“扮演”。

　　郁峰和张京鹏有时也会感慨，如今创造个卡通形象真不容易，连当时“大哥大”绿豆蛙也难逃“短命”的宿命，而悠嘻猴却在腾讯QQ上保持着转发率第一位。没准这真的要归功于两个人的不务正业、不按牌理出牌。
