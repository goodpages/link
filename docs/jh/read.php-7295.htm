诗哲魂：中国人文学科落后于科学技术的现状令人担忧

 

在中国谈论人文学科或人文精神必然涉及到政治，因为，中国的传统政治与今天的政治都有一个死结：那就是对人们言论与思想自由的钳制。人文精神必须在政治框架中进行，必须要带着镣铐跳舞，这意味着人文思想者必须要服从当前政治的需要。将人文学科纳入到政治框架中总体安排之后，其实就是限制了中国自由思想家、以及人文精神主义者仰望星空，进行形而上追寻的多种可能性，也就限制死了人文精神批判性、想象力和创造性；从此，也就失去了对科学技术盲目强大的约束作用；丧失了对社会进行制导与校正的人文精神历史作用。

 

我这里所说人文学科落后与弱化，是指相对于今天中国科学技术发展状态来说。它有规模，却缺少独立自由精神；有成果，却很少具有真正地人间关怀；有专家，却很少具有世界级的文化大师；有创造，却多是一个政治需求实用解释与研究；有学者，却难以保持着学者清流和卓然超群智慧。其中，令人悲哀的原因是，中国人文精神脉络中断了、失缺了、堕落了、或完全被政治所俘虏了；在商业化和全球经济主义主宰世界里，一个集体人文主义消失的社会，让我们的公共知识分子，或背叛、或分散、或流亡、或消沉了……

 

如若想在当今政治中寻找一个可以自由言说领域、独立自由思想空间，恐怕很难从人文精神领域中寻找。相对于人文学科，科学技术的发展空间非常之大，科学无禁区，在科学技术上，可以完全可以个人自由发挥与想象，它充满了冒险精神和自由力量，相对于人文学科它具有了最为明显的优势，这样说不为过。而人文学科却始终在政治钳制下从未能松动过一时，现在说人文学科落后于科学技术现状，并不是说人文学科没有规模，而是指它缺少内在这种如同科学自由地冒险精神，独立的精神价值判断；缺乏真正民主争论的精神，缺乏政治宽容的生态环境，政治对自由思想的钳制仍旧非常利害。政治这个死结，几乎扼住或停止了中国人文思索精神，窒息了人们如同科学技术本身一样的鲜活创造力。

 

近代社会肇端，从“五四”运动以来，中国的科学总是大于社会的民主，技术的力量总显得比人文思想更为重要；尤其是改革开放以来，科学技术已经成为社会唯一的发展观念时，人文精神主导下各类社会学科，在中国，正遭遇最为困难的发展时期。当社会发展的双轮缺少了另外一个驱动力时，高速发展科学技术社会究竟全带给我们什么样的命运呢？当人文精神中所包含的智慧与学科，对社会问题不敢直面，不敢直言，不敢碰撞，不敢大胆进行创造性思考的时候，我们为中国人文学科所处的悲哀境地困惑和担忧。

 

科学救亡、科学救国、技术致富、技术强国等意识，远远超过了人文思想的启蒙作用与反思作用；加之，政治对人文学科的钳制愈来愈严重，窒息人文思想自由力量日益强大起来。在缺少人文精神约束下科学技术主义泛滥成灾情景下，中国文化中最具有独立精神的人文学科发展极为缓慢，有时甚至是陷入到非常荒诞的境地。人文学科与人文精神堕落到仅仅作为政治解释的工具、御用宣传工具，这无疑是中国文化精神的悲哀。人文学科中自由精神与独立丧失，这是中国缺少对科学主义泛滥的制衡标志。它最终将会引导我们彻底丧失了人文精神的思想警觉，最后落入彻底地西方化，最终也抵抗不了它必然的历史命运和结局。

 

人文学科能够独立和自主发出自己声音，这是西方世界各国成功的经验之一，它包括学术自由，新闻独立的自由，思想传播的自由，言论发表的自由，适当的放宽人文精神探索领域和范围，最大化的满足人文精神自由创造性发挥，对科学主义泛滥社会采取有力的抑衡是非常必要的。如若用政治强制控制它们发出声音，另外一个可能抑制社会盲目发展约束性功能就会丧失。人文主义精神与学科就是担负起如此重任，才会在西方历史上掀起了一次次启蒙主义思想运动，一次次文艺复兴运动到来，它的主要作用就是抵御社会单向度的沿着某个盲目道路发展。

 

在中国，从事人文思想与精神活动的知识分子，目前，是世界上最为胆小知识群体。其原因：一是长期以来文革遗风导致的政治迫害，其阴影还笼罩在知识分子身心上，导致了今天自由知识分子集体创造精神的萎缩与恐惧；二是高压的政治意识形态，塑造、控制和板结了人们鲜活人文主义者思想创造的自由，造成了人文思想领域中充斥着虚假和不真实思想表达；三是科学技术主义强大到足以让人们放弃人文思想所有领域和阵地，中国人文的思想境界，无耻的堕落到已经令人难以忍受的地步；四是缺少自由言说和尊重独立自由精神的社会氛围，恐怕因为思想犯罪而丢掉饭碗现实，决定了大脑自由创造力彻底全面的退化，颓废的文化精神与堕落品质主导着他们的生活，这恐怕是中国人文精神独立的丧失，自由面临的萎缩，创造精神被毁灭的真实困境与现状。

 

中国的人文学科总处在一个被政治控制或驱使阶段，许多时候，强大的政治意识形态板结了人文学科的鲜活精神，抑制了人们独立与自由的创造性和想象力。

 

中国政治问题的核心就是，如何控制人文学科中自由地精神力量，独立的思考力量，特有的价值判断力量。在科学主义和技术力量强大的今天，因为，自由民主精神是通过人文与科学技术一起发生作用，它是社会健康发展的双轮，缺一不可。但是，目前的情景是，中国人文学科一直处在政治强迫之下，难以独立和自由按照自身规律去发展，丧失了它对于中国社会舆论导向作用，对人文精神的启蒙作用，对科学技术监督作用。许多时候，是科学技术的盲目性决定我们的社会方向性，这是非常危险的。

 

缺少了人文学科对科学主义至上的制约功能，缺少了自由人文精神为前导的社会舆论，注定要迷失社会前进的方向，这是人文学科在中国现状非常令人担忧地方。如若，它继续在政治意识形态高压之下前行，仍旧采取如此的思想进路迫压人文精神自由，中国人文学科就真正变为政治御用工具，彻底丧失了他的反思、批判的社会基本功能。这恰恰是中国当代政治的保守与僵化的问题实质所在。现在的情景是，应该更加放开言路自由领域，更加尊重自由与独立的创造精神，满足人们思想多元文化的精神世界需要。中国社会的保守与僵化政治传统，就是对人文精神领域干预太多，抑制太狠，发挥的不好，让它一天天丧失了鲜活生气，这也是中国社会为什么一直以来未能出现世界级文化大师的根本原因。

 

以科学来制导社会发展方向的选择并没有错，错误在于我们放弃了人文精神与学科制衡的力量与作用，这是中国近年来的一个失误，但其原因并非政治本身问题，而是前一个历史阶段，文革精神保守与僵化思想的遗绪所为。长期以来，中国传统政治对自由思想钳制滞后性引来后果，存在于我们每一个人心中，这种受其迫害阴暗心理，是中国人普遍的精神状态，也是中国政治控制思想力形成习惯和惰力。因此，在中国谈论政治是非常危险的事情，可是自由受到政治钳制太多的时候，我们无法感觉自由和幸福的生活。如今，在缺少制衡科学技术泛滥社会盲目发展中，多少人的命运是因为科学主义强大造成环境悲剧带给他们的。从这个意义上看，政治对人文精神制衡作用与管制作用，早已超出了政治本身，它带给我们的内心的恐惧与灰暗和没有希望的生活太严重了。

 

在中国面对全球化境遇时刻，忽视了从人文主义立场去发现和观察问题，完全依赖科学技术盲目性建构我们的未来，放弃了人文精神制导作用，这会是非常危险事情。世界上许多优秀的民族文化，对于思想和精神领域中自由放任，对人文精神采取独立判断价值尊重，对人文学科完全以非政治态度与立场处置经验，才可能给一个民族注入鲜活力量与智慧，才会给一个国家带来更加美好的未来。我们的国家与民族必须要警惕这种对人文精神毁灭做法，纠正政治传统对思想钳制严密控制错误，否则，中国，在未来的国际社会中，在未来文化上，就一定不会取得更加辉煌业绩。

 

我们必须要反省这种状态是如何发生的，为什么中国社会不能具有比较独立人文学科健康发展呢？其根本的原因就在这里。

 

不是要人为区分科学与人文之间的矛盾，而是从我们自身经验中感觉到，科学主义的强大几乎把人文精神视为可有可无存在状态，这绝对是一个错误；把人文学科置于政治严密控制下也是一个错误。人文精神呼唤出来的道德力量，是校正科学主义正确发展方向最为重要智慧。人文学科千百年来培养起来的批判意识，人文精神所表达出来世界关怀和宽阔的胸襟，不能因为我们坚持一种科学技术主义至上理想生活方式，就被彻底毁灭了。我想，如果后人看到我们今天，如此对待人文精神与思想状态，一定会责备我们；因为，我们犯了一个大的错误，一个不允许犯下的错误，用政治抑制了鲜活人文精神注入我们灵魂，从而丧失了一个社会健康发展道路。

 

科学主义强大到一个民族可以放弃所有文化的发展方面，去追逐它科学技术的单一社会功能，这必然导致中国人文学科没落与衰退。一直以来，人文学科处在解释和维护科学技术发展的奴仆地位，处在任人宰割状态，这是人文学科历史上最为悲哀阶段。现在我们可以正视这个问题了，并非是取消或削弱政治的作用，而让政治发挥其真正地社会功能作用，回归到政治本体中去，而不是仅仅为钳制人们思想自由与人文精神功能而存在；充分让人文学科得到更加自由发展的空间，更加理性和宽容的环境，更多地鼓励智慧者，参与并理解其人文精神自由地重要作用和功能，这才是纠正我们对人文主义精神缺失的一个良好的历史开端。（ 2007-11-9）
