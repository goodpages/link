2016年，一名游戏策划师对VR市场的看法

最近正好有些时间，打算写点东西，一方面算是对自己的一个小总结（等老了可以看看），另一方面也想尝试是否能够通过一些不同的方式结交到志同道合的人。


　　我想也许我应该在一开始就说明一点：以下文字仅代表我个人的看法。


　　对VR前景的预想：VR将创造出N个平行世界


　　它将真正的实现足不出户行走天下甚至实现次元生活。VR技术将和电脑科技一样彻底的改变人类社会的运行方式（这远比智能手机的影响要大得多得多）。


　　人们将能够在虚拟的世界中处理除了生理需求之外的所有事物，当然也必然包括工作和娱乐。


　　它将取代现在主流的所有其他沟通方式。到时将很少有人把主要精力放在现实世界里。


　　而这一切的开始时间可能会比我们预期的晚一些；


　　这一切的发展也可能会比我们预期的快得多；


　　5年之后，VR将作为新颖的娱乐设备被大众接受；


　　再过3年，VR会成为大都市人们工作方式的选择之一；


　　用不了3年，VR将成为很多生活在大都市的人们工作方式的唯一选择；


　　而他们的下一代恐怕对某个或某几个平行虚拟世界的认知要远超过现实世界了；


　　而这个界限可能在于：从法律上正式承认某个虚拟世界中的NPC为一个“物种”；


　　回到现实吧！


　　身为一个游戏策划师，我非常期待能够使用VR亲手构架虚拟世界（从某种意义上来说我之前的工作也是一样），并让进入这个世界的人们感到快乐甚至惊叹。


　　让我们先来分析下VR本身


　　对于关注或已经开始虚拟现实（VR）研发的人来说，我们受限于2个客观条件：



  • 硬件条件；
  • 软件条件；


　　就硬件条件而言，成熟的VR体系需要包含众多的全新硬件设备，比如：



  • 全景显示器（头盔）——这个不需要说明了。
  • 全景音响——你需要通过声音获得远比你想象的更多的必要信息，比如物体的距离，移动速度，保持你的平衡感，模拟真实的反馈等等。
  • 语音识别——你不仅仅需要别人知道你在说什么，电脑也需要知道你在说什么，这点在VR游戏中尤其重要；
  • 手部感应器——你还会需要键盘，但那是VR世界的键盘不是你办公桌上的；你也需要抓取，投掷，劈砍；相信我你需要有触觉，这至少包括压力反馈，甚至材质反馈；
  • 腿部感应器——你当然需要移动，也许还想跳一下；
  • 其他感应器——我不确定你是否会想尝试一下飞翔？潜水？骑马？或者仅仅是让你的屁股感觉你坐下来了等等。


　　但一开始就说了，上面这些是“成熟的VR体系”，就现阶段而言，我们需要取舍，现实一点总是干正经事的规则，那么我们来看看软件条件：



  • 我们需要构架世界规则的工具——引擎类软件
  • 我们需要制作世界部件的工具——比如图形编辑类软件
  • 我们需要给VR世界更多真实感的工具——比如声音编辑软件
  • 等等


　　在VR展现雏形的现在，能做些什么？


　　人们都希望能够在产业之初成为进入该领域的第一波人，无论是出于对成就的向往还是对财富的渴望。


　　如果你是硬件公司，也许应该开发一款简单但是充分体现出“设备特性”的小游戏，就像智能手机上的愤怒的小鸟。这也许能帮助你最大限度的占有本就不大的市场，为以后的进一步扩张开个好头。


　　如果你是软件公司，那么根据你的软件特性开发“具有独特功能和效果”的游戏，会引起像我这样的软件使用者的注意


　　而众多的第三方公司围观者，有些都还没有想清楚到底怎么迈这一步，有些在踌躇现在究竟是不是迈入VR的好时候。


　　无论如何，如果你是和我一样将VR视为目标领域的开发人员，那么现在是开始准备的最后几分钟了。


　　我们需要做一些积累和尝试，我们都知道最好的积累方式就是通过项目的实际开发；我相信我们同样也会对一个观点有所共识：公司是盈利性质的。所以我们针对VR领域开启的项目必须满足以下条件：



  • 能够积累VR项目硬件和/或软件的开发使用经验；
  • 项目可以为公司产生盈利；
  • 严格控制项目的开发周期和技术范围（现在不是做大玩意儿的时候）；
  • 不论你是打算以后从VR转移到其他平台，还是从其他平台转移到VR，在底层框架编写时你应该留有一定余地；
  • 想清楚你到底要的是什么，再开始做。如果你的最终目标很大，那么你应该设定计划，并且从专业的角度入手。你没有那么多时间浪费在回头路上；


　　“你会怎么做？”


　　我也时常这么问自己，每一次思考会让答案越来越清晰：



  • 我不会选择开发VR游戏，至少现在不会；（除非和有资格的团队合作）
  • 我会选择使用能够开发VR游戏的引擎来开发PC游戏；（我想我会选择UE4）
  • 我会认真考虑在一开始放弃国内市场；（这也许是最能保证盈利的方式）
  • 我会选择“只有我才能完成设计”的游戏类型；（提高成功几率最直接的方法就是减少你的竞争对手，如果你能制造条件令对手无法和你竞争就是最好的）


　　最后的一些字


　　也许你和我一样，都是寻找VR世界大门的游戏开发者。我们都在尽自己所能磨练技能，补充弹药，寻找同伴，准备战斗。我并不知道你在哪里，但你并不是独单。


　　欢迎大家和我联系一同向VR游戏世界前进！
