“中国最美女黑客” 现身网络

　　专门破解软件和教人使用黑客工具，昨接受信息时报采访坦言既是爱好也为赚钱

　　美女加上黑客通常马上就会成为焦点。早前，波兰知名美女黑客Joanna Rutkowska就因发表了一篇曝光了英特尔CPU一个缓存漏洞的文章而声名大噪，一举成为安全业内的焦点人物。中国的黑客不少，但女性黑客却寥寥可数，美女黑客就更为罕见了。甜美的相貌加上高明的技术，令上海女生“YingCracker”近日人气大涨。

　　收费教人控制“肉鸡”

　　在网络论坛上，一篇题为“中国最美女黑客”的帖子引来不少网友的关注。昨日，信息时报记者与这名QQ名为“YingCracker”的美女黑客取得了联系。进入“YingCracker”的QQ空间，只见她在日志中发布了不少关于破解、入侵等技术的基础教程，更不时放出一些免费的小软件供网友下载使用，如“QQ登录密码加密算法分析和QQ登录器编写”教程、IP修改工具、Office密码清除工具等。

　　当然，并非所有的软件都是免费的。“YingCracker”告诉记者，进入这行既是因为个人的爱好，也是因为家里并不算富裕，想通过自己的技术赚点生活费。黑客可以通过卖软件、教学等途径赚到钱，例如，一套蚂蚁帝国V6.8攻击器售价是500元，PDF漏洞网马下载者软件售价是2400元。由于技术好，慕名前来拜师学艺的人也不少。据了解，一般一个具有初级计算机知识的人要学会Oday技术（也称为信息安全技术），至少要花上两至三个月的时间，而她教一个徒弟的收费是4000元。“学这个的人挺多的，有的是因为个人的原因，想搞‘肉鸡’（即控制别人的电脑），有的是因为商业的原因，想搞流量，入侵国外的网站，通过挂插件之类的赚钱。”她说。

　　不少网友成为“粉丝”

　　除此之外，她还会替别人破解软件，破解一个软件的价钱通常是500元至5000元不等，具体的价格是根据实际破解的难度，正版软件的价钱，以及更新的费用来设定的。她向记者透露，以她现时的技术，每个月平均可以赚到约15000元。

　　除了在个人QQ空间上发布一些免费软件和语音教程外，“YingCracker”同时也是一个名为“我爱黑客安全组织”论坛的版主。而据她所知，中国“学这个的女生几乎是没有的”，美貌加上技术令“YingCracker”的QQ空间人气大涨，不少经常到她博客取经的网友都成了她的忠实fans。不仅是破解、入侵、漏洞分析的高手，还是美女，更令网友将她封为了“中国NO.1美女黑客”。“你的技术把你的魅力提上去了，也许这就是你吸引人的地方吧，崇拜死我了”，某网友留言说。

　　常被人冒名借以行骗

　　随着名气的增长，知道“YingCracker”的人多了，冒充她在网上行骗的人也多了起来。虽然技术高明，但这些网络骗子也令她相当头痛。最近，就有人在网上自称是她的软件代理，通过卖她的软件来骗钱。该骗子声称，可以以1000的低价贩卖她自创的PDF漏洞网马下载者软件，有买家上当受骗后便来找她晦气。对此，“YingCracker”感到相当无奈，只能在日志中一再提醒众买家，从未让任何人代理出售她的0day软件，让大家千万不要上当。“网上什么人都有”，她说，每天加她QQ的人很多，有想用她的教程骗人的，有调戏、 忽悠她的，甚至还有性骚扰和勒索的。

　　而或许因为内容敏感，昨日记者发稿时，网络论坛上那篇介绍她的帖子已经被删除了。信息时报记者 刘敬
