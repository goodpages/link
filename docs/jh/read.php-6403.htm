猎豹移动收购[别踩白块儿] 开启手游全球化

GameLook报道 / 日前，手游发行商猎豹移动对外宣布，耗巨资完成了对单机手游《别踩白块儿》的收购，正式将这款2014年度全球手游下载榜第9名的作品收入囊中。这也预示着猎豹移动以休闲游戏为切入点，开启手游全球化布局。

《别踩白块儿》是一款考验玩家双手灵活性的游戏，在有黑格子和白格子的四列四行画面里，只能碰黑格子而决不可碰白格子，随着画面的变换，玩家需要不断点击黑色格子，使画面向前，玩法十分简单。

《别踩白块儿》全球下载量突破1.6亿

据悉，早在2014年6月，由猎豹游戏独代的《别踩白块儿》下载量已经突破1亿，截止目前，该作的全球下载量已经突破1.6亿，并获得谷歌全球I/O开发者大会点名推荐。游戏一经上线便登顶全球40多个国家或地区的App Store以及Google Play游戏类免费榜，并在超过一百个国家或地区排名前十，连续两个月获得iOS平台全球游戏下载总榜第一。

《别踩白块儿》3月22日~6月24日在中国区iPhone免费榜的成绩

值得注意的是，2014年6月正是猎豹移动独家代理了《别踩白块儿》的全球发行权，并整合猎豹清理大师（Clean Master）、金山电池医生等旗下App资源在全球范围内大力推广，使得该作持续升温，迎来第二波下载高峰。据App Annie《2014年热门趋势报告》显示，《别踩白块儿》取得2014年度全球手游下载榜第9名。在加州旧金山举办的谷歌全球I/O大会上，演讲者着重介绍了这款游戏，并现场演示了玩法。

猎豹移动旗下用户超过8亿 开启游戏全球化布局

在刚刚结束的猎豹移动全球年会上，猎豹移动CEO傅盛在年会演讲中，对公司的现状及未来的战略走向做出了明确的阐述，游戏全球化成为公司战略很重要的组成部分，收购《别踩白块儿》则是猎豹游戏走向全球化进程中具有里程碑意义的一步。

猎豹移动用户数据，到2014年Q3猎豹移动活跃用户数达3.4亿

傅盛在年会上透露：“猎豹移动产品总安装量超过8.6亿，也就意味着全球有8.6亿台手机曾经使用过我们的软件，我们的移动月度活跃用户有3.4亿。这意味着全世界每个月有3.4亿台的手机跑着我们的App。无论在美国、日本、中国、印度，你几乎能在地图上能找到的任何角落，都有人使用着我们的Clean Master、CM Security、CM Browser等等。”

而当提到游戏时，傅盛也首次提出了猎豹游戏全球化的战略：“我们的游戏全球化才刚刚开始。我们要把中国游戏带到全世界，也把全世界最好的游戏带到中国。”随着将《别踩白块儿》收入囊中，猎豹旗下又新增一款杀手级流量产品，可以为后续游戏提供海量的优质用户导入，这也给了傅盛和猎豹移动更足的底气。

据了解，从2014下半年开始，猎豹移动便开启了休闲游戏（含轻度游戏）全球发行平台的筹备工作，一方面在全球范围内寻找优质的休闲游戏开发商和产品， 另一方面探索旗下流量的导入模式，力求更快速、更精准、更直接的将优质产品推荐给用户。

休闲游戏利于游戏分发

手握如此巨大的流量资源，猎豹移动为何将休闲游戏列为游戏全球化的切入点？ 对此，猎豹移动游戏事业部总经理曹兴邦认为，休闲游戏拥有上手难度低、用户群足够广泛和文化差异小等三大特点，非常利于猎豹移动借助旗下工具类软件统一向分布在全球各个国家和地区的用户进行分发，相比重度游戏而言，休闲游戏发行能将猎豹流量的使用效率放的更大。

曹兴邦以《别踩白块儿》作为案例进行了说明“游戏玩法足够简单明了，甚至不需要太多文字说明，游戏本身也不存在文化差异的问题，各个国家的玩家都能很快的明白并喜欢上游戏的玩法，上手操作也足够简单。猎豹移动仅在安卓平台上，便能在海外每日为《别踩白块儿》直接导入10万以上的新增用户，从后台分析，各个国家的用户留存差异不大，这一点是重度游戏无法做到的。”

据早先的消息显示，猎豹移动除了拿下《别踩白块儿》外，此前还独代了三消类产品《糖果萌萌消》，意欲在用户基础最为广泛的休闲类手游市场中抢得一席之地。另据猎豹内部人士透露，公司业已重金签下了另外两款休闲游戏产品，均拥有着极其知名的IP和市场影响力，并将于3月份陆续发布。
