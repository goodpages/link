在8：15起床

终于没有人问我为什么每天发天空的照片了，呼！但是，你知道的，许多人的小名叫做“十万个为什么”，所以，新的问题又出现了：和菜头，你为什么能睡到早上8：15？


这种问题看多了，我才想起自己不是个正常人。许多我习以为常的生活习惯，在别人那里完全是匪夷所思。为什么我能够睡到8：15？因为我的住处距离我的工地只有3.7公里的距离。我曾经在夏天的时候步行上下班，大约需要走30分钟左右。与之对比，我知道很多人每天上下班要耗费两个小时以上的时间。


我不是今天才开始8：15起床，事实上，在我职业生涯的大部分时间里，我都可以睡到8：15。没有什么特别的原因，只是因为我住得离工作地点近，原则上不超过15分钟车程。在我年少需要打拼的岁月里，我住单位单身宿舍，走路到办公室需要5分钟。有些时候，我甚至直接在办公室里打地铺。这样一来，早上起身去食堂打早点，又能省下5分钟时间。


即便是我后来北漂，公司在寸土寸金的中关村，我两手空空一无所有，仍然拿出月薪的三分之一在海淀南路倒座庙租了一套房，这样我每天早上只需要步行10分钟就能抵达公司。听起来很傻，很不划算是么？那我问你几个问题：


如果你的同事和你在智力、体力、能力上都不相上下，你能拿什么跟他们拼？你大概只有微笑和精力两个可选项。那么，如果你每天需要在通勤上耗费2个钟点，一周五天，天天如此，请问你哪里来的微笑和多余精力？别和我坐公交和地铁其实是在休息，在那种环境里，你每时每刻都要保持警觉，不想错过了站，更不想下车的时候发现口袋被割开了一条口子。人在公共交通系统里承受的精神压力，甚至要比工作时更大。出了车门，人早已经精疲力竭，连一句话都不愿意说。


在我看来，结束了一天辛苦的工作。只需要走路15分钟就能到住处，把房门一关，所有的辛苦和烦恼都立即关在门外，这对于职场人士来说，是最大的幸福。同样是走进家门一头栽倒在沙发上，我倒下15分钟之后，能恢复元气起来下楼吃晚饭。但是，坐了一个小时车子的那个家伙，很可能再起身的时候已经是次日凌晨。于是又要宵夜，又担忧发胖，就这么翻来覆去睡不着。等好容易入睡，没过2小时，闹钟又响了。


当然，住得远一点，房租也就便宜许多。这个道理我也懂，但这也是我和很多人有所分歧的地方。房租便宜的意思是因此有结余，手头因此而宽裕一些。换句话来说，这种想法认为个人资产的增长主要靠储蓄。有这种想法很正常，可我要问一句：这种想法是从哪里来的呢？很大可能是家传。你的父母亲戚告诫你，钱一定要省着花，银行户口上有钱，心中才不会发慌。我的父母也做过同样的事情，我也同样相信过。


问题在于：我父母的那一辈人，青年时代处于计划经济之下，我父亲月薪70多块钱拿了很多年，我母亲30多块钱也拿了很多年，绝大多数家庭同等贫困。在他们的时代里，收入的全部就是工资，个人唯一能做的事情就是在家里养点鸡，或者单位集体宿舍后面的自留地种点蔬菜，聊以补充家用。如果他们不储蓄，利用银行那点可怜的复利，根本不可能看到财产有任何增长。


但我所处的时代完全不一样了。现在银行的定期年利率是3.15%，每年的通胀又是多少？靠节省下的钱去储蓄，基本是入不敷出的一个局面。一个人唯一能做的事情就是投资自己，和储蓄相比，转职更为靠谱。和转职相比，晋升更为靠谱。和晋升相比，跳槽更为靠谱。这个社会里自由经济的那一部分，为每一个人的能力和经验开出了对应的市场价格。所以，与其去努力结余，不如努力提升自己，换一份收入更高的工作。


可是，如果你为了省一点钱，住在遥远的郊外，那么你每天将不得不在通勤上耗费两个小时。这就意味着你每天都处于精疲力竭的状态，遇见有挑战性的工作机会，你是最早被耗尽的那一个。这也意味着你并没有多少余暇时间可以思考沉淀，可以自学成长，因为在路上除了看网络小时、打电子游戏，你什么都干不了；这还意味着你是所有聚会里走得最早的那个，因为过了11点就没有了地铁，于是你同样是挤出精力参加社交活动，最终却没有落下多少人脉---谁能记得走得最早的那一个呢？


最重要的是：你不得不重复上班---下班---赶车---睡觉的生活模式，偌大一个城市里，各种文化活动都和你完全绝缘。十年下来，你会变成一个除了工作之外一无所有，内心空空如也的人。完全没有任何可以称之为精神生活的东西存在，这也意味着你生命中的好奇、渴望、创意，都被无休止的通勤生活所毁灭。你变成了一个极度乏味，没有任何创见的人，被生活彻底吞噬。


热爱生活，相信未来的人，会在买房之前先买车。同样的道理，把人生视为长程跑而非冲刺的人，会努力保证自己每天的睡眠。如果现代生活意味着我们每一个人都必须是企业战士，那么，每天多睡2小时的战士毫无疑问能够活得更长。因为他有额外的精力去提升技能，打磨战技，寻找战机。钱如果不能用来投资自己，那么无论怎样的节省都是一种惊人的浪费。像我那样拿三分之一的薪水租一处近一些的房子，想起来都觉得愚不可及，但在我看来，那是再划算不过的自我投资。


如果把时间视为一种货币，那么，把时间花在通勤上毫无疑问是一种贬值。这种贬值能增加你储蓄户口上的结余，但是两相对比，你依然是亏损的。因为同样的时间，如果你拿去做的不是通勤而是别的，哪怕是拿去加班，起码也拥有了更多晋升的机会，意味着有了收入提升的潜在可能。十年时间下来，人生将会是完全不同的两个样子。


现在，你还对我8：15起床有什么问题么？
