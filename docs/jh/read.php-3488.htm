中移动平台战略——面向企业应用的MEAP平台

前言

　　对于中移动来说，基于移动通信与互联网融合的市场空间将是未来竞争的重点，其中巨大的企业客户群是全业务运营的着力点。因此，对于期望主导产业链的中移动而言，针对企业的行业应用，特别是移动行业应用开发平台的奶酪不可不动。
不 管是SAP、BlackBerry还是谷歌，都把移动企业应用作为根本性战略，而作为中移动平台战略之一的MEAP平台（Mobile-based Enterprise Application Platform，基于移动的企业应用平台），也于近期推出，面向企业客户和企业管理软件开发商。

　　据了解，MEAP是基于浏览器的移动企业应用开发平台和运行平台，更重要的在于它能很好地把企业应用和移动能力结合在一起，实现普通企业应用实现不了的功能 和应用，例如手机对应用的访问、短信交互、手机定位等。中国移动希望借助于MEAP平台构建企业移动应用的产业链，MEAP平台既能够提供企业急需的移动 应用，同时能够帮助开发者在最短的时间打造全面支持移动能力的企业应用。

　　MEAP平台——面向企业市场的移动企业应用

　　对于中移动而言，MEAP平台最重要的作用将是为移动企业应用提供支撑平台，包括现在和未来的企业应用，并在对外合作上持开放态度，所有应用开发者都可以在MEAP平台上进行开发、部署甚至获利。

　　据了解，基于MEAP平台的“点逸在线”已经在4月上线，包括移动CRM、移动办公等应用，并且支持PC浏览器、手机客户端、WAP、短信交互访问方式，手机客户端目前支持Symbian、Windows Mobile，支持OMS的手机客户端软件也将于近期推出。所有移动企业应用都在统一的MEAP平台上运行，提供统一的用户体验。
 
　　MEAP平台——面向移动企业应用的定制平台

　　MEAP平台除了提供为企业市场提供移动企业应用外，还提供了移动应用的开发环境，从而降低开发门槛。由于MEAP的移动应用开发环境具有入门简单、开发快速等诸多特点，它特别适合支持应用个性化和开发。

　　目前，MEAP平台目前已经实现了超过20种移动企业应用的构建，支持多种应用访问方式，特别是对手机访问应用的能力做到了极致，实现企业应用的随时、随地使用，短信、彩信、手机定位等移动能力也作为MEPA平台的内置能力。同时，对企业应用维护人员和开发人员开放了MEAP定制开发环境，提供应用数据建模、业务规则建模、UI建模、移动交互建模等移动企业应用必需的全方位定制能力。
 
　　一统江山

　　企业客户希望在使用不同的应用时获得一致的使用体验，然而行业应用SI厂商自成体系，技术实现千差万别，各个开发行业应用的SI厂商都有自己的一套技术体系，各个业务产品都有自身的业务操作习惯和业务风格，因此加大了最终客户的操作难度，降低了客户接受度。

　　显然，提供统一的开发标准和统一的用户体验，已经成为MEAP平台的首要任务。MEAP平台更重要的作用则在于能够降低移动企业应用开发门槛，为企业提供更多、更好的应用。

　　以Salesforce主导的AppExchange为例，AppExchange是自定义应用程序发布和共享中心，大量由 Salesforce公司及广大 Salesforce 社区用户贡献的 500 多种应用程序在这里分门别类，任何人都可以浏览、试用、下载以及部署这些预置应用程序及其组件到自己的 Salesforce.com 环境，而无需从头开始自行创建。 用户不仅可以利用 AppExchage 下载、安装别人共享的自定义应用程序，他/她也可以将自己创建的能满足独特需求的自定义程序发布到 AppExchange 站点与全球客户共用或交换。这种高效、灵活、强调分享的应用程序架构使用户可以快速部署能够满足自己独特需求的自定义应用程序，从而极大扩展了标准 Salesforce 平台的应用价值。

　　吸引开发人员和软件开发商加入以MEAP平台为基础的开发社区，从而形成中移动的企业应用发布和共享中心，应该说是MEAP平台的终极目标。
 
　　总结

　　由于统一的企业移动应用开发平台能够提供企业客户统一的用户体验，降低移动应用开发门槛，并最终实现企业应用统一发布和共享。因此，对于期望主导产业链的中移动而言，企业移动应用平台的奶酪不可不动。

　　面向企业应用的MEAP平台，显然将会影响企业市场移动应用的未来。
