黑客产业链模式解读：数据“裸奔”时代如何终结？

今年3Q大战的焦点：奇虎360推出“隐私保护器”软件引发“3Q不兼容”以及诉讼事件，让网络安全隐私观念开始深入人心。然而，12月30日晚间，曾宣称10万年才能被攻破的360密盘产品不足两月便被黑客拿下。黑客已经成为网络安全的重大威胁。

　　网页篡改、上传海量恶意网页、破坏搜索引擎公正性、网站“挂马”或者被挂上恶意代码是黑客常见的攻击手段。

　　目前黑客群体的一个基本特征是：呈现多元化，黑客产业利益链已悄然形成。根据CNCERT的初步估计，目前这条“黑色产业链”的年“产值”已超过2.38亿元，给中国互联网产业造成的损失则超过76亿元。

　　实际上，在互联网领域的“黑客经济”早已有之，但让人惊讶的是，黑客经济已经衍生出了黑客培训、信息窃取、恶意广告、垃圾邮件、敲诈勒索、网站仿冒等多种盈利模式。更可怕的是，黑客们编写的木马病毒程序或黑客行为在当今的互联网市场已经成为“商品”。

　　这种商品的产销模式是：“生产者”负责编写木马病毒程序，“销售商”订制或购进病毒商品而后进行倒卖，“消费者”向销售商购买或预订病毒程序，而后利用病毒工具实施网络攻击。他们主要采用种植“暗链接”以提高目标网站点击率，非法获取他人信息。

　　网站被黑，有超过30%会被同时上传木马或恶意代码，也会让访问者中毒或丢失重要密码及隐私信息;据统计，在互联网上每天新产生的所有网页中，有近5%都是网站被黑后挂上去的，其目的是通过特殊的网页内容来实施搜索引擎作弊。黑马股票排名、彩票预测等网站诈骗案件，许多是通过搜索引擎来显示排名，进而蒙骗受害者。

　　黑客还会帮商家删除政府网上对商家不利的信息。因而，近年来政府网站频频被黑，也侧面反映出，黑客经常性的光顾政府网站。但这并非唯一原因。黑客们还有另外的考量因素:政府类网页搜索引擎给的评重高、网页级别高，黑这类网站易于获得更高的搜索排名、更高的点击率，可以赚更多的钱。而政府网站本身其技术力量薄弱，在黑客攻击面前，往往不堪一击。

　　黑客产业链渐成规模，有人分析认为，网络运营监管的缺失是网络安全问题严峻的原因之一。

　　但从源头上根治，站长建站的初始阶段就应存安全意识，选服务器，买空间域名、筛选IDC建站商家，应从安全性上把关。据了解，国内知名IDC服务商中国诺网从企业全球邮、美国空间、美国达拉斯机房各个环节构筑了站长建站的安全网络，装载了最大的入侵检测系统，可以防大部分DDOS攻击。

　　另外，为回馈站长的支持，近日推出一系列优质安全IDC建站产品秒杀优惠，150M美国空间，秒杀价：280元;G10企业全球邮，秒杀价：640元;500M美国JAVA主机，秒杀价：1000元等各类优惠，以及代理预存返款高达15%，每天仅限5名!时间持续到2011年1月31日。

　　中国诺网www.nuo.cn 表示，对于被黑网站，不仅被非法侵入、占用大量带宽和服务器资源，处理不及时，也会影响网站在搜索引擎中的排名;同时，这种毫无价值、含有恶意代码的网页破坏了搜索引擎排名的公正性，破坏互联网生态平衡，堪称业界毒瘤。而在这一方面，对搜索引擎、开办网站、网店以及域名申请注册的等均要加强审核，对网络安全管理制度进一步完善，才能最终从源头上根治。
