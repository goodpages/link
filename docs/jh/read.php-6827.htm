拆穿那些让人变蠢的“心灵鸡汤”

有毒鸡汤一：“如果你爱她，你应该让她幸福”

逻辑分析：

这句台词，在中国众多的狗血爱情肥皂剧中，被滥用了N多年，通常在剧中被好基友用来安慰失恋的哥们，但是事实上，这则温暖了无数人心的鸡汤疗伤名句，其实是违反逻辑的。

如果你爱一个女孩子是为了她幸福，那么你应该将她介绍给马化腾，因为马化腾有钱有地位，他比起你而言，马化腾更有可能使她幸福，但是在现实生活中，你并没有将你心爱的女孩子介绍给马化腾，恰恰相反，你去追了她，这说明了什么？这说明了：你爱一个女孩子，并不是为了她幸福，而是为了你自己幸福——爱情的本质，就是如此的自私。

为什么鸡汤大师会脑袋犯傻、分不清这一点呢？因为我们的鸡汤大师，搞混了“亲情”与“爱情”这两种人类的感情，将它们俩混为一谈了。

“亲情”和“爱情”，都是“情”，但是二者有着本质上的区别：“亲情”是一种基于血缘关系而产生的感情，你爱你的儿子，为了儿子的幸福，你愿意将儿子介绍给一个异性，而不会有吃醋的感觉。但是，“爱情”却不同，爱情是一种基于“占有”而产生的感情，你爱一个女孩子，你绝对不会将她介绍给更优秀的异性，而是希望将她占有，哪怕你明明知道你自己的条件不够别人好，你还是要参与竞争，为什么？因为你不是为了她幸福，你是为了你自己幸福。

让我们假设一下，假如“如果你爱她，你应该让她幸福”这条鸡汤成立的话，人类社会会变成什么样子？假如它成立的话，那么世界上所有的男人都摇身一变，变成了红娘，每个人都在努力将自己爱上的女孩介绍给更优秀的男人，荒唐不？这样一来，没有人会结婚，也没有人会生育后代，长此以往，人类将会灭绝。

有毒鸡汤二：“如果你觉得身边的人都错了，那么就是你错了”

逻辑分析：

这一则鸡汤在最近几年中国的互联网上被转载N次，欺骗了无数的善男信女，事实上它是错误的，因为它否认了先知先觉者的存在。

人类对世间真理的认识，是一个从0到1，从1到10，一传十，十传百的过程，也就是说，它是一个逐步普及的过程，早期猿猴是吃生肉的，总有第一个猿猴吃上了熟肉，你不能说，因为身边的猿猴都吃生肉，所以第一个吃熟肉的猿猴是错的。

同理，假如上述鸡汤成立的话，那么在清末年代，第一个解放缠足的女孩子就是错的，因为身边的女人都缠足，如果她觉得身边的女人都错了，那么就是她错了，是这样吗？你能这样说吗？

同理，当全世界的人民都认为太阳绕着地球转的时候，只有哥白尼第一个站出来发表异议，他说是地球绕着太阳转，你能说因为身边的人都说太阳绕地球转、所以哥白尼是错的吗？

同理，曾经人类是不吃螃蟹的，为什么？因为担心螃蟹有毒，不敢吃，结果出现了第一个敢于吃螃蟹的人，他吃过之后，告诉人们：原来螃蟹是可以吃的，不但可以吃，而且还很好吃！ 那么，我们能说：因为身边的人都不吃螃蟹，所以第一个吃螃蟹的人也是错的吗？

我想，道理讲到这里，已经是十分清楚明白了，让我们回头再审视一下这句有毒的鸡汤：“如果你觉得身边的人都错了，那么就是你错了”，这句鸡汤不但在逻辑上有误，而且它更是有碍于一个人、乃至有碍于一个民族的创新精神，它鼓吹庸俗，它鼓励人们都当一具随大流、不思考、没有灵魂的僵尸，行尸走肉，一个人信了这则鸡汤，他就是一个庸人，一个民族信了这句鸡汤，它就是一个没有出息的民族。

有毒鸡汤三：“追求梦想，永不放弃，只要坚持到底，就一定会成功”

逻辑分析：

这是在各种明星访谈、名人传记中出现N次的励志桥段，问题是：这碗鸡汤尽管好喝，但它却是有毒的，不少人在这碗鸡汤的激励之下，执迷不悟，走上歧路，甚至是不归路。

问题出在哪里呢？问题就在于：这则鸡汤，否定了“优胜劣汰”的市场规律，它误导人们：优胜劣汰不存在，优也胜，劣也胜，后者只不过是时间问题。稍有常识的人都知道：这显然不符合事实。

就拿娱乐明星为例，那些荣登“鲁X有约”、对观众宣言“追求梦想，永不放弃，只要坚持到底，就一定会成功”的明星，在整个庞大的演艺从业人员群体中，占多大的比例？答案是：万分之一。也许更低。

一将功成万骨枯，在每一个光耀荧屏的娱乐明星的下面，有千千万万个同样努力、但却始终不得志的二线、三线、四线演员，而他们的人数，远远，远远，远远超出了你所认识的明星人数。像王宝强这样能从不要劳务费、只要盒饭的“路人甲”一直混到一线演员位置的，只是极少数之中的极少数之中的极少数。

那么我们到底还要不要追求梦想呢？我们要不要坚持到底呢？而坚持到底，是否就一定会成功呢？对于这个问题，我觉得比较严肃客观的说法，是这样的：有梦想的人，当然应该追求梦想，但是建议你不要蛮干，而是要留意市场给你的反馈，如果市场给予你积极的反馈，那么说明你至少应该坚持一段时间，而如果市场一直给予你负面的反馈，那么说明你在这一行的市场竞争力并不占优，而这个时候，是否选择坚持下去，就是你自己愿不愿意赌一把了，赌赢了，你就可以荣登“鲁X有约”，对全国人民说“坚持到底就是胜利”，而如果你赌输了，你就只能默默服输，悄悄该行了。

一句话，市场有风险，下海须谨慎，“坚持到底就是胜利”这种鸡汤，你读一读，笑一笑就好，不要当真，走一步，看一步，密切留意市场信息，才是一个成熟的人对自己负责任的态度。

有毒鸡汤四：“是金子总会发光，酒香不怕巷子深，桃李不言，下自成蹊”

逻辑分析：

首先，“是金子总会发光”这句话就是一句谎言。金子是不发光的。这个不用我证明，你看看你的戒指，再瞅瞅你的项链，它们发光吗？它们不发光。

其次，酒香就不怕巷子深吗？多深都不怕吗？这个显然也不科学。你的酒再香，也得看巷子有多深，巷子如果是30米深，也许在巷口还能嗅到酒香，而如果巷子是100米深，则无论你的酒有多香，都闻不到。

桃李不言，下自成蹊，也是有条件的，这桃树、李树，得在人们活动的范围内，如果是生在荒无人烟的深山里的桃李，人们步行100里山路才能摘一筐，成本高于收益，这样的桃李没有人要，最终的结局只能是掉一地。

同理，无论你自认为是一个天生的喜剧巨星、大歌星、大律师、大魔术师、大什么都好，你都要离乡别井、向北、上、广、深靠拢，因为只有在大城市，你才有“发光”的机会，无论你多么有才华，你都不应该呆在贵州的偏僻山村、坐等刘备三顾茅庐，你的酒虽然香，但是你的伯乐离你太远，他闻不到，你不应该受鸡汤耽误，从明天起，你应该收拾行装上路。如果呆在家里，你出人头地的几率无限接近于零。

同理，大城市里的剩男剩女们，不要故作矜持，为鸡汤所误，你虽然是金子，但你是不发光的，只有太阳照在你的身上，你才会发光，从明天起，你应该打扮打扮，出去走走，多参加社交活动，多露脸，多认识些新朋友，曝光的机会越多，嗅到你酒香的人越多，你“脱光”的几率就越高。呆在家中，你的桃李，最终只能洒落一地，无人问津。

有毒鸡汤五：富翁对渔夫说：“你应该借钱买条船出海，赚了钱雇几个帮手增加产量，之后再买几条船，办一个捕捞公司，然后将公司上市，你就可以成为亿万富翁，然后你就可以像我一样到海滨度假，晒晒太阳，钓钓鱼，享受生活了。”渔夫不屑地辩驳道：“我现在的生活不就是你说的结果吗？”

逻辑分析：

这则鸡汤想说的话是：有钱和没钱，富裕和贫穷，其实是一样的，富人不要自鸣得意，穷人也不用杞人忧天，其实大家的结局，都是钓鱼，一个样。

事实真是如此吗？

生活是充满意外的，别看这会儿惬意，万一有什么疾病灾祸，富翁和穷人，哪个更能抗？我们不妨脑补一下：

渔夫不听富翁的劝告，安于现状，十年之后，渔夫和富翁同时得了癌症，富翁有钱，到美国找了最好的医生，及时动了手术，将癌症治愈了，康复之后，晒太阳、钓鱼的日子照旧。

而渔夫由于一直只有吃饭的钱，得了癌症之后，被天价的医疗费难住了，100万！治不起！他只好到处找亲戚朋友伸手借钱，可是他发现：亲友的手机都换号了。

怎么办呢？于是渔夫只好等死。临终之前，渔夫躺在病床上，忍受着癌痛的煎熬，他连镇痛药的费用都承担不起，结果，渔夫活活痛死在床上。

这才是富翁和渔夫故事的最可能的结局。这个完整的狗血剧本告诉我们：富裕也许不可炫耀，但是贫穷却绝对不值得讴歌，贫贱之家百事哀，钱不是万能，但没有钱是万万不能，富裕不是耻辱，贫困不是荣耀。

顺便忠告各位：吸取渔夫的教训，趁年轻，买足重大疾病保险，对自己负责，也是对家人、对亲戚和朋友负责。

有毒鸡汤六：“上帝给你一张嘴、两只耳朵，就是叫你多听少说”

逻辑分析：

这则鸡汤的问题在于将“生物学知识”和“人际交往策略”混为一谈，鱼龙混杂，逻辑混乱，和稀泥，混淆视听，误导人民群众。

人为什么有一张嘴、两只耳朵？对此，生物学最合理的解释是：在动物进化的初期，单方向感知声波的动物只能感知一个方向的天敌，而不能感知另一个方向的天敌，因此很容易受到攻击，所以这个分支就灭绝了，而同时能感知两个方向声波的动物则存活了下来，并开枝散叶。说得更明白一些，人有两只耳朵，是动物在进化的过程中，经“优胜劣汰”自然选择的结果。

也就是说，人有两只耳朵，是要告诉你：不但左边有老虎，而且右边还有狮子，你往前跑，才是对的，这才是人有两只耳朵的生物学意义。

在人际交往当中，人应该“多聆听，少说话”，这个也许是对的，但这个只是人际交往的策略，它与你具有多少只耳朵，在逻辑上，并没有什么关系。
