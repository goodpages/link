一次重新爱上数学的机会， 从公式背后的故事谈起

【编者按】回顾数学史，特别是古代世界的数学史，人类研究与学习数学的途径和推理方式各不相同。在这一时期之内，数学逐步进化，脱离了孕育它的学科——测绘、税收、建筑和天文学——变成了一门独立的科学。

在现代世界中，数学是一个高度一致的学科。世界上任何一个国家都会对同样的等式（例如a²＋b²＝c²）得出同样的认识和理解。代数、几何、应用数学和分析作为数学发展的四大支流，相互交融并以最为奇妙的方式相辅相成。

“数学的最大吸引力之一是：一个今天证明为真的方程永远如此，它不受时尚潮流的影响，它放之四海而皆准，它不受审查修改或者立法控制。”

——《无言的宇宙》

《无言的宇宙：隐藏在24个数学公式背后的故事》揭开那围绕着数学和方程的神秘莫测的面纱，而被盖茨先生列入今夏书单的《what if那些古怪又让人忧心的问题》时而让人思考，时而引人发笑，而这一切都将从“数学”这个话题开始。

(左：顾森，右：李淼)

6月27日，腾讯大家携手未读在言几又书店举行科普阅读沙龙。这场题为“数学大爆炸·一次重新爱上数学的机会”的活动，邀请了李淼、顾森两位嘉宾，与读者一同探讨那些不曾发觉的数学之美。

------------------以下是沙龙对话实录-------------------------

【历史上不容小觑的数学公式】

顾森：我举两个伟大的数学定理。一是，根号2为无理数的发现。“大自然的一切皆为整数之比”的思想曾长期统治古希腊数学，直到毕达哥拉斯的学生Hippasus发现，单位正方形的对角线长度不能表示为两个整数之比。数学系统被修改，人们开始重新思考。这是历史上第一次数学危机，对数学的影响和发展特别深远。

另一个例子要跳到上个世纪——哥德尔不完备定理。大致意思是任何一个能够足够强大到表示出自然运算的体系，至少存在一个结论是正确的，但不能被证明。这也是数学家们对数学逻辑非常深刻的思考。人们最终发现，我们使用的数学系统是不完备的，这个系统里面部分正确，但无法证明。同理我认为可以类比到宇宙之中。人类研究宇宙奥秘，其本质是一堆原子在研究自己。正如哥德尔不完备定律一样，身处宇宙内部将永远无法把自己的一切搞清楚。

李淼：《无言的宇宙》一书中反复提及的E=mc²，也是极有用的公式。麦克斯韦方程，是由英国物理学家詹姆斯·麦克斯韦在19世纪建立的一组描述电场、磁场与电荷密度、电流密度之间关系的偏微分方程。它的发现，经由特斯拉、奥斯特等人努力，到了法拉利突然炒出这个概念，最后由麦克斯韦建立。他加入所想，完备了这组方程并发现了电磁波，这非常伟大。它是人抽象出的东西，其真实存在不存在？它非常有效的模式是什么？我觉得人类智力进展是一个非常奇妙的过程，当物理学家发明了微积分这类东西，人们就用上了。

现在，依然是物理学的天下。比如IT行业为什么发展得越来越快？是因为这个摩尔定律，就是结成电路做得越来越小，而且成本越来越低。但是这个摩尔定律也被打破了，也许到那个时候被量子计算机取代了。我本人经历了三次工业革命。我小时候只有自行车，而最小的时候乘坐的火车是烧炭的蒸汽机发动的。后来高铁开始用电，属于第二次工业革命。第三次工业革命也随即到来。有可能在我有生之年，会经历第四次工业革命，与量子计算机有关。

【我们，为什么学数学？ 】

顾森：我们从小到大学数学，都是用现成的共识和定理解题。很多人学到毕业仍旧不知道公式、定理由哪些科学家提出。但事实上，背后的故事反而更精彩。包括很多数学上的结论，其实最开始，可能人们的猜想就是错误的，甚至得到了一个完全相反的结论，到后来才逐步逼近这个真理。这些都是课本上没有的。有很多书纯粹是讲故事，而《无言的宇宙》把很多数学公式的来历和对人类的影响都说出来，这一点很厉害。

对人类的发展意义影响更大的我觉得是物理公式。数学公式其实也不是数学当中最有意义的东西，真正有意义的多数是定理，而不是由什么等于什么。

李淼：我认为即便是文科生也需要跨界来学习。我非常支持高考改革不分文理。一千年前并没有什么分科。我们在人类科学的指导下，分工极度细化以后，科学发展到一定程度以后，一定要回到过去。就是要跨界，不跨界将来你的这个就业的机会和各方面，都会受到极大的限制。所以我非常期待在我有生之年看到，高考不再分科。

顾森：如果说为了考试，那假如取消数学考试，我们还会学吗？我认为数学应该分两个层次，一是小学100以内的加减乘除，学会算账。再往后的解方程，几何结论等等，生活当中可能一辈子也用不上。数学这个玩意儿学到初二就够了，再往上走，或许纯粹出于兴趣爱好。到一定专业的领域想要做一些全新的东西，为人类造福，数学物理才存在意义。

（左：李淼，右：顾森）

【与数学物理相关的“那些古怪又让人忧心的问题”】

李淼：在很多情况下，我们生活中遇到的现象存在不可预测的因素。但话说回来，我们可以知道哪些主要因素在起作用。比如我用物理学解释，人的步行不能超过每秒4米。为什么呢？一个是地球的重力加速度，每秒9.8米，一个是你的腿长大概1米左右，竞走最快的速度就是每秒4米左右。物理学在有一些方面可以进行简单计算。

顾森：用理论去解释生活当中的实际问题，跟生活相差甚远。但我们会建立一些数学或物理的模型去计算一下我们应该怎样做才能做得更好。分析事情的乐趣常常大于实际意义。这也是What If一书有意思之处。

【用一颗包容的心重新看待数学的美】

李淼：我是学物理的，在40岁以前，经常没事找几何题自己证明一下，这个过程非常享受。其实物理学的美，80%、90%的程度跟数学的美是一样的，在数学中的一条定理，在物理学中就是对问题的一个描述。比如我告诉你人类的走路，不是跑步，因为跑步会用到肌肉或其他的，你走路不可能每秒超过4米。物理不要求精确，数学要求的是精确的。比如从量子宇宙学的角度来看，这个世界为什么我们的宇宙在膨胀，而且会加速膨胀？这是一个例子，但不是数学的例子，它是一个纯物理学的美。它在某种程度上是收敛的，在某种程度上可以表达出物理学的美。我觉得《无言的的宇宙》这本书提到的24个公式，有一多半和物理学相关。

但是我现在在写诗，里面有个人的爱好。但是归根结底，我们在聊的是一个理想状态，人的理想状态。比如说，你不喜欢数学，你也得会背乘法口诀，有一些事情也是必要的。我觉得学习数学可能会让你将来的发展是更丰富多彩的。有一个当代知名的诗人，牛津大学毕业的，他同时是一个古代希腊的文明的专家，他在一本集子里面——一部希腊经典作品，里面有悲剧、戏剧、诗歌，他写了一个前言。他说，古希腊是世界上存在最文明的人类，因为一个运动员同时是诗人，同时是哲学家，一个军事指挥官，一个将军同时是哲学家。我们要有这种宽容心和对多样化的存在的一个包容心。

顾森：在理想状态下，其实，各个学科都有它的一个美，而且这些美，我觉得是相通的，最后任何两个学科学到极致之后，都有相通之处。如果真想喜欢数学，有很简单的数学现象都可以让你感受到数学之美。有时候真正的数学之美，可能更多地体现在数学学科本身。以一个很有趣的物理学的小知识为例，物体抛出，用某种方式可以很稳地接住，但如果这个物体两个轴都能在空中很好地旋转，即使非常端正，它也无法稳定旋转。这背后有很专业的解释，但你可以发现物理学的有趣。多观察，多看，你会发现数学物理的趣味。
