和大鳄怎么斗？ARM用独特的商业模式，给大鳄设置了一个壁垒

“英特尔的商业模式老套。ARM的商业模式则属于21世纪。”九成以上的手机芯片，都含有ARM公司的技术。其设计出的芯片内核蓝图（一种程序），非常省电，适合用在便携式数码产品中，每年可以从40亿个芯片中获得数亿美元的权益金收入。为什么英特尔这个半导体大鳄没有进入这片市场？原来ARM的商业模式很独特，它只向芯片商（如中兴通讯）收取非常低微的“授权费”，这就在这一片市场上，给行业大鳄设置了一个高壁垒。



ARM(Advanced RISC Machines)是一家英国芯片公司，用它的内核设计出的芯片，耗能非常低，这使它在手持设备成为消费潮流的年代非常醒目。

　　全球销售含ARM技术的芯片数量太庞大了，2008年就有40亿个！全球95%的手机里至少有一个含ARM技术的芯片，它占了40亿个芯片中的65%。另外，很多数码相机、MP3播放器，甚至数字电视中，都有这种芯片。

　　为什么这家ARM公司能占有这么大市场呢？我们先要了解一下它赖以生存的RISC技术。

　　在最初发明计算机的数十年里，处理器都采用CISC指令系统。随着计算机性能日益增强，内部元器件越来越多，指令集变得日趋复杂，影响了计算机的效率。

　　研究发现，计算机中80％的程序只用了20％的指令集。于是“RISC精简指令集”这个概念被提出来，它抓住“CISC复杂指令集”的缺点，通过减少指令种类、规范指令格式和简化寻址方式，大幅提高了处理器性能。

　　这个RISC比较精简，就像自行车，在走街串巷这种活动中很灵活，不会有太多能耗；而复杂指令集CISC则像卡车，其目标是在干线上进行大吨位货物的运输。

　　英特尔的处理器，使用的就是复杂指令集CISC，其处理任务的能力强，在桌面处理器领域大行其道。

　　而ARM采用“精简指令集RISC”，体积小，是公认最省电高效能的方案，适用于手机、数字电视、iPod、数码相机、PDA、上网本等这些便携式电子产品。


ARM在与英特尔等的竞争中，找到了很好的定位。

　　上世纪80年代初。剑桥大学一群师生创立了橡果电脑公司Acorn，决定设计一种能充当下一代机器的数字中枢的处理器。这种芯片就基于“RISC精简指令集”。

　　但英特尔、德州仪器等巨头进入个人电脑市场，对橡果形成很大冲击。“当时我们想，每家公司都有自己的CPU，如果我们开发出一套能重复使用的核心程序码，帮想做CPU业者简化设计过程，不是很好吗？”ARM的创始人之一图多·布朗回忆说。

　　于是在1990年，布朗等十几名工程师从橡果拆分出来，并由橡果、苹果电脑和VLSI技术公司共同出资成立新的合资公司—ARM，最初资本只有400多万美元。

　　此时，在RISC结构方面的竞争对手都着眼于提高性能，ARM则聚焦成长最快的手持式设备市场，注重省电节能，成功地和在PC微处理器市场占优的英特尔进行了市场区隔。


ARM的商业模式是这样：将研发成果授权给芯片公司或生产商，先收一笔高达数百万美元的“授权费”。之后如果这些公司用ARM技术研发出芯片，每销售一个芯片，再按比例付“权益金”。

　　每个产品的权益金，通常也就几美分，但这是源源不断、边际成本几乎为零的收入。2008年，ARM收入5.5亿美元，其中权益金2.7亿美元、授权费1.9亿美元。

　　三星、诺基亚、IBM、英特尔、高通等全球主要半导体公司、芯片设计商和制造商，都向ARM买授权，然后再在其技术的基础上，添加自己的设计并推出芯片产品。最后再由下游的代工厂生产出芯片。


2000年，ARM又对其授权模式进行修改：针对不同客户，收取不同费用，以向小型半导体设计公司渗透。

　　这不仅增加了收入来源，还使ARM系统快速成为产业标准。全球GSM手机80％已采用ARM规格处理器，平均每个手机会使用1.9个含ARM技术的芯片，智能手机如iPhone使用得更多（而且权益金是普通手机的6倍）。

　　接着在2005年，ARM又推出一项极赋创意而大胆的商业模式：将部分处理器技术放在网上，供芯片设计公司免费下载使用。

　　芯片设计公司可以在芯片量产的时候，再缴纳“授权费”，这就大大降低了它们的风险和成本。

　　这是否是一种冒险行为呢？

　　“那些用户，需要在网站上进行登记并经过我们审核后才能下载，而且下载前需要承诺保密协议。”ARM中国区总裁谭军说，“芯片设计公司从网上免费获得的技术，只能用于设计和开发，若要进行量产，还必须从ARM公司获得完整的授权。ARM与可以投产的代工厂签订了一个第三方协议，芯片设计公司如果在投产时拿不出ARM的授权书，代工厂将拒绝进行投产。”

　　值得注意的是，在ARM的收入来源中，“授权费”的比重正在逐渐下降，“权益金”的比重却逐年增加。

　　这意味着什么呢？“授权费”下降（固定费用下降），有利于吸引厂商使用其规格，扩大市场占有率；而“权益金”比重的提高，意味着将视客户产品销售量的高低，而收取变动的收入。这有助于减少客户的经营风险，形成共赢格局。


ARM轻资产的商业模式，让它不必从事生产，不用承担生产成本。一个好处是，其毛利率达90%（研发费用占总收入近三分之一，使其最终净利润只有15%）；另一个好处则是，芯片厂商、供应商们都不会将其视为竞争对手，可以毫无阻碍地进行合作研发。


　　特约读者戴巍点评（戴巍为英飞凌半导体手机通信工程师）：无可否认，ARM公司在“便携式芯片”IP核中取得统治地位，是靠核心技术赢得市场的典型公司。由于不涉及生产及市场营销之类的高成本流程，这种公司的财务报表，非常受股票市场推崇。

　　但其潜在的危机就是：1、由于只提供技术，而没有生产服务等项目，ARM公司必须保持其技术的绝对领先，否则一旦技术上落后或战略错误，其公司很快就会陨落—毕竟MIPS、英特尔等公司一直等着ARM出错呢。


2、由于只提供技术，ARM没有自己的芯片厂，即没有芯片渠道。现在“便携式芯片厂商”还比较多，渠道比较丰富，但手机等产品降价很快，导致芯片也必须降价。芯片渠道利润下降，并购定会发生。当芯片渠道整合成两三家时，其和ARM公司的议价能力也会变强，会反噬ARM的利润，毕竟渠道为王有一定的道理。所以觉得“英特尔投资公司”的作法值得ARM效仿，即积极投资下游市场公司，加入离消费者最近的行业，熟悉市场并引导市场与公司的设计战略发展一致。
