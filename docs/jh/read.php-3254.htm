ARM模式不败神话：联合企业超越Intel

第1页：“Wintel”的垄断地位开始瓦解

　　ARM确立了独特的商业模式，作为与美国英特尔公司不相上下的半导体产业的核心企业，该公司的地位在不断增强。半导体MPU（超小型运算处理装置）可以说是数字产品的心脏，而设计MPU的企业正是ARM。

　　虽然该公司2010年的销售额才不过4亿660万英镑。但对于美国苹果、美国谷歌、韩国三星电子、日本索尼以及任天堂等全球著名IT企业来说，ARM的存在已经不可或缺。例如，在市场持续迅速扩大的智能手机和平板终端用MPU领域，ARM占有了9成以上的份额。Gartner Japan的首席分析师山地正恒指出，“由于ARM的MPU耗电量较低，适合用于智能手机、平板终端以及游戏机等便携终端，因此扩大了份额”。

　　“Wintel”的垄断地位开始瓦解

　　便携终端市场是一个快速增长的市场，美国苹果、谷歌和微软（MS）在该市场上不断进行着激烈的份额之争。在胜败的反复交替中，唯一不变的就是无论胜者是谁，ARM都不会成为败者。

　　ARM的势头甚至会改变长期以来支配IT产业的“Wintel”。Wintel一词取自个人电脑操作系统（OS）“Windows”和提供核心硬件MPU的Intel，这是一个象征着两公司“亲密无间”和在IT市场上寡头垄断的词汇。

　　但在2011年1月，微软宣布其新一代OS“Windows8”除英特尔的产品外，还可在ARM的MPU上运行。英特尔之所以强大，主要在于其对支持Windows的MPU的技术垄断。而现在，有一半的份额被ARM夺走了。

　　起因是采用ARM公司MPU的iPhone和Android终端开始进入微软曾经“独霸天下”的个人电脑市场。微软判断，要想通过最早估计会在来年上市的Windows8进行反击，就必须与ARM合作。

　　第2页：ARM已经在几个方面超过了英特尔

　　ARM首席执行官（CEO）华伦·伊斯特（Warren East）对此表示欢迎，他认为此举“铲除了涉足个人电脑市场的障碍”。ARM计划以涉足个人电脑业务为契机，进军要求更高性能MPU的服务器领域。美国调查公司IDC预测，采用ARM公司MPU的个人电脑全球份额在2015年之前将扩大至13%。

　　目前，ARM已经在几个方面超过了英特尔。例如，ARM面向智能手机、平板终端、家电、游戏机以及汽车等多种用途设计的半导体每年约供货61亿个。而根据Gartner的统计，英特尔的供货量为3亿2000万个左右，其中大部分面向个人电脑和服务器。营业利润率方面，英特尔为32.1%（2011年4～6月期），而ARM为44.5%（2011年4～6月期）。虽然英特尔也开发出了低耗电量MPU，并面向家电和便携终端销售，但ARM的地位目前并没有受到动摇。

ARM “巨人”

　　那么，ARM为何能在与销售额高达436亿美元的巨人英特尔的竞争中取胜呢？ 答案是ARM构筑的特殊的商业模式。

　　ARM日本法人的社长西岛贵史介绍说，“ARM的周围存在数十万亿日元的业务”。目前被称为SoC的片上系统，除了MPU外，还在一枚芯片上集成了负责存储器管理和通信等多种功能。ARM就在从事其中的MPU设计。

　　面向家电厂商和汽车厂商供货的SoC是由半导体厂商为其附加MPU以外的各种功能后开发制造和销售的。ARM的收入来源是向各厂商收取授权费，以及根据半导体供货数量收取的每个数日元至10日元不等的专利使用费。处于核心位置的ARM虽然很小，但通过将周边的半导体厂商都裹卷进来，产生了巨大的离心力。

　　另外，从财务方面可见的特点是，ARM通过不自行生产的“无厂经营”方式。这是高收益的原因所在，但如果单就没有工厂这一点而言，从事半导体设计和开发，而把生产委托给代工企业（受托生产公司）的无厂半导体厂商也是同样的模式。只不过，ARM从事MPU设计这一核心业务，通过授权业务，实现了更加领先的商业模式。

　　第3页：ARM的业务优化

　　如果换一种方式来理解ARM的业务，那就是有多达250家半导体企业将MPU设计外包给了ARM。与各公司分别开发相比，委托给ARM统一开发会更加节约成本。

　　ARM在半导体厂商的客户家电厂商中也非常受欢迎。半导体上运行的软件的编写方法由MPU决定。从2010年秋季开始在产品中采用ARM公司MPU的富士通半导体执行董事井上表示，“采用ARM公司的MPU后，客户企业反映"由于技术人员对此熟悉，所以软件开发变得更加容易"。

　　采用ARM的MPU后，还有望实现保护效果。因为，如果想更换MPU，就必须改写之前编写的所有软件。而如果继续使用ARM的MPU，可以直接利用原有软件，优势非常大。

　　此外，ARM的商业模式还可以作为反收购策略，这一点比较有趣。2010年媒体传出了苹果欲收购ARM的报道，致使ARM股价上涨。不过ARM的CEO伊斯特推翻了这一报道，他表示，“谁都没有收购ARM的必要。这是本公司商业模式的意义所在”。

　　这里可能需要稍作解释。ARM向签订授权合同的半导体厂商提供所有需要的MPU设计信息。ARM从半导体厂商手中获得的是“数亿日元的授权费。而且作为专利使用费只有半导体价格的1～2%”（伊斯特）。但ARM的总市值超过了1万亿日元。如果苹果看上的是资本收益那就另当别论，但如果苹果只是想要技术，还是签订授权合同要划算得多。

　　这种特殊的商业模式究竟是如何形成的呢？追本溯源，这要追溯到曾经的教育用途计算机厂商英国Acorn Computer公司。

　　20世纪80年代中期，ARM作为Acorn公司的MPU开发部门而诞生。但当时个人电脑市场由美国IBM和苹果掌控，Acorn的经营日渐萧条。用于MPU开发的资金和人才都不足，因此只能尽量以简单的标准进行开发。这种状况催生了ARM拥有开发简单、低功耗MPU的基因。

　　这期间，Acorn注意到了为开发便携式信息终端而需要低耗电量MPU的苹果公司。1990年，在Acorn、半导体厂商VLSI Technology以及苹果的共同出资下，ARM从Acorn中独立出来。如今在剑桥郊外仍能看到当时的创业地点一间马房。

　　布朗社长回顾创业时的情景表示，“我们身处剑桥，拥有的半导体业务经验非常少。正因为如此，才形成了这种（崭新的）商业模式。如果是在美国硅谷创业的话，也许最终ARM只是一个普通的无厂企业”。

　　不过，这种商业模式并不是在ARM独立后立即就运转顺利的。虽然苹果的信息终端等采用了ARM的MPU，但迟迟未能畅销。

　　情况开始出现转机的时机是在手机实现数字化后。1997年，ARM的MPU被芬兰诺基亚的手机采用，并成了畅销产品。“除授权收入外还能获得专利使用费，业务范围得到扩大”（布朗社长）。

　　如今步入了从手机到家电，乃至汽车都需要联网的时代。需要进行数字处理的产品不断增加，因此必然要使用半导体。据统管便携终端业务的主管劳伦斯·布莱恩特（Lawrence Bryant）介绍，“在ARM公司MPU的产品之间，软件能顺利联动。掌控着智能手机市场的ARM公司的MPU将进一步加速普及”。

　　第4页：ARM与厂商共谋

　　在与厂商的共谋发展

　　配备ARM公司MPU的SoC供货量在2010年约为61亿个，今后会进一步增加。如此之大的数量，仅凭一家公司终归无法生产，只有将其分散到众多企业之间，ARM的业务才能顺利推进。这自然而然地形成了可以称得上是“ARM经济圈”的一大网络。

　　ARM的成功还对处于同一经济圈中的半导体厂商的业务产生了巨大影响，从事个人电脑用图像处理半导体业务的美国英伟达，现在还在涉足用于智能手机、平板终端和车载信息终端的半导体业务。随着Windows8对ARM的支持，该公司还将进军个人电脑用MPU领域。以通信、半导体为主力业务的美国高通也计划进军个人电脑市场。

　　对于日本企业而言，ARM的崛起则是各企业走向世界的机会。松下半导体公司系统LSI业务组长冈本吉史坦率地表示，“在我们宣布采用ARM的MPU开发电视用途高性能SoC后，来自海外企业的咨询便出现了增加”。

　　据说真核生物细胞中的线粒体曾经是单独的生物，后来在不知不觉间被细胞吸收，形成了线粒体和细胞之间相互依存的共生关系。就半导体厂商涉足SoC、在相互依存的同时又不断变化，最终共同发展壮大这一点而言，ARM与真核生物相似。而且，ARM经济圈不断扩大的潜力目前还看不到终点。
