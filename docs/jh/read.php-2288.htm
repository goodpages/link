智能化是视频监控的必然趋势

1. 海量视频监控系统如果不加视频分析将只是“可视”而不能有效的“监控”
　　现阶段安防的主要任务是大量兴建视频监控，无论闭路电视监控还是网络监控，都主要是解决“有”监控的问题。但是在每个单位、每个城市都安装视频监控后，如何管理这些视频，如何让这些监控系统发挥效力则是下一阶段的重点。
　　智能分析设备还没有成为必装的设备。从布控到报警再到应急反应，都是用肉眼盯住电视屏，即所谓“人眼盯住电子眼”。
　　这种不加分析的监控模式被大量的事实证明是不可靠的：
　　（1） 人眼易疲劳：科学实验证明，人眼盯住目标超过22分钟就会走神，况且长期面对一个大屏幕甚或一堵电视墙，会非常枯燥，麻木；
　　（2） 人眼视角非常狭窄：人的有效视角只有20－30°，视觉敏锐视角仅10°，超过了这个限度，目标出现就只有用“余光”，非常不可靠；
　　（3） 从人眼看见目标到判断出“有情况”再到用对讲机喊话有一个时间差；况且人不是机器，面对着几十个屏幕组成的电视墙，非常容易疲劳，此外人也会有吃饭、如厕、打瞌睡的时间段，这个时候出现情况，监控会出现“空档”；
　　（4） 特别是，在夜晚，海面，风、雪、雨、霜等恶劣的自然环境下，目标即使出现，人眼也难以判别。此时的视频监控，如不用视频分析技术识别，形同虚设；
　　（5） 这种模式只能事后取证而不能事先布防和事中处理：不可否认，事先防范，预防犯罪和事中快速反应，始终是安防的首要任务。如果不能在新上视频监控之后配套装置智能视频分析设备，则会使大量资金建设起来的监控系统形同虚设，用眼睛日日夜夜盯住显示屏的人，就会成为机器的奴隶。
　　我们看到有许多案例，盗贼得手的当时没有被发现，只有事后翻看录像，即便事后取证，也难以挽回损失，监控一旦失去事先防范和事中识别处理，必然成为“事后诸葛亮”。
　　这就形成了一个奇怪的现象：
　　
　　2. 智能化是安防数字化、网络化之后的必然趋势
　　 由于有这些必要性，我们有必要给电子眼装上大脑，即大量运用智能视频分析技术来识别海量视频中所含的目标及其信息。
　　智能视频分析是一种人工智能，是“所见即所知”的技术。其核心在于计算机视觉（Computer Vision）和机器学习（Machine Learning），简单来说就是让机器（电脑）能像人一样接受训练，克服人眼的缺点，辅助人来判断目标、报警和采取行动。
　　形象地说，智能视频分析是一种从大量视频中寻找出大概率事件的出现的规律，先教会机器（电脑）识别，然后由机器值班，“指认”出视频中出现的目标。
　　为什么说安防数字化和网络化之后必然是智能化？因为数字化和网络化带来了海量视频和堆积，并且被不加分析地无意义记录（不停顿录像），会造成是一种极大的浪费、污染甚至公害。所以智能化处理，将视频信息加工、标签、整理和驱动就十分必要。
　　
　　3. 视频监控只有经过机器（电脑）分析才能有效率，才能既监又控
　　为什么这样说？
　　我们以贝尔信bellsent智能分析编码服务器为工具，先分析一下视频监控在每个阶段的作用。
　　（1） 辨别目标并打标签：在前端，视频采集中，bellsent智能视频分析设备能通过像素变化，在滤除了环境干扰后能即时发现人和车辆这二类目标，并用不同颜色的方框和拖尾来分别标识目标大小和移动轨迹。
　　（2） 预警布防：由于能即时发现闯入目标，bellsent设备能自由设定预警区域即防区，从而能让机器值班，在重点区域张网以待，从而降低了对于人眼的依赖性，让监控系统时刻处于“备战”状态，极大地提升了安防等级。
　　（3） 报警处置：目标闯入时，bellsent设备能以声光电、短信息、截图发送彩信等方式实时报警；特别是其预留了双向语音接口，可通过交互方式发出语音询问，警告闯入者已经被发现；特别值得指出的是，bellsent设备预留了GIS信息接口，可警示目标闯入的对应位置。
　　（4） MPEG-4压缩编码，数据打包传输：所以bellsent的设备全称应当是《智能视频编码服务器》
　　（5） 事件触发提前录像：ioiamge设备内置了虚拟录像软件，可事先设置由事件触发提前5秒预录像，并全程硬盘录像。
　　（6） 驱动PTZ摄像机自动接力跟踪：这是bellsent在全球独一无二的智能功能——通过在后端手动点击目标或由目标闯入防区自动触发标识并跟踪目标，让目标在防区内被锁定和接力跟踪。
　　（7） 克服人眼盲点：在低照度、高噪点，野外恶劣条件下，bellsent设备能忠实地坚守监控岗位，灵敏地发现目标。
　　
　　
　　4. 提高侦测率，缩小误报率是智能视频分析的重点和难点
　　侦测率与误报率是一对如影相随的矛盾。一种好的视频分析技术既不能漏报，也不能乱报。否则，如果有情况，设备频频漏报，用户就会感到用的不放心；反之，设备“神经过敏”，没有情况却乱报一气，只会变成扰民、烦心的工具，则最后必然被下岗。
　　
　　5. 目前智能视频分析还只是初级人工智能，只能辅助而不是代替人工监控
　　现阶段我们对智能视频分析还处于计算机视觉从算法走向应用的初级阶段。形象地说，现在的智能水平只能相当于一个2、3岁的小孩的智力，只能认识“有人和车辆来了”，“有人丢包了”之类非常简单的事件。至于需要仔细辨识的街头偷盗、打架、超车之类的复杂行为，则尚未开发出来。
　　技术需要积累，产品需要试错，指望智能视频分析去代替人脑、完全取代人眼，解决一切监控问题还不现实，至少还需要一段相当长的时间。
　　bellsent公司的设备目前只能检测出特定区域内的人和车辆的6个行为模块：目标闯入，徘徊确认，取走物品，放置物品，非法停车，自动PTZ摄像机跟踪。
　　此6个模块都是完全成熟、可靠的，bellsent智能视频编码服务器首先应用在青藏铁路工程，四川省数十所监狱和近200家看守所防越界逃跑改造工程，《远望》号科学考察船，深圳博物馆新馆，深圳海关，上海浦东机场，河北某国营煤炭集团井下作业工程，南京地铁工程，武警北京总队，总装防化品仓库等重大项目。
　　
　　6. 提高智能视频分析产品的性价比是规模化应用的前提
　　 不可否认，由于智能视频分析产品进入市场很晚，其价格 与用户目前可接受的心理价位还有一定距离。与同类的网络视频服务器、编解码器相比显得很“贵族”。
　　 但任何产品的价格都与其应用量成反比。有理由相信，随着智能产品在各行业各领域的广泛应用，其价格也会越来越平民化，被应用到寻常百姓家。
　　 可以预料，在突破了价格瓶颈后，bellsent等智能视频分析产品必将成为视频监控领域的必装产品。
　　
　　7. 系统兼容性是智能视频分析结合到现有监控平台的必需前提
　　 能不能给摄像机装上大脑，关键在于已有视频监控系统与新上的智能系统是否能兼容。
　　 为了解决这个问题，深圳贝尔信科技有限公司自主开发成功了终端客户控制中心的监控管理服务器Monitor&Management Center，简称MC。该系统能够实现传统的监控设备和智能视频分析设备集成在一起实现集中管理。MC目前可以集成以bellsent为主的多家智能视频分析产品，后台的录像存储支持海康威视、大华等的主流DVR。通过MC，可以轻松实现对智能视频分析设备的布防、撤防、清楚报警、功能设置、分屏浏览等基本操作，也可以实现GIS报警显示、报警触发录像、报警自动弹出画面、语音报警、自定义日程与事件、报警日志查询等高级功能。使用户通过MC友好、易用的人机界面，轻松实现对智能监控设备和传统监控设备的集中管理,迎接智能监控时代的到来。
