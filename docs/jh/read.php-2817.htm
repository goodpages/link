磊友CEO黄何：愿做荒山掘金者

他们有一个形象的名字——金种子。他们诞生于去年12月，在全国关注小微企业发展，北京转变经济发展方式的大背景下，他们来得及时，来得令人惊喜。他们从中关村的两万家企业中精挑细选，他们定位于高科技，他们聚焦于战略性新兴产业领域。

　　为了金种子，中关村制定了相关政策，确定了扶持流程；为了金种子，中关村认定了一批创业导师，给予创业辅导与培训、空间拓展与选址、项目推介与融资等专业化服务。中关村要给予金种子最大的帮助。不但有沃土，还将有持续的阳光、水分与营养。

　　他们的到来也恰逢其时， 在金种子之前，中关村已有两个工程——“十百千工程”和 “瞪羚计划”，分别着眼于大企业和中型企业，助力他们做强做大和高成长。如今，中关村铺垫了一条初创企业由小变大的发展快车道。

　　中关村如此不遗余力地聚焦金种子企业，不仅仅因为这是中关村的责任，也不仅仅因为这些企业的技术水平高、商业模式新、持续创新能力强、创业团队和治理结构完善。

　　最重要的是，这些企业正是中关村的未来与明天。

　　创业优先选择三类人

　　初见黄何，你看不出他和别的“80后”有什么不同，但是在大多数“80后”还在为做什么工作好而烦恼的时候，他已经开始创业了。

　　2004年大学毕业后他成为互联网大军中的一员，多年互联网行业的打拼，让他成为了腾讯管理层中的一员，这是大多数“80后”还在憧憬的梦想，他已经提前一步完成。2010年中的一天，他突然间觉得好像没什么可以干的了，就这样一辈子打工吗？自己陷入一种突如其来的恐怖中。

　　2010年6月，迷茫中的他辞去了工作，回到家中陪妻子，过起了家庭妇男的生活，做饭、睡觉成为他生活的重心，也有了时间可以思考自己以后的路。

　　这时，他特别好的一个朋友、一个女孩子，自己去创业了，这事给他的冲击很大。“一个女生都敢创业，要不我也创业吧。”想到就做，他跟媳妇儿商量，媳妇儿的一句话坚定了他的信心“那你去吧”。家庭妇男的生活只持续了短短的两周时间，就被创业的忙碌取代了。

　　他开始有意识地接触一些对自己有帮助的人。当知道他有创业的想法后，一些风险投资开始联系他，希望能有合作的机会。那时的他还对外界抱有戒心，而来自创新工场的一次主动邀请，让他坚定了创业的信心。他选择入驻创新工场，开始了他的创业梦。对他来讲，创新工场最大的帮助是让他知道了怎么开始创业，也是在那时结识了公司现在的首席CTO赵霏。

　　创业首先需要组建自己的团队。关于团队，黄何有自己的一套人品论：“创业初期选人特别重要，关系到整个团队的稳定。初期选定的人是不是能跟你一直做下去，只能看人品。人品好的话，这些人就能一直跟你到最终，人品不好的话都走光也有可能。所以，最早找创业伙伴有时候真的不用考虑太多，就看人品，就看命了，能找到特别好的一起做，也是一种运气，没有办法去挑选特别合适的。”

　　黄何告诉记者，刚开始创业时他的团队优先选择三类人：第一单身；第二特别熟或者认识多年，其父母同意和他一起去创业的；第三刚毕业的学生或者师弟师妹。当时他的创业伙伴要么就是年纪特别大，要么特别年轻。

　　2010年8月，他们获得了创新工场的天使投资，9月磊友在创新工场成立。移动互联网的快速崛起，让他们看到了机会。仍处于完善之中的HTML5，被他们作为支点，未来也许他们会撬动地球。

　　做事应选对时间

　　“任何一个行业在发展过程中，企业进入的时间点不同，就要做不同的产品。如果你做的东西太超前的话，可能需要很长的时间才能用到，企业可能等不到那天就已经破产了。”这是黄何自身经历的感悟。

　　公司刚开始运作的时候，大家一致认为HTML5一定会火，而且还有一个很美好的愿景，“有那么多低端手机，如果能在低端手机上支持游戏或者应用的话，将是一个多‘牛’的事情”。公司做的第一件事情就是做可以在JAVA、塞班上运行的HTML5，8个月的努力，终于获得成功，但突然发现很尴尬。做了一个平台，却根本没有人为低端手机开发内容，完全是在一个不合适的平台上做了一个完全没有人要的产品。

　　发现问题后，公司开始拼命转型，完全放弃了当时的低端产品，开发高端品牌iPhone、iPad、Android等。幸好转型速度比较快，让公司渡过了第一次重大危机。

　　成功转型后，公司做的客户端软件和工具，市场反响很好，很短的时间就有了上千万的用户群体；但就在这时，另一个让黄何尴尬的事情出现了，虽然用户多，但是没有愿意付费的。黄何开始将目光放到了游戏上，在他看来，“游戏的钱是无法控制的，就算不想赚钱，用户会觉得这个游戏里面什么都不卖，太不可思议了。如果创业做游戏，会发现游戏真的赚钱要比其他容易多了”。

　　黄何给记者做了一个比喻，“如果把基于HTML5的游戏当成是一个金矿，我们最早做的就是在生产挖矿的铲子。做好了一套铲子，却发现人们觉得太贵都不买，直接拿着锅碗就开始挖。我想那就自己招一批人开始挖，反正我已经生产出了更专业的铲子，根据自己的实力招一百、一千的人开始挖，等我挖出金矿，他们就会去想这个铲子挖得很快啊，看到效果就会来买这个产品了”。

　　今年2月10日，磊友发布了全球首款完全以HTML5技术开发的大型多人在线战略网页游戏《黎明帝国》；6月19日，第二款基于HTML5技术开发的角色扮演类手机网页游戏《光辉战纪》开始封测。它实现了不同的设备可以一起玩同一款游戏，不论是Android，还是iOS，亦或是Windows Phone 7，都可以共聚一堂。

　　庆幸入选金种子工程

　　每个企业发展都不会是一帆风顺，磊友也一样。2011年春节将至，刚从创新工场毕业的磊友，却为年关难过而苦恼。虽然公司已经获得上千万美元风险投资，但是资金却迟迟无法进入中国境内银行账户，眼看手头所剩的资金就要告罄，急需一笔钱来渡过年关。这是一个很尴尬的现象，“谈好了融资，但是钱还没到账户上，公司却有可能先挂了”。

　　就在这时，磊友成为北京中关村“金种子工程”首批示范性科技企业之一，得到了一笔扶持资金。金额虽然不大，但是却支撑着公司渡过了年关，并顺利地引入了风投资金。

　　“企业越早入选金种子，得到的帮助就越大，企业初期发展很困难，需要的帮助有很多，钱、政策、法务、人等等，金种子工程可以帮助解决一些很现实的问题。我比较幸运，在最困难的时候入选了金种子工程，它对我的帮助真的很大。”这是黄何对金种子的最直观的认知。

　　在黄何看来，HTML5的发展还需要多种因素的推动。首先是浏览器，必须要有好的浏览器才能支撑游戏的运行；再有，这个市场必须要形成一个成熟的产业链结构。就是有人生产游戏，有人卖游戏，最后这个钱大家一起分。有了产业链，这个市场才能成熟；还有HTML5相关技术也需要不断完美。比如说开发者对于HTML5技术本身的理解或者是我们能做一些切合实际的事情，也会加速它的发展。

　　磊友就是“第一批吃螃蟹的人”，黄磊认为，“我们现在做的，就是在这个行业什么都没有的情况下，用自己的力量去撼动一下，先不去想未来有什么，先把这扇门打开，让更多的人认识这个行业，从而加入这个行业，一起去促进行业的发展。其实我更多地希望有大公司进来推动。因为一个新技术、新领域不是小公司可以推动的。希望Google、腾讯等巨头加入，一起来推动行业的发展”。

　　漫漫创业路，黄何正在一步一个脚印地不断前行。

　　商报记者 韩琮林

　　一个“80后”的创业者，创业理念务实而又深刻，谈话间不断地夹杂着“赚钱是我的终极目标”、“做公司的本质是追求利润”等。他就是北京磊友信息科技有限公司（以下简称“磊友”）CEO黄何。

　　名词解释

　　是用来制作网页的标记语言，它可以在不借助插件（如Adobe的Flash）的情况下在浏览器中播放音频和视频等。HTML5 将成为 HTML、XHTML 以及 HTML DOM 的新标准。

　　2011年底， 两个重要的网页开发工具Flash及Silverlight先后宣布停止移动互联网领域的研发和更新，确立了HTML5在移动互联网网页开发技术领域的龙头地位。HTML5将迎来一个高速发展的阶段，甚至由此引发一场更为深刻的互联网革命，基于HTML5技术开发的Web game也将成为移动互联网游戏时代的宠儿。
