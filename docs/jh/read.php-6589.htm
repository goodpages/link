一位小创业者血泪史：培养过很多技术大佬，但我还在发传单

猎云注：在互联网这个高度竞争机会稍纵即逝的行业中，不少人抓住了机遇一鸣惊人，而更多的是失意者。以下为嫁拍网 CEO 陈晓峰的一篇反思文章，从中回顾此前在百度担任某业务负责人先后带过的“小弟”，如今却成了众多公司的中坚力量，此外还包括离开百度做风车网之后屡屡错过的机会。如今，他创办的嫁拍网还在初始阶段，获得了 300 万美金 pre-A 天使投资，他也在大街上散发传单，为公司发展拼搏。

　　以下为全文：

　　当 35 岁的我被老板催单子、站在阜成门地铁站门口发传单的时候，我的人生跌入谷底。我怎么会混到这个地步？

　　清华毕业后，我做过百度高级经理，在人人网主持过风车网旅游项目，带领过 360 搜索引擎的开发。我带过的手下都已成为各大互联网公司的大佬，公司累计估值超过 100 亿美金，而我自己却沦落到在地铁口发传单，人生为什么会变成这样？往事袭上心头。

　　那些我带过的大佬

　　滴滴打车 CTO 张博

　　张博是我招进百度的。张博技术水平很强，很有干劲，而且能说会唱。在他的面试评价表上，我给了全 5 分的满分。

　　张博加入后不久，我被忽悠着开发百度日本，要不是百度日本最后解散，他会加入我的百度日本技术团队。

　　我比张博先离开百度，离开百度后，我们一起吃了顿饭。我说：“张博，我在人人网做风车项目，过一段时间我肯定会自己做的。我在风车学到了很多，已经为创业做好准备了。”张博投来羡慕的目光。我说：张博，你也准备创业吧，你是创业的料，百度的层级绑住了你的翅膀。3 个月之后，张博离开百度，华丽转身。

　　两年之后，滴滴打车完成 C 轮融资，估值 40 亿美金，我则进入一家旅行社发传单。

　　学霸君 CTO 敖路

　　一个周末，我在百度大厦加班招人。我翻了翻百度 2008 年面试拒掉的候选人，挨个挨个联系，就这样，我找到了敖路（人称路哥）。

　　路哥是 PHP 大神、Dota 大神。一个月时间，他独自开发完成百度日文文库，无 bug 上线。一个月时间，他成为 Dota 战队队长，带领“粉红妹妹”战队赢得百度 Dota 大赛冠军。

　　2012 年他跟随我一起做风车项目，前后我们一起打拼了两年。

　　两年后，学霸君完成 A 轮 500 万美金融资，我加入只有 3 个人的微拍团队，重新开始写代码。

　　新浪微博首席架构师惠新宸

　　早年的百度，还是 C 语言的天下。我是推动百度 PHP 发展的第一人。我从公司挖掘出惠新宸（人称鸟哥），然后成立百度 PHP 委员会并担任主席，鸟哥等 PHP 大拿担任核心工程师。我们 6 个人的团队，奠定了百度 PHP 的基石。

　　风车网被陈一舟关闭后，新浪微博技术 VP 和我聊，我问有没有 PHP 高手推荐，我说，得鸟哥得天下。

　　很快，在百度一直不被重视的鸟哥华丽加盟微博，成为微博技术形象代言人。

　　美团网架构师刘鹏程

　　我是技术出身，在百度，我做了最大的 C 语言编程框架 UB，是百度 PHP 委员会主席，培养了很多技术大拿。现任美团架构师的鹏程是最杰出的代表。

　　我还在百度做社区经理时，鹏程是我们部门的高工。后来，我被忽悠到百度日本发展，不再直接管理鹏程。但有一天鹏程跑来跟我讲他的困惑：“帝哥，我技术发展遇到了瓶颈，不知道怎么提升。”

　　我问他：“职称评定谁评定你，你和他们熟悉么，你们有机会一起讨论技术么，你们周末一起出去爬山么？”。然后我向他介绍了我著名的“圈子论”。他说：“帝哥，管理上你是 T9。”鹏程非常聪明，之后两年时间，鹏程从 T6 升到 T9。然后也华丽转身，担任美团网架构师。

　　与此同时，在只有三个人的微拍团队，5 年没有写过代码的我，开始写 PHP 代码。我请鹏程吃饭，请他帮忙审核技术架构。他吃惊地问我：“帝哥，你怎么还写代码啊？！”。正是因为那顿饭，我被赶出了微拍团队，原因是技术不过关。

　　百度泰国事业部总经理陈国浩

　　在百度日本，我机缘巧合认识了国浩。国浩在五道口创业 7 年一直失败，当时他的情绪很失落，我们一起聊了一下午。

　　一个月后，国浩找到我，他说：“晓峰，我决定关门了，想和你学点东西”。

　　我给他介绍了一堆大公司的机会。他对我说：“晓峰，我找你不是让你帮忙介绍工作，我就想跟着你学点东西。”

　　是否招国浩，我思考了很久。创业 7 年的他也许不适合大公司发展，如果让他担任日文视频搜索经理，可能会伤害一整个团队。我问他：“做开发，你愿意么？”国浩认真的点了点头。之后他很快晋升实习经理、经理，然后被调到泰国开发新市场。

　　三年后，我在百度蹭打印机免费打印传单的时候，碰到了国浩。国浩已经晋升百度泰国事业部总经理，能说一口流利的商务泰语，是百度泰国员工的“爸爸”。

　　国浩问我：“峰哥，来百度忙啥事？”我尴尬地说：“商务合作，商务合作”。

　　反思：为什么我会沦落到地铁口发传单？

　　我是个成功的管理者，管理过 200 多个员工，没有一个恨过我，包括被我开除的。他们中间，出了七个 CTO，三个架构师，两个 A 轮公司 CEO，10 多个年薪超过百万的。

　　为啥我就沦落到在地铁口发传单呢？

　　反思 1：创业者千万别在北京买房

　　开复老师很早就教导我们，如果有可能创业，别太早贷款买房。

　　思绪回到 2012 年 1 月 18 日。那一天，陈一舟告诉我，风车变现太慢，不愿意继续投入，想把团队解散。而就在前一天，我为女儿买了一套学区房，每月还款 4 万。

　　投资人 William 找过我，他说：“晓峰，风车网是我用过最好的产品，碎片化旅游 APP 分享一定有未来，你把团队带出来，我掏钱你继续做。”

　　虽然当时我非常想接受他的投资，但是背负巨大房债压力的我根本没有选择。背着买房后的压力，我一年换八个工作，最多时候打三份工，做外包，还流落到旅行社卖机票。

　　后来 William 找到彭韬做了面包旅行，一样的碎片化分享，一样的扁平 UI。

　　反思 2：看见机会就得上

　　有好的创业机会就上，自己能力上的欠缺反而不重要。

　　刚离开百度，我遇到过一个很好的创业机会。当时创新工场在为豌豆荚找 CEO，张磊建议我去试试。我当时应声就说：”我一个技术宅男好多东西都不会，还没有准备好！”。

　　我选择了去人人网内部创业。朋友劝我不要去，我执意不听。我说：就算风车失败了，我也学到了东西，这也是收获。结果我收获了失败的经验，错过了豌豆荚的发展。

　　反思 3：并不是每个人都合适创业

　　惠新宸非常明白自己的优势在于对技术的深挖，但是国内的环境，技术创业的可能性非常小，他肯定适合在 BAT 里深造。

　　在目前这样一个人人都在创业，行业缺 CTO 的疯狂时代。每个大公司出来的技术同学，都应该仔细考虑下：自己是不是合适创业，是不是能够承受失败带来的后果？可以参考我写的《想创业，你理性考虑了吗》。

　　2014 年，刘鹏程想离开百度出来创业，他再次咨询我的意见。我言辞很激烈，和他聊了半天后，他决定不出来创业，转而寻求更加合适自己发展的平台。随后，鹏程进入美团担任架构师。入职的第一天，他给我留言：“谢谢帝哥，人生的道路选择上，你第二次帮了我”。

　　结束语：不要温柔地走入那个良夜

　　2014 年 5 月，我加入一家融资千万美金的婚庆网站，担任 CTO。在一个月时间，我组建了婚庆行业最强大的技术团队。7 月，靠着走街串户，靠着身兼数职，我还完了所有 300 万房贷，时隔买房两年。

　　9 月，我开始张罗自己的创业项目：嫁拍婚纱摄影平台；12 月，Pre-A 融资到位；今年 3 月嫁拍成为最大在线婚纱摄影平台。

　　因为买房，错过无数机会；因为买房，现在自由的我已经不再年轻。幸好，我没有掉进钱眼，幸好梦想一直还在。

　　不要温柔地走入那个良夜，年长者应当在日暮时燃烧咆哮；怒斥，怒斥光明的消逝。最后一浪过，善良人喊叫说自己的事业，虽脆弱，本可以光辉地舞蹈在绿湾，他们怒斥那光明的消歇。

　　——献给所有 85 前的创业者
