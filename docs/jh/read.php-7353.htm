MagicNotes：生命的意义

MagicNotes，思绪随风飞扬，偶尔在这里停留。

奥斯特洛夫斯基说过，生命的意义是：人最宝贵的是生命，生命每个人只有一次。人的一生应当这样度过：回首往事，他不会因为虚度年华而悔恨也不会因为卑鄙庸俗而羞愧；临终之际，他能够说：“我的整个生命和全部精力都献给了世界上最壮丽的事业——为解放全人类而斗争。”

打住……我不是作家，我的答案不是这个。

最近看了《翠星之加尔刚蒂亚》的OVA，以前的一些想法又浮现出来，这里记录一下。

《翠星之加尔刚蒂亚》这部动画的设定非常引人入胜。他的故事背景是几百万年几千万年后的地球，由于能源枯竭气候恶化、进入无比漫长的冰河时期同时水平面上涨淹没了地球的大部分陆地已经变得不适合人类居住，这这之前，人类分化成两派并离开了地球进入宇宙。一派是人类为了适应环境而进行的人工进化，名叫西迪亚斯，他们拥有叫“光虫”的外部组织，其本质是一种高科技纳米机械所构成拟似生体组织，吸收电磁波后便能高效地转化为能源，不但可以提供大量能量，其本身的结构使得其能够组成坚硬的外壳，并通过自我复制进行“成长”。另一派是因为伦理道德观念，一直坚持人类特性的“原地球大陆联盟”，他们保持了人类的原始身体，不断发展高科技，然后离开了地球在宇宙居住并开始漫长的航行。

地球是人类生命的摇篮，但人类总不能永远在摇篮里生活吧。

让我们回溯得再早一点，在人类之前，生命的出现与演化。

很多年以前，在熔融状态的原始地球已经有单细胞生物的生存。不要小看那些单细胞生物，它们可是我们地球大部分物种的祖先。它们忍受着无数年的火山爆发、闪电交加，在这艰苦的环境中活了下来，为的是什么？对，就是为了进化成更高级的生命。不一定是它们自己的意愿，而是自然的选择。在他们艰难进化成高级形态之后，然后又一次次遭遇生物的大灭绝，比如恐龙时代。比如恐龙体型一般过于庞大，消耗巨大，在自然选择与生物进化上被认为是一个错误的方向，于是自然又重新进行一次洗牌。不管是疾病、寒冷也好，还是巨大陨石撞击地球也好，反正这一次大家伙们全部灭绝了，体型小的活下来并继续向前进化。

幸存的生物，面对的是劫后的世界，那是个极其恶劣的环境，然而他们还是非常努力地活下来，并进化成现在这个生物多样化的地球。为的是什么？我觉得，人类的出现，回应了自然选择的期待。生命的繁衍，与生物的进化，其目标就是，高等智慧的出现。重新审视人类的历史，也是非常曲折与漫长。直到近一两百年科学技术的进步，人类的发展才得以高速发展。为什么自然界最终选择了人类？因为人类进化出强烈的好奇心，拥有成为高等智慧的基本素质。

那么，为什么生命的进化方向一定是高等智慧的出现呢？为什么呢？

低头看看你的手上的皮肤，上面的某一个原子，它来自你几天前吃下的一块牛肉，那头牛吃过一根草，这根草吸收了一片草原中的某些成分，这些成分是被一座远古火山喷出来的，那座火山喷出的都是地球内部的岩浆，这些岩浆在地球形成之前只是太阳系中的粉尘，这些粉尘来自一片星云，这片星云则来自太阳的前身星——一颗蓝巨星——垂死时产生的超新星爆发。而这颗蓝巨星诞生于更早的几亿年之前的第二代恒星死亡后的遗骸中，一直追朔到宇宙刚诞生的时候，所有这些东西都是一些夸克、胶子……你身体中所有大于铁的原子，都诞生于某颗超新星生命的最后时刻。

终有一天，太阳也会变成蓝巨星，然后超新星爆发，毁灭她的所有行星，包括地球。那时，地球的一切，组成生命的那些物质，又重新回归到宇宙。

不要觉得宇宙离我们好远，因为我们本身就是宇宙的一员。组成我们身体的物质，都曾经是宇宙里的星云尘埃。我们人类具有人的属性，具有社会属性，但大家往往都忽略了，我们也具有宇宙属性。

如同上面所说的，我们的摇篮终有一天会消失，我们人类就需要离开摇篮，回到我们的最初诞生的地方——宇宙。我们头顶上的那片苍穹与星空，才是我们生命最终极的意义所在。

我们人类的追求，比如事业，比如爱情，比如金钱与权力，比如疾病与寿命，在这终极意义的前面都变得渺小无力。人会死，花会谢，一些你现在看似有价值的东西，实际上毫无意义。最终只留下一个影像，在这亘久长流的人类历史中，成为一个模糊的闪烁。

回到我们个体。我们的寿命就短短的几十年，在这终极目标前，我们能起什么作用呢？首先，我们生存的意义肯定不是吃喝玩乐一辈子。一般人，没什么特长的普通人，完成基因传递，其自然意义也就完成了。一些基因不好的个体，甚至连基因传递也无法实现。但是如果你不甘心只完成基因传递就死去，你可以选择努力去推进人类的历史进程，这就是我们个体现在的意义。比如投身医学，突破人类的身体局限，让人类能够更好地发展。比如投身信息科学，让人类的科技更迅猛地发展……等等。凡是历史上推动了人类进程的人，都被称为伟人，并被人类历史永世流传。我们人类个体的意义，大概如此。

抬头仰望那星空苍穹，那就是我们的过去，现在和将来。
