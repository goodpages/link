程序功能：

这个程序类似区一般的调色板。

（1） HUESAT框口的x方向代表的是饱和度（0～255），y方向代表的是色调（0～360），可以用鼠标移动十字来选取自己所需要的颜色，而你所选取的颜色的HSL和RGB都会在相应的编辑框中显示，还有这种颜色会在COLOR框口中显示出来。

（2） LIGHT框口是在当前的色调和饱和度确定的同时可以用鼠标移动竖线选取自己所需要的颜色的亮度。同样编辑框的数值和COLOR框口的颜色也会相应地变化。

（3） COLOR框口显示的是当前选中的颜色。

（4） 通过改变编辑框中的数值，HUESAT框和LIGHT框中的十字和竖线的位置会做相应的修改，还有COLOR框会显示这些参数所对应的颜色。

（5） 只要按下【自动（默认值）】按钮，对话框就会回复初态。

 

1．算法和理论：

【算法】这里主要用到两种颜色空间HSL和RGB之间的相互转换来完成调色板中颜色的选择。主要的算法描述如下：

a) GRB→HSL的算法描述。

步骤1：把RGB值转成【0，1】中数值。

步骤2：找出R,G和B中的最大值。

步骤3：设L＝(maxcolor + mincolor)/2

步骤4：如果最大和最小的颜色值相同，即表示灰色，那么S定义为0，而H未定义并在程序中通常写成0。

步骤5：否则，测试L：

If L<0.5, S=(maxcolor-mincolor)/(maxcolor + mincolor)

If L>=0.5, S=(maxcolor-mincolor)/(2.0-maxcolor-mincolor)

步骤6: If R=maxcolor, H=(G-B)/(maxcolor-mincolor)

If G=maxcolor, H=2.0+(B-R)/(maxcolor-mincolor)

If B=maxcolor, H=4.0+(R-G)/(maxcolor-mincolor)

步骤7：从第6步的计算看，H分成0～6区域。RGB颜色空间是一个立方体而HSL颜色空间是两个六角形锥体，其中的L是RGB立方体的主对角线。因此，RGB立方体的顶点：红、黄、绿、青、蓝和品红就成为HSL六角形的顶点，而数值0～6就告诉我们H在哪个部分。H＝H*60.0,如果H为负值，则加360。

(算法引自：http://blas.cis.mcmaster.ca/~monger/hsl-rgb.html)

b) HSL→RGB的算法描述。

步骤1：If S=0,表示灰色，定义R,G和B都为L.

步骤2：否则，测试L:

If L<0.5,temp2=L*(1.0+S)

If L>=0.5,temp2=L+S-L*S

步骤3：temp1=2.0*-temp2

步骤4：把H转换到0～1。

步骤5：对于R,G,B，计算另外的临时值temp3。方法如下：

for R, temp3=H+1.0/3.0

for G, temp3=H

for B, temp3=H-1.0/3.0

if temp3<0, temp3=temp3+1.0

if temp3>1, temp3=temp3-1.0

步骤6：对于R,G,B做如下测试：

If 6.0*temp3<1,color=temp1+(temp2-temp1)*6.0*temp3

Else if 2.0*temp3<1,color=temp2

Else if 3.0*temp3<2,

color=temp1+(temp2-temp1)*((2.0/3.0)-temp3)*6.0

Else color=temp1

(算法引自：http://blas.cis.mcmaster.ca/~monger/hsl-rgb.html)

 

【理论】这里用到两个颜色模型：

任何颜色C都可以可用光电三原色的三种基色：红(R=red：700nm)、绿(G=green：546.1nm)、蓝(B=blue：435.8nm) 配出：

C = rR + gG +bB

其中，r、g、b为三基色的比例系数，满足r + g + b = 1；R、G、B为三基色的单位。

任何颜色都包含三个元素：色调，饱和度和亮度。而在HSL空间中色调是由旋转的角度决定的，饱和度是由径向的长度决定的，亮度是由在中轴（z轴）的位置决定的。

 

2．方法和技术：

a) 这是一个对话框程序。为了在给用户一个友好的界面。我们把对话框分成两大部分：显示颜色框口和编辑框口。而显示颜色框口又分成三个部分：HUESAT框口（x方向表示SAT的变化，y方向表示HUE的变化），LIGHT框口（x方向表示LIGHT的变化），COLOR框口（显示当前选择的颜色）。编辑框口有6个：分别可以修改各个参数的值（红，绿，蓝，色调，饱和度，亮度）。此外，还增设了一个按钮，便于用户框口恢复初态。

b) 在HUESAT框口显示颜色。为了在框口中依次显示各颜色象素点，我们先设定显示框口的大小，

hs.left = 10; hs.right = 11+vi; hs.top = 10; hs.bottom = 371;

然后再利用HSL与RGB的转换关系，用一个循环：

for (i = 0; i <= 360; i++) {

for (j = 0; j <= vi; j++) {

HSLtoRGB(i, vi - j, 128, r, g, b);

pDC->SetPixel(j, i, RGB(r, g, b));

}

}

依次描出各点。

c) 在LIGHT框口显示颜色。为了在框口中显示当前HUE和SAT下了亮度变化（0～255），我们先设定显示框口的大小，

lv.left = 10; lv.right = 11+vi; lv.top = 400; lv.bottom = 440;

然后再在固定HUE和SAT的值前提下，通过改变LIGHT的值绘出该框口的颜色。

d) 在COLOR框口显示颜色。在当前的参数m_red, m_blue, m_green, m_hue, m_sat,m_light的控制下，利用颜色填充函数

void FillRect( LPCRECT lpRect, CBrush* pBrush );

显示当前选择的颜色。

( 5 ) 利用各绘图函数如：

CPoint MoveTo( int x, int y );和 BOOL LineTo( int x, int y );

来绘出表示鼠标位置的十字（在HUESAT图上），和竖线（在LIGHT图上）。

 

3．要模块及其功能：

本程序分为七大功能模块。而这七大功能模块又是相互联系的（表现在功能模块中的函数调用别的功能模块里的函数），各功能模块及其功能如下：

a) 初始化模块。它主要包含两个重要的函数：

CColorDlg::CcolorDlg(CWnd*pPared):Cdialog(CColorDlg::IDD,pParent)

来设定各初始参数和设定各用户框口大小。

Onpaint(),画初始界面中的各种东西，包括：填充HUESAT框口、LIGHT框口的颜色和COLOR框口的颜色，还有画HUESAT框口中的十字和LIGHT框口中的竖线。

b) 颜色空间转换模块。它主要包含三个重要的函数：

void HSLtoRGB(int hue, int sat, int light, BYTE &r, BYTE &g, BYTE &b)

完成由HSL到RGB的转变工作。

double rgb(double t1, double t2, double t3) 一个计算颜色的函数。

void RGBtoHSL(BYTE r, BYTE g, BYTE b, UINT &hue, UINT &sat, UINT &light)完成由RGB到HSL的转变工作。

c) 动态改变颜色模块。它主要包含两个重要的函数：

void DrawColRect() 画当前颜色的函数。

void DrawLightRect() 画当前色调和饱和度下的亮度变化框的函数。

d) 编辑框的消息映射模块。它主要包含四个部分：

i. 获取哪个编辑框被修改的函数。（共6个）分别是：

void OnChangeEditGreen()

void OnChangeEditRed()

void OnChangeEditBlue()

void OnChangeEditHue()

void OnChangeEditSat()

void OnChangeEditLight()

ii. 通过改变对话框中的数值来改变参数m_red,m_blue,m_green从而改变m_hue,m_sat,m_light的值，并改变HUESAT图中的十字位置和LIGHT图中的竖线位置。（共3个）分别是：

void OnChangeEditRed()

void OnChangeEditGreen()

void OnChangeEditBlue()

iii. 通过改变对话框中的数值来改变参数m_hue,m_sat,m_light从而改变m_red,m_blue,m_green的值，并改变HUESAT图中的十字位置和LIGHT图中的竖线位置。（共3个）分别是：

void OnChangeEditHue()

void OnChangeEditSat()

void OnChangeEditLight()

iv. 通过调用RGBtoHSL函数完成转换，并用函数SetDlgItemInt改变对话框中色调，饱和度和亮度的值。函数为：

void ChangeHSLVal()

通过调用HSLtoRGB函数完成转换，并用函数SetDlgItemInt改变对话框中红色，绿色和蓝色的值。函数为：

void ChangeRGBVal()

e) 画标志模块。它主要包含了两个函数：

void DrawCross(BOOL erase) 画十字函数，它利用函数重载调用了

void DrawCross(CDC *pDC, int x0, int y0)。

void DrawLine(BOOL erase) 画竖线函数，它利用函数重载调用了

void DrawLine(CDC *pDC, int z0).

f) 鼠标控制模块。它主要包含了两个函数：

void OnLButtonUp(UINT nflags, CPoint point),按下鼠标左键确定当前选择的颜色。

void OnMouseMove(UINT nFlags, CPoint point),按下左键并拖动鼠标来选择颜色。

g) 复位模块。这里只有一个函数，

void OnBotton(),重新调整各个框的状态，使它回复初态。
