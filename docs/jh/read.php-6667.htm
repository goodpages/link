独立游戏：创意 、发布与交流

《Flappy Bird》火了。这款触控操作小鸟避开绿色管道飞行的像素风格手游横扫数个国家的App store，甚至在产品的发布者阮哈东(Dong Nguye)将游戏下架之后，各种层出不穷的模仿作品也蔚然成风，牢牢占据着app store的位置。而这款风靡全球的游戏，没有强大的制作团队，也没有投入海量的资金，仅仅是一名越南游戏制作人花费了3天时间就制作而成。

《Flappy Bird》

　　也许是仅仅3天的游戏制作时间、足够简陋的画面和《Flappy Bird》取得的巨大成功形成了强烈的对比，才使得《Flappy Bird》这以一人之力获得的成功显得不可思议。其实，无论是在不少中国游戏玩家眼中已经过去的单机时代还是如今大热的APP上，独立游戏制作人，都不是什么新鲜事物，而在苹果的App Store上，更是有大量的独立制作游戏取得了不俗的成绩。

　　创意： “小而美”的游戏灵魂

　　所谓独立游戏，就是指由能够独立控制游戏开发的内容和走向的开发者所制作的游戏。这些开发者可能自己出资，也可能利用第三方资金，但他们--而不是资方或发行方--一定会保留对游戏内容的控制。就像阮哈东一样，大多数独立游戏开发者都和玩家保持着密切的联系，玩家的意见也更容易对游戏的开发方向造成影响。由于要与资金更充裕的大作竞争，独立游戏的定位一般都是“小而美”，强调游戏性、创意和人性化体验。由于摆脱了资方和发行方的控制，独立游戏给予制作者的空间更大，也因此可以创造出更多风格独特的作品。

Indie Game: The Movie这部纪录片记录了独立游戏人的心路历程

　　独立游戏的“独立”体现在游戏上，就是美术风格、游戏机制和体验方面往往会比商业游戏更锐意探索，而这些探索往往能为游戏业界提供新鲜血液。比如一举拿下多项大奖的Journey(旅)结合了电影的叙事手法，神话学的“英雄成长道路”，以独特的美术风格和机制，造就了具有禅意的游戏体验。制作人陈星汉的其他作品Flower和flOw在操作上颇具创新之处，可以为市面上流行的强调为玩家提供满足感、功利化的快餐式游戏提供新的修正思路。

Journey

　　再比如在多个平台发布的《LIMBO》，利用黑白两色创造出丰富的世界。对于追求画面与特效的游戏开发界来说，《LIMBO》和《Flappy Bird》这样的独立游戏做了足够多的减法，揭示出对游戏而言最重要的是游戏性，是玩家能在游戏中获得的体验。

　　开发与发布：众筹网站与平台

众筹网站

　　游戏引擎和各种可视化编程工具的发展使制作游戏的门槛越来越低，网络社交媒体和各大游戏平台又使游戏的宣传和发行变得前所未有地容易。移动平台的兴起，手机游戏的发展，都使玩家的群体不断膨胀。在技术门槛越来越低，好游戏越来越多的今天，以全职或兼职方式投身独立游戏开发的人也越来越多。而以人的体验为本的独立游戏为玩家创造出丰富的体验，而玩家的回馈也为成功的开发者带来滚滚财源。《Flappy Bird》预计每日为开发者带来5万美元的进账，《Minecraf》面对所有平台的版本卖出了3300多万套，为制作者带来的收益超过1.5亿美元。

　　现在的独立游戏制作人不再是只想阮哈东那样在家里利用空余时间来进行研发，或是寻求某个投资人的赞助。Kick Starter和Indiegogo等私募平台的发展，也为许多独立游戏人解决了募资的问题。2013年，Kick Starter为8000个游戏项目募得了2亿零200万美元的资金。其中1亿8600万美元资金所支持的项目最终都募资成功，进入了正式开发。其中光《星际公民》一个游戏就募集到了数千万美元的资金，为制作者带来了丰厚的收入。

　　由于只要购买License就能发布游戏，苹果的App Store是移动平台的独立开发者发布游戏的主要渠道之一。Steam则更多地面向电脑游戏，包括PC和MAC，独立开发者只要交100美元申请了Green Light项目之后，就可以通过Steam平台发布游戏。不过这些发布了的游戏还需要经过社区玩家的投票，和其他Green Light项目的游戏竞争之后才能上架。各大传统的游戏主机平台也都为独立游戏开发者提供了空间，PSN、XBLA、WiiWare或DSiWare等等都曾涌现出Castle Crasher，ILO&MILO等各式各样的好游戏。

Steam平台

　　一些第三方的平台也为独立开发者提供了渠道。Kongregate、NewGrounds、Armor Games、Desura、GOG等网站将各方面资源整合起来，为游戏开发者提供信息、博取关注、获得盈利，也使玩家能以极低的成本接触到大量出色的独立游戏。像Humble Bundle这样的网站，将游戏的销售以趣味化的方式呈现出来，不仅使一些老游戏得到复兴，也为独立游戏的盈利提供了新思路。

　　交流：独立游戏制作人的盛会

　　然而有平台展示游戏并不能保证盈利。如何让你的游戏从成千上百的优秀游戏中脱颖而出，是独立游戏制作人必须面临的课题。《Flappy Bird》的成功依托于社交平台的口碑营销，也有运气的成分在内。一种更保险的方式，是带上作品去参加独立游戏大会，在这些大会上你不仅能认识其他开发者，获取到更有效的行业信息，如果你的游戏得了奖的话，更高的媒体曝光率能为你的游戏带来更多的用户。

　　这方面比较有名的大会有98年开始的独立游戏节(IGF)，这是独立游戏领域的商业盛会，每年一次，由媒体和Game Group组织共同评选出最佳独立游戏。许多优秀的独立游戏都曾从这个大会上受到人们的关注。如《Braid》、《Minecraft》、《蜡笔物理学》、《Limbo》、《机械迷城》等，2013年度包揽多项大奖的作品是《小贩人生(Cart Life)》。

　　不过，更多独立游戏的大会侧重点在于技术。Global Game Jam(GGJ)和Ludum Dare的参会者都必须在48小时内作出一款独立游戏；Google的Summer of Code， Python的PyWeek和PyDay，以及Ruby的Ruby Day，都是技术类的开发活动，旨在帮助或引导开发者利用特定的技术工具完成游戏开发。

中国独立游戏节(IGF China)

　　此外，许多游戏界的行业大会，比如E3、PAX、GDC等都设有独立游戏环节。GDC甚至专门开设了中国独立游戏节(IGF China)，不仅展示了不少优秀作品，更为开发社群的合作创造了机会。

　　近年来，国内独立游戏界涌现出不少专注于独立游戏的网站，比如拼命玩游戏、独立星球、游资网、Indienova、乐活游戏人等等，也为独立游戏的交流和普及提供了平台。不过，总的来说，国内的独立游戏开发界缺乏一个统一的交流平台与发行渠道，许多开发者都是各自为战。也许，Flappy Bird的成功所引起的对独立游戏行业的关注，会促进这个行业的发展，使越来越多的好游戏能进入人们的视野中。
