按：此文的起源是校园生活版的一个小孩子发的帖子，我认为该贴将数学描绘得太过不堪了，故而有了这篇说教文章。需要申明的是，我是作为一个外行，从自己的体验（这种体验已经停留在很久远的记忆中了）来表述自己对数学的认知和感受，论坛内各位方家众多，班门弄斧的另一个目的在于希望能引发更多的关于学科性质与研究方法的讨论，把这个帖子做成一组笔谈，目前学术论坛版还比较缺乏这方面的讨论。

数学与社会科学的关系——我的个人体验

学校论坛上，看到一个本科大一的小孩在那抱怨，题目叫“我讨厌你！数学！”。
回想自己小学三年级之前，大概心态也差不多。那时我总搞不清楚乘法和除法的区别，在“A架飞机B小时耗油X升，C架飞机D小时耗油多少升”这类问题上总是绕的晕乎乎的。我很感谢我爸。他总能给我讲好玩的故事，不知不觉中把数学思维绕进去，结果在三年级下学期我明白了方程是什么意思。四年级上学期的期末考试中有一道题，大概是已知三角形的三个角之间一个是一个的多少倍，求每个角是多少，很多人没做出来，它们还不知道方程是啥子。我先用方程解出来，再导出数学式子，美其名曰两种方法。我原来一直是在班级第十到第十二之间晃悠，那年靠着数学考了全班第一，这个位置也基本上一直保持到高中。从那以后对数学的兴趣就一发不可收，因为我发现逆向思维冥思苦想一时半会想不出方案的问题，靠方程这种顺向思维的方法都能很快迎刃而解。小学时间真是松散休闲，又没有什么压力，而在此境况下再加上兴趣，那学习的效率和速度就特别快，小学六年级的时候基本上把整个初中三年的代数和几何都看完了。当然是跨越式发展、大跃进式的看，是原理层次的看，只是把教科书上的习题做了，不可能专门做更多的题以为巩固。即便如此也足以令我我兴奋不已了。我还记得当时在班级里如同摆摊算命一般、跟小学同学卖弄笔算开平方的“手艺”时那种神气劲儿。
中考数学148分。进了高中后，对历史兴趣猛增，学校也鼓励我学文。高二之后文理分班后，我一直较着劲儿，专门和理科班排在前面的那几个比数学，因为数学语文外语这是唯一可资比较的交集了，而我更看重数学，每次考完试都去年级办公室看那几个理科班的前几名的成绩。高二那年上体育课的时候，由于想偷着去网吧上网和打游戏（2000年，当时网吧还是新鲜事物），结果一边往前跑一边回头提防着被老师看见，一个不留神撞到学校大门的小侧门的门槛上，出血，到医院缝了六针。在家住了一个月。我记得撞头的第二天早上起来，第一件事就是找出五道解析几何题，做出来了，我放心了——这说明我至少还没有痴呆。用数学检验脑力的想法是受小平同志晚年说过的一句话的启发，我记得纪录片里已经快九十岁的他骄傲地讲道：“能游泳，说明我身体还行；能打桥牌，说明我脑子还行。”哈哈，多可爱的老头儿！
高考数学132分，五分的选择题错了一道，后面的计算题大概也有点纰漏。总体上我对此是满意的，基本落在平时成绩浮动范围之内。大二之后，高数课也没了，纯粹数学的书看的少了，除了大四上学期曾经又专门温习过一遍。此后基本上都是在经济学的文献中看到数学，有不懂的再返回数学书中去找专门的解释。不得不承认，现在的功利性心理已经不少了，尽管这种功利性仍然是在学术的范围内的，因而也只是一种有限精力在无限的学科分界之间的内部再分配。但我总还是很怀念那种没有为了应用的目的而学习数学的时光，那时的开心完全是来自数学自身，数学在那时就不只是单单服务于经济学、历史学、政治学等等其他学科的手段。当然这种精力的分化也是必然，学术有分工，术业有专攻，再加上各种无形的制度压力将你的无差异选择曲线逼成现在的样子。
我对数学的感情是深厚的。我甚至认为最最基础的学科甚至不是语言，而应该是数学。
数学不是数字，数学是逻辑；数学不是计算，数学是推演，是证明。人不能永远靠归纳、靠联想、靠类比、靠比喻来表达自己的思想，来了解世界，来发现真理。
中国古代数学思想有着丰富的实践土壤，从天文地理到水利灌溉，甚至风水占卜，土木建筑，这些在古代社会的基本生存需要引导着实践性数学的萌发。中国数学思想的欠缺之处是没有发展出一种基于纯粹学术研究性质的、建构性的理论体系，故而中国的代数与几何思想渊源流长， 遗憾的是却始终未能发展出一套具有抽象性的符号语言的公理体系。中国有勾三股四弦五，没有a^2+b^2=c^2的毕达哥拉斯定理；中国有杨辉三角，没有二项式定理。这是中国数学思想的一个弱点，当然这丝毫不影响中国数学思想的丰富性和历史意义。
我想再没有什么时刻比人用自己的理性和头脑思索出一件事情的结果来更为引人入胜了。如果对数学产生了厌倦甚至厌恶，那不妨回忆一下高中的立体几何课上，当你的老师在一开始为了说明一个二面角、异面直线的关系而用硬纸壳折来拆去，而你发现自己的头脑中已经建构起一个matrix体系的时候，不需任何实践就可以在头脑中清晰勾勒出各种线、角、面的关系的时候，在那种形象思维与逻辑思维的交相辉映之中，你建立起神奇的立体感和逻辑感，体会到数学的伟力，感悟到逻辑的力量，你脑子里已经有了一个看不见的手，他可以让你神机妙算而完全无须躬身践行。
科学的目标是提供解释，提供给人们一种理解世界的方式，在此基础上如果能够对未来的发展趋势做出大致准确的预测，那就可以得到相当程度的认可了。故而预测是建立在解释的基础之上，只有首先能够将历史事实、既有的过程说清楚，才有资格、有可能谈预测未来的问题。即便如此，社会科学中的规律也只能是一种“趋势律”，不可能达到自然科学中预测的精确程度。
那么，数学在社会科学中的作用有多大？相当大。但是这主要是运用数学的逻辑建构和推演的功能，通过数学这个工具来仿真、模拟社会历史过程。社会是人组成的，人的预期、人的心理、人的一切主观活动都会对社会发展变迁产生影响，但数学对这些主观性指标的模拟是有限度的，比如在经济学中探讨风险、不确定性的时候只能引入概率来描述人的主观活动，人对风险的评估是一种主观概率，但这种主观概率的获取又不能通过先验方法，而是经过反复实验获得经验数据。
这就引出自然科学与社会科学的重大区别。自然科学着重于对原因特别是终极原因的解释，其基本思维方法是从现象到本质、执果索因，比如是首先发现苹果落地这个现象，然后由此为出发点去思索造成这种现象的原因（当然苹果落地这个故事的真实性已经受到质疑，这属于科学史的范畴，不去讨论）。自然科学的价值在于挑战甚至颠覆人的感官秩序，质疑人的本能，推翻人的常识。自然科学总是要提醒人们：不要过于相信你的感觉，你感觉到的可能是错的。日心说地心说的争论就是自然科学的很好教案。自然科学并不关心作为客体的人对于自己所处的这个世界的想法，但社会科学不同。
社会科学必然是先验的，但与自然科学致力于揭示不同于人的感官秩序和常识世界的真实世界相比，社会科学必须基于人的主观世界和感官秩序，必须从常识出发，尊重常识，否则就变成了巫术、玄术、妖术。如果说自然科学都不能做到完全“客观”的话，社会科学就更无可能。因为它的方法则是相反，它并不在乎终极原因。它要研究的就是人的主观世界，由于人的主观世界的种种神秘莫测的因素而产生的可被所有人观测到的“客观”的行为。对社会科学而言，因果链条的起点不是最重要的，相反，它从人类学的意义上应当是被给定的一个预设，对经济学而言，这个预设可能就是“理性人”。很多人试图从各种角度否定这个预设，就像经济学界长期有一个争论，小农经济中的小农究竟是不是理性人？农民到底有没有理性？这完全是无意义的辩论。谁没有理性？难道有的人就比别人高人一筹？所以有些人不理解别人的行为，那只能说明你们的目标设定不同，目的不同，理想不同而已，不能说你理解不了的事情就是非理性的，这就是中国传统的经济学教学中时常讨论“雷锋是不是理性人”这个问题的最大荒谬之处，因为目标是没有理性与非理性之分的。
所以，尽管自从“理性人”这个概念明确从穆勒那里说出来至今的一百多年里受到不少人的诟病，但是它们否定了这个预设，实质上仍然加上其他的预设。总之“预设”是社会科学必不可少的一个起始台阶，这与自然科学不同。若如此则社会科学岂不是成了无用之学？它的起点是人的常识性的通用属性，所要解释的也是人的常识性生存世界，这还有什么存在的必要？有道理，也没道理。有道理的地方在于社会科学的“硬度”的确就是不如自然科学，所以也叫“软科学”（国家有软科学基金会，各地有软科学课题）。没道理则是由于社会中也有隐蔽的逻辑链条存在，需要人们去发现和揭示。我这里要引用哈耶克的一个观点，他提出：社会科学的终极目的就是去研究和解释“属于人类行为的结果但又不是人类有意设计出来的结果的这样一种自然秩序”。因此社会科学家不会执着于对行为动机的探究（当然有些心理学家是很执着于人的本性的研究的），他们将人的行为模式作为先验条件接受下来，进而研究在此动机下的个人的并非自觉设计的、无意识的活动如何形成了一种井然有序的局面，研究微不足道的个人行为是如何形成更宏大的社会历史过程的。
羊肠小道就是一个典型的例子。每个人都在各自寻找自己的道路。经过一段时间以后，有一些道路因为更经常地得到利用就变得更易于行走，最后，就“世上本没有路，走的人多了，也变成了路”。社会科学想要解释的就是这个千千万万个体无意识行为之下形成的清晰的模式。
我觉得这正是社会科学的魅力所在。数学在帮助社会科学完成它的这一根本任务的过程中是居功至伟的。因而我固然很乐于看那些运用计量方法对历史数据进行处理、拟合，最终比较成功地对未来发展趋势做了相对正确的预测的文章；我更乐于看那些运用人的大脑的独特理性和逻辑思维能力对一个社会现象的自然演化过程进行动态模拟的那些精致漂亮的数学模型，它们甚至未必十分高深，但是解释力极强。比如当年卡尔多命名的、从猪肉价格波动的事实中提炼出的蛛网模型，比如宏观消费理论中的永久收入、生命周期模型，比如如今从政治学到经济学到社会学到处都在应用的讨价还价博弈模型等等，这些模型的设立是出于解释的需要，目的是为了描述过程，而市场正是这样一个神秘的过程。
最后要补充一点。理性是有限的，数学也在逐步丧失其确定性（湖南科技出版社有本《数学：确定性的丧失》，是著名数学家克莱因写的，他还写过四卷本的《古今数学思想》，很有趣）。但是我们还是要承认在很大范围内数学的伟力，就像我们必须在很大范围内承认牛顿经典力学的放之四海而皆准一样。
扯的太远了，收尾，回到开头说的那个帖子来。其实财经类专业本来就离不开数学，说到底，是离不开理性的思维和严密的逻辑，否则一切靠肾上腺素拍脑袋，那和热血愤青有什么不同。那个在论坛发帖抱怨的小孩子还年轻，刚刚大一，应该尽快补起来，来得及，绝对不能放任下去。
