唱吧：从散伙到App中国榜冠军

创业团队年初曾有人提出散伙；上线5天后成为App中国区总榜第一名

　　陷入抓狂和烦躁一年多后，34岁的陈华的二度创业之旅“柳暗花明”。

　　他治下的团队，在2012年上半年“琢磨”出了手机应用软件“唱吧”。上线苹果App Store仅5天，可以让用户对着手机“想唱就唱”的“唱吧”，即飙升为总榜的“状元”。

　　“唱吧”一时风靡。不仅众多青少年为吸引到粉丝的喝彩纷纷引歌扮俏，如韩红、宋丹丹等艺人也时常会跑到“唱吧”上“小露一手”。

　　陈华称，截至目前，“唱吧”在近5个月内即吸引了近百万的用户。同时，已有风投愿意为这款“还未找到合适的盈利渠道，并且近期不打算盈利”的软件掏出近千万美元。

　　创业团队没人会唱歌

　　这个“既没有做过音乐，也没人唱歌好听”的团队里，有人提议“做唱歌分享”，而这个提议起初未获认可。

　　“如果这个方向再不成功，我们就散伙吧。”2012年春节前的一次碰头会上，有位工程师，突然对陈华和团队的其他成员说。

　　其时，这个“既没有做过音乐，也没人唱歌好听”的团队里，有人提议“做一款普通用户可以手机上唱歌并分享的软件”。但该提议一出，立刻得到了呛声一片，“为什么用户要听一个陌生人唱歌？他又不是明星。”

　　“提出一个，否决一个”已使这个迟迟无法获得突破的团队“濒临崩溃”。2011年2月陈华从阿里巴巴搜索业务离职二度创业后，曾创立过定位电商导购促销网站的最淘网。

　　“互联网的流量太过集中于几大网站，而购买流量的代价又太高，所以最淘网并没有达到我们起初所设想的爆炸性增长预期。”陈华说，最淘网在2011年9月上线，但仅仅两个月后，“就被我们扔掉了”。

　　随后，陈华带着来自酷讯、阿里巴巴、开心网的创业团队，开始寻找一个“靠谱的”产品方向。一开始，陈华就将研发的方向定位在移动互联网，“我们无法跟互联网巨头抢生意，而移动互联网随着智能手机的普及却有很大的市场潜力”。

　　而具体到做哪款产品，团队内部却众口不一。“开了无数次会，每次会上大家都会把自己想到的七八个方向提出来，一起权衡优缺点。”陈华说，讨论大多无疾而终，他为此“焦灼地连续几晚睡不着”。

　　比如，他曾经设想开发一个声音的微博，让用户在上面给他的粉丝讲故事和笑话，抑或是唱歌朗诵，但一经讨论，大家就批评说这款软件并不适合原创能力差的人玩，“估计没什么市场”。

　　综合判断，陈华觉得做唱歌软件的方向还算“靠谱”。不过，在碰头会上无法说服反对之声的他，“直到一周后还在犹豫做还是不做”。与团队和董事会几番沟通后，他下定决心，“没什么对与错，往这个希望最大的方向尽力吧。”

　　开发时间只有两个月

　　陈华将喜欢被围观夸赞、有明星梦的女孩作为“唱吧”最主要的刚性需求用户之一，“她们来了，宅男自然就来了”。

　　2012年春节后，陈华团队开始投入到唱歌软件的研发中。开始时，陈华赋予这款软件的功能是“做声音的‘美图秀秀’”——“通过对声音过滤、增加回音等效果处理，让歌者有‘原来我唱歌这么好听’的感觉。”

　　为此，工程师翻阅了很多关于声音的理论书籍，并专程跑到唱片公司去请教，“我们尽量使这款软件的声音处理达到初级录音棚及专业歌手接受的水平。”

　　此间，陈华的妻子建议这款软件起名为“唱吧”。陈华理解这个名字带有多重含义：“它既是一个鼓励大家唱歌的动词，也是一个唱歌的地方，更是一个听人唱歌的酒吧”。

　　而此后的市场推广中，“唱吧”后面还被加上“你的手机KTV”以强化其与音乐播放器的区别。

　　陈华还力主在“唱吧”里融入评论等社交元素。这招致了很多团队成员的不理解。“有人说，就是听人唱首歌有什么可评论的，直接加个‘赞’或‘否’就完了。”但陈华认为，如若那样，唱吧将“非常僵硬，一点也不好玩”。

　　“我还提出一点，一定要吸引到美女上榜。”陈华将喜欢被围观夸赞、有明星梦的女孩作为“唱吧”最主要的刚性需求用户之一，“她们来了，一些喜欢看美女的宅男自然就来了”。

　　于是，“唱吧”允许用户在唱歌过程中，可以上传一些自己的照片或其他图片。由此，粉丝听歌的同时，还可以在视觉上享受到“美丽的元素”。

　　同时，为推广的考虑，“唱吧”只支持微博、QQ、人人网等社交账号登录。陈华称，这容易借助三者庞大的用户群，实现一传十、十传百的病毒式传播。

　　据陈华介绍，“唱吧”正式上线前，其团队就已提前规划好了几个月内的发展路径，“包括了几月上线App Store，什么时间点推出新版本，模仿者出来后怎么办等问题。”

　　今年5月31日，“唱吧”正式在App Store上线。陈华称，从创意到正式上线，只有不到两个月的时间。

　　中国好声音学员“进吧”

　　除了韩红、宋丹丹等艺人，“唱吧”还与《中国好声音》达成协议，邀请一批学员到“唱吧”开唱。

　　出乎陈华团队的意料，上线仅5天后，“唱吧”在App中国区总榜排名便飙升至榜首。此后至今，“唱吧”排名一直保持在前列。“‘唱吧’已经累积了近百万的用户。”陈华称。

　　比数字更能体现其影响力的，是“唱吧”所创造的“玩法”在各个年龄层次的用户间的风靡。自称“唱吧中毒重度患者”的网友发微博称，玩“唱吧”后“手机充电勤了，嗓子干得快了”；一位七十多岁的老人，则向陈华建议说“唱吧”应加入京剧。

　　“唱吧”还得到了一些专业歌手和艺人的垂青。9月2日，韩红在其个人微博中上传了其利用“唱吧”录制的《我的歌声里》、《狮子座》等有别于其演唱风格的歌曲。而宋丹丹也曾为儿子用“唱吧”演唱的《亲密爱人》打气喝彩。

　　陈华所力挺的“评论”及推送功能极大增加用户的活跃性。常常会有通过消息推送得到消息的粉丝，第一时间赶到发布新歌的“歌手”的留言板中“抢沙发，抢板凳”。而“求关注、求粉丝”等字眼也常常出现在留言板里。

　　在陈华看来，“唱吧”的重中之重仍是增加用户规模。为此，“唱吧”与当下大红的《中国好声音》达成协议，邀请一批学员到“唱吧”开唱。此外，“唱吧”还与腾讯微信合作过K歌大赛。

　　“‘唱吧’目前还没实现盈利。”陈华说，购买带宽和版权的花费仍依赖于其2011年时从VC拿到的首轮融资。据陈透露，“唱吧”未来可能会选择出售虚拟物品的盈利模式，不过他也做好了“三年不盈利，五年不上市”的准备。

　　目前有传言称，“唱吧”以20%的股权换取红杉中国1500万美元的注资。按此计算，“唱吧”的估值已达到7500万美元。要知道，加上陈华在内，目前“唱吧”只有16名员工。

　　陈华对此未予证实。他只是说，“目前想投‘唱吧’的风投非常多，第二轮融资正在商讨之中。”
