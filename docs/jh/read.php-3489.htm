企业移动应用引领未来发展趋势

在2012博鳌房地产论坛上，用友移动应用事业部总经理傅强就移动时代下企业应用变革问题接受了记者专访。

云计算时代的到来，使得企业信息化这一话题又有了新的生命。在云端不断增强的服务性能之外最显著的特征就是在端的精彩表现。单纯用PC来使用ERP的时代将一去不复返。手机也早已不只是打电话的工具，以手机、平板电脑介质为代表的移动终端应用将为企业信息化带来巨大变革。

移动应用带来企业信息化商业模式变革

移动应用不只是在手机上运行软件那么简单，它改变了人们的行为——就像现在大家发微博，都是用手机来发，慢慢发现手机比电脑好使、方便、随时随地，手机上的微博功能没有发生什么变化，但对人的行为发生了很大的改变。

对于使用者或企业带来的改变，一是全员化管理。全员化的管理就是从末梢到人，未来的移动ERP就是企业有多少人就有多少人在使用，而不仅仅是企业老总、管理层、财务人员，这就是全员化，比如针对管理者，像一些高管或一些业务主管，他们经常面对的是审批，查询数据；比如现场的工作人员，销售、服务工程师、维修工程师，包括物流送货员，他们也必须依赖这些移动设备去工作。用友现在在这两种不同类型下面重点提供移动应用。二是实时化管理。之前审批单据时，必须用电脑登录到办公网，然后再去处理业务，移动应用则可完全抛弃电脑束缚，直接通过手机登陆客户端，出差也好、开会也好，都可以处理，方便快速。这种情况下，ERP的概念就会发生改变。其前提还是基于云计算的方式，但是所带来的改变不能简单的理解为ERP的移动化。移动应用将带来企业信息化商业模式的创新变革，这必然会成为一个趋势。 

企业级移动应用的显著特性

企业移动应用涉及到企业信息化应用场景的完善、扩展，带来ERP的延伸，让ERP无所不在，通过广泛的产业链合作为用户提供低成本整体解决方案。

企业级移动应用分三大类，一、基础信息服务；二、移动管理；三、移动现场。

第一类基础信息服务比较好理解，第二类移动管理，用友为企业提供移动管控解决方案、移动协同办公解决方案、移动商业分析解决方案，移动管控包括企业关键流程管控、企业关键数据管控、企业人力资源管控；移动协同依据用友ERP产品特点提供消息中心、协同管理、公文管理、公共信息、文档管理及通讯录六个模块，分别支持iPad、iPhone、Android客户端；移动商业分析应用不再受局限，具有强大的云搜索及语音搜索功能。

第三类移动现场的业务是把ERP流程延长了，传统的ERP还触及不到，比如说促销员的数据采集，在门店采集数据，跟踪销售人员的位置，这都是以前ERP不具备的。以前ERP是一个技术，不是一个产品，后来成为一个遵循ERP规则的产品。

客户群低门槛

用友移动应用不仅为大型企业进行移动信息化支持，中小企业甚至个体户都能在此找到适合自己的移动应用解决方案。之前的信息化系统，动辄几百万，门槛很高，但移动应用往往不会有那么高的门槛，甚至于手机都不用买。比如跟中国电信合作，企业要给1000名干部配这种应用，按照传统模式买一个软件，加上1000个许可，再加上1000个手机，还要再办1000个上网卡，费用将会很庞大。但在移动应用下只需将之与运营商打包，客户不用一下付出很多财力，每个月给电信付套餐费就可以了，对于小企业来说，成本支出也在可受范围之内。移动应用软件针对客户群采取一种更低的门槛方式，给客户带来更好的体验。

对于员工信息化素质偏低的企业也不必担心这部分的成本投入问题，因为“简单应用”用友已为用户考虑到。相对于PC应用而言，能配置PC进行ERP系统操作的岗位，大多有良好教育背景和专业培训，而由于移动应用的场景差异大，使用者差异大，人员规模大等因素需要移动应用不需要专门培训简单易用。比如，我们的客户中有一个消费品企业客户上营销数据采集解决方案，使用者是全国在商超内的近5000个促销员，30~40岁的职高女性占多数，有计算机使用背景比例较低，不可能集中培训。要让这个群体能够使用起来，应用设计必须简单，稳定可靠，十分钟内可以上手，最短时间内实施完毕并投入使用。

移动应用不是水平应用也不是行业垂直应用，无论是企业市场销售、经营管理、资源管控、决策支持的方方面面，从企业业务员、送货员、服务人员、工人到业务主管、企业高管，移动应用都有其适用的应用场景和应用价值。移动应用在企业信息化的各个领域都是无所不在的。如果可能，一个企业可以使用成百上千的移动应用。

另外，虽然移动应用的主角是手机，但并不是说只有手机上的应用才是移动应用。不同的场景下需要不同的移动终端。手机作为通用型消费品在企业级应用上存在许多缺陷，比如：电池续航能力，一/二维条码读取，RFID识别，IC卡读写，三防（防水/防尘/防震）耐用等方面。因此，平板电脑、PDA、存储式IC卡识读器、条码枪等都可作为特殊场景下的移动应用设备。

移动信息化应用价值体现

移动应用不仅面对ios系统，也面对使用Android系统的用户。同时，移动应用可帮助客户间接降低成本，相对于开源节流的方式，增加了收入以后，自然成本就相对低了。提高了工作效率，成本也会降低。移动应用给企业带来的价值可浓缩成三部分：实时信息采集器、业务流程加速器、信息价值放大器。

♦全员化实时信息采集器

相对于PC终端，移动终端在时间和空间上有效地突破，推动企业ERP系统整体运营实时化，企业大量存在的现场工作人员（如：销售员、市场督导、渠道经理、现场服务工程师等）将通过移动应用企业实现实时地互动。全员化不是被动的，而是通过手机这一个人常备的通用介质让这些群体得以实现信息化，成为企业信息化最庞大的神经末梢，使得企业通过这些神经末梢可以及时、准确获得一线业务信息，并及时做出反馈。

♦业务流程加速器

大多数企业使用了ERP系统后，业务都在系统上设定并运转，优势在于规范流程执行和高效的数据统计汇总，但是同时也发现很多企业在信息化达到一定阶段后便提出了新的问题——流程搁置，而在企业上了移动审批应用后该周期明显缩短。可以设想，一个市场响应更快的企业所具备的竞争力，加速器是移动应用给企业带来的重要价值。

♦企业信息价值放大器

信息的价值在于准确、及时、响应、分享。准确的问题在ERP中已经不是问题；及时的问题跟上面的问题一样，只有通过高管或者企业全员随身携带的手机，加上信息推送机制才能有效解决；信息看到了如果没有反应就只能说信息到此为止了，价值也就被终结了；信息需要被分享，让更多的关系人也能准备、及时地分享，信息的价值会被成倍提升。

强大技术支持保障信息安全

安全问题本身是一个很大的问题，经过分析，用友移动应用较好的保障了用户的信息安全。

首先，移动应用帐户跟ERP帐户一体化，不会单独设置新的用户进入体系。假设手机遗失，通过服务端可以实现关机，手机不存储信息，只是一个终端，没有数据库，没有任何文件，即使手机丢了，也不会出现数据流失的事情。手机被别人捡到，没有专业的技术是破解不了的，即使破解了，移动应用只是一些碎的数据，比如说一张单据，或者审批，是或者否，你很难获得企业的核心资料。

其次，虽然它是通过互联网进行传输，但是手机方面有移动的Sever，它所交互的数据是有限，而不是把后门打开，一丢就全丢了。即使出现数据丢失的情况，也仅仅是涉及到一小部分的数据。

用友做了好几层安全保障，新的移动应用平台进一步提升，如果每个产品自己做，在安全保障方面遇到的问题就更大。一些小厂商所采取的措施，风险其实更大，本身这个企业就是规模很小，而且第三方厂商接入到ERP，它的接入过程容易被截取，数据不是很安全。用友的接入是内嵌方式，跟用友NC和U8接入是直接接入，不是通过第三方的接入。

随着3G时代的到来，率先拥有移动应用对于企业来讲不仅是抓住了这一历史性的机遇，更表现出了大胆的创新精神和和前瞻性。在新技术、新理念的支撑下，移动应用将为那些“先知先行”的探路者企业提供超越同行竞争对手的最佳机会。
