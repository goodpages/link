GIS(Geographic Information Systems,地理信息系统)是多种学科交叉的产物，它以地理空间为基础，采用地理模型分析方法，实施提供多种空间和动态的地理信息，是一种为地理研究和地理决策服务的计算机技术系统。其基本功能是将表格型数据（无论它来自数据库,电子表格文件或直接在程序中输入）转换为地里图形显示，然后对显示结果浏览，操作和分析。其显示范围可以从洲际地图到非常详细的街区地图，现实对象包括人口，销售情况，运输线路以及其他内容。 

　　地理信息系统（GIS）技术是近些年迅速发展起来的一门空间信息分析技术，在资源与环境应用领域中，它发挥着技术先导的作用。GIS技术不仅可以有效地管理具有空间属性的各种资源环境信息，对资源环境管理和实践模式进行快速和重复的分析测试，便于制定决策、进行科学和政策的标准评价，而且可以有效地对多时期的资源环境状况及生产活动变化进行动态监测和分析比较，也可将数据收集、空间分析和决策过程综合为一个共同的信息流，明显地提高工作效率和经济效益，为解决资源环境问题及保障可持续发展提供技术支持。

数字图像处理技术在监控领域的应用

今天的人类生活在一个信息时代，图像是人类获取信息、表达信息和传递信息的重要手段。研究表明，在人类接受的信息中，图像等视觉信息所占的比重达到了75%。而与此同时我们又生活在一个数字时代，随着计算机技术及网络技术的迅速发展，几乎所有的信息都可以以数字的形式呈现在人们眼前。因此，在这样一个时代里，数字图像处理技术的研究和应用就已经成为现代生活中必不可少的一部分。
一、图像处理技术的起源
图像处理技术简而言之就是用计算机对数字图像进行处理，他起源于20世纪20年代，当时英国伦敦通过海底电缆向美国纽约传输了一幅照片。这只是图像处理技术最初的应用，它采用的数字压缩技术为后来图像处理技术的快速发展奠定了基础。而1964年美国的JPL实验室处理了太空船“徘徊者七号”发回的月球照片，标志着数字图像处理技术开始得到真正的实际应用。其后，卫星遥感、军事、气象、医学等学科的发展推动了数字图像处理技术的快速发展。
二、图像处理技术的应用
数字图像处理技术的迅猛发展为人类带来了巨大的经济效益和社会效益，大到应用卫星遥感进行的全球气候监测，小到指纹等生物特征识别技术的应用。数字图像处理技术已经融入到生产生活、科学研究的方方面面。
遥感图像的几何变换可将图像与地理坐标相配准，实现遥感图像与有关矢量图件的叠加。图像的边缘检测技术在军事上则可以帮助军方快速的找到机场公路、桥梁等关键设施。随着遥感图像分辨率的不断提高，遥感的应用范围已逐步扩大到资源调查、气候监测、军事侦察等领域。图像处理技术在指纹、面部等生物特征识别中的应用非常广泛，包括图像增强、分割和识别等技术。美国一家名为“Iriscan”的高科技公司研制出的红膜识别系统已经应用于美国德克萨斯州联合银行营业部内。特别是美国9.11恐怖袭击之后，生物识别技术已经广泛的应用于美国的机场、码头、道路、关口等重要的场所，并逐渐发挥出巨大的作用。
三、图像处理技术在监控行业中的应用
图像处理技术在监控行业的研究和应用还刚刚处于起步阶段，但他的实际应用价值已经引起了行业中许多人的注意，并投入了巨大的人力和财力进行开发和研究。随着科学技术的发展，特别是监控行业数字化以后，人们对监控图像质量的要求进一步提高，数字图像处理技术应用于监控行业已经成为大势所趋，通过数字图像处理技术提高监控图像的质量，真正体现数字化硬盘录像机的价值，也已经成为整个行业中的共识。让我们来具体分析图像处理技术在监控行业中应用的背景
首先，随着科学技术的不断进步人们对监控图像质量的要求越来越高，提升监控图像的实用价值已经成为社会向整个监控行业提出的新要求。
其次，计算机整体性能得大幅度提高和应用数学领域获得突破（特别是代数和几何运算、图像变换、图像增强、图像复原、图像编码、图像融合技术的突破）为图像处理在监控行业中的应用提供充分的技术和理论保障。
最后，高端领域应用技术的平民化（图像处理原来是应用在航天、军事等高端领域，随着科技的进步特别是计算机性能的提高使得这种高端技术应用于我们的日常生活当中已经成为现实）。
对于图像处理技术在监控行业实际的应用，我们必须先从图像质量谈起。在监控图像中普遍存在着这样一个现象——光污染，对于光污染我们有这样的定义：对于视频监控系统，由于现场安装环境的影响（如白天、夜间、晴天、阴天和太阳光线运动等），图像存在着逆光、强光、弱光、模糊、偏色等缺陷，使得现场资料模糊不清发挥不出应有的作用。这种光线对图像质量的影响通常被称作光污染。
我们在实际监控设备应用当中经常会遇到这样的情况，各种各样的监控图像由于受到光线的污染会图像质量会有这样或者那样的缺陷，使我们根本无法能够获得图像中的有效信息，对于用户来讲这是一种很大的损失。我们就曾经碰到过查询监控录像时因为图像质量太差，而致使警方没有足够的证据抓住罪犯。试想，如果如果我们使用DVR所抓拍的图像有一大部分都达不到使用的要求，那我们做数字录像监控设备还有什么意义呢？这对我们监控行业的发展来讲是一种灾难。监控图像质量上的问题给我们监控行业特别是生产硬盘录像设备的企业提出了更高的要求，也促使了数字图像处理技术在监控领域的应用。
随着视频数字技术的发展，网络化和智能化是未来监控行业视频数字技术发展的两大必然结果，而我们的数字图像处理技术又是智能化数字视频最基本的应用单元。可以说现在监控行业数字图像处理技术仍然处理起步阶段，但它的应用将为数字视频智能化的发展起到建设性的作用。现在监控行业中图像处理技术的主要还是涉及到下面两个方面：
1． 经过图像处理还原出人眼感受不到的，但又现实存在的原始信息
2． 纠正图像中存在的逆光、强光、弱光、偏色等缺陷，还原真实信息
我们知道，公安干警侦破案件需要寻找各种各样的线索，而有时候我们看起来很不起眼的一点信息，可能也会对案件的侦破起到决定性的作用。通过图像处理技术，我们可以更深层次的发掘数字图像当中的有效信息，从而为警方破案提供有利线索和证据。 
四、监控行业数字图像处理技术原理
数字图像处理技术在监控行业实际应用的效果还是非常明显的。随着科技的进步,市场需求的增加，现在也已经发展成为监控行业中的一个新发展方向和技术亮点。
1． CIF格式
谈到数字图像处理技术，我们首先就要介绍CIF格式因为我们的录像是CIF格式的，而我们的图像处理的图片也是从我们录像资料中所获得的。
CIF格式标准是CCITT国际电报电话咨询委员会于1984年提出的，1990年12月通过了H.261即P×64kbit/s视听业务用的视频编解码器。其中P是一个可变参数，当P=1或2时支持四分之一格式即QCIF，而当P>6时支持通过中间格式即CIF，其色度信号分辨率为176×144像素，亮度信号分辨率为352×288像素。
2． 数字图像的构成
在数字图像处理当中我们把一幅CIF格式的图像分成三层。根据杨—赫姆霍尔兹的视觉三基色假说一幅彩色图像由R（红）、G（绿）、B（蓝）三原色组成。在CIF格式中我们把亮度层和绿色分量分为第一层它是352×288像素,把蓝色分量分为第二层它的分辨率为176×144像素，红色分量分为第三层它的分辨率为176×144像素。
3． 数字图像处理原理
了解了数字图像的结构我们就可以对数字图像通过各种数学算法进行计算和变换从而达到处理的功能。我们以汽车牌照为例，在监控设备实际应用当中我们常常会遇到这样的情况，由于车辆的牌照在交通道口经常会受到对面车灯强光等或外部光源的照射，使得摄像机拍摄出来的车牌照片反光，人眼根本无法识别。我们看不清照片并不是说数字图片中没有车牌号码的颜色信息，而是由于光线的原因（或者说是光污染）使得车牌号码颜色信息被强的亮度信息掩盖掉了。通过我们的图像处理软件可以对图像的亮度层进行处理，而把车牌的信息显现出来。经过图像处理技术处理以后我们就可以清晰的分辨出车牌的号码信息。这种技术在实际的应用当中会非常有实用价值。同样，对于一些由于逆光、弱光、暗光、偏色或综合因素影响的监控，用图像处理技术同样可以达到很好的效果。
我们知道监控图像，由于受到前段摄像机质量、压缩解压缩的影响本身图像质量就已经收到了很大的影响，如果再加上光线对图像的影响，使得目前使用的数字硬盘录像机有一大部分达不到实际使用的要求。那么我们该如何处理才能使这些由于受到光污染本来应该可以看清我们却看不清的图片能够看清楚呢？我们的方法有三步：首先，按照数字图像的构成原理，把数字图像分为三层。其次，我们把分开的图像，根据我们的要求，分别进行处理。最后，我们再把处理完的图像再合并，重新组成一幅新的图片。以上所有的步骤都是基于时域和频域的各种数学算法的基础上来实现的。我们做数字图像处理软件的宗旨是尽可能的提取图像中某些对我们有用的信息，扩大图像种不同物体特征之间的差别，改善图像的视觉效果，并对这些特定信息进行增强，而不是像一些图像编辑器一样对图片进行修改和渲染，这完全是两种概念。
