视频图像识别分析软件，掀开监控新篇章

视频图像识别分析软件（IVS，Intelligent Video software），许多人也称之为智能视频软件，智能视频（IV，IntelligentVideo）源自计算机视觉（CV，ComputerVision）技术。计算机视觉技术是人工智能（AI，ArtificialIntelligent）研究的分支之一，它能够在图像及图像描述之间建立映射关系，从而使计算能够通过数字图像处理和分析来理解视频画面中的内容。而视频监控中所提到的智能视频技术主要指的是：“自动的分析和抽取视频源中的关键信息。”如果把摄像机看作人的眼睛，而智能视频系统或设备则可以看作人的大脑。智能视频技术借助计算机强大的数据处理功能，对视频画面中的海量数据进行高速分析，过滤掉用户不关心的信息，仅仅为监控者提供有用的关键信息。

    近期，一份来自IMS Research调查机构的研究报告预测，下一个五年内，视频内容分析市场将空前繁荣，从2004年的677万美元急增至2009年的8392万美元，涨幅达65.5%，并且，许多开发商对其前景相当乐观。北京兆维泰奇科技有限公司的技术总监沈宇枫认为，虽然目前IVS正处于商业化及市场应用的初始阶段，但是其功能已经逐渐为大家所了解，预计近几年市场规模还会继续不断扩大。

    汉王科技股份有限公司移动视觉事业部的总经理童剑军也认为，目前IVS已经成为监控领域的一个热点。过去的五年里，大概总共才只有10家IVS研发公司，而如今，厂家的数量不止10倍地增长，对于产品的研发方面，也在积极进行中。

产品优势明显 适应目前需求 
    在目前日益复杂的视频监控系统的状况下，IVS不仅要保持日常管理操作的简单和便捷，还要考虑系统覆盖范围越来越广的特点以及兼顾系统的经济、有效性。最后，它还必须将监控系统与门禁控制系统、报警系统融合到同一个平台，允许集成其他相关技术，同时保持系统的灵活性和可扩展性。随着目前监控行业的迅猛发展，IVS产品目前所具备的优势已可以满足当前的需求，主要为以下几点。

操作方便 
    随着目前技术的不断进步，监控系统也演变得更加复杂。比如在平安城市监控这样复杂且大型的工程中，单靠人工操作是一件很困难的事，成千上万个监控点，无法同时在显示屏或电视墙上监视。即使是分级管理，如此大量的观测点，也很难及时发现案情。所以，传统的以人力为主的解决方案面临着必须解决超大视频数据容量的现实问题。

    而IVS的使命是管理好复杂的系统,同时方便工作人员进行操作。即便系统包含许多其他的系统和部件,如门禁控制等,IVS也应该是操作员的唯一工具。“用户希望系统的控制界面简单，就像操作家用电脑一样方便。”全球领先的市场研究机构IMS Research的安全及ID调研室主任Simon Harris说道。

    尽管优秀的DVR也拥有在大量数据中搜索某个事件是智能分析功能,但这种功能只有操作员坐在DVR旁边操作才有效。而IVS则可以在几秒之内,回放由NVR记录的某个城市24小时的监控录像中的一个片段，并当定义事故发生后,操作员可以点击不同的小片段进行回放,且回放质量与实况播放的相差无几。由此,明显减少了大型企业的安防系统开支。

集成、兼容性强 
    由于监控系统是一个容易与其他应用系统,如与报警和门禁控制集成的开放式系统,便于其他部门与IT部门之间互为通信的系统。因此，IVS要求集成性非常强。

    近来,针对安防设备的客户调查显示,可集成性几乎总是排在优选次序的首位。几乎每一家重量级的安防制造商都研发了相应的网络化设备,包括门禁控制、入侵探测、零售点信息传输、视频监控和存储子系统等。这些设备基于网络且采用开放式通讯协议,可以方便地集成来自不同厂家的其他设备。

    GE Security 视频软件方案部的总经理Robert Siegel认为，目前终端用户需要的是可集成的系统解决方案，而不是孤立的零部件或系统。随着目前模拟安防系统向IP视频监控系统过渡，那些站在前沿的安防主管都希望将模拟系统转变为真正的数字化系统。“然而，他们在要求满足急剧增长的安全需求的同时，预算还要跟以往持平。出于这点考虑，新的系统必须能够兼容原有的模拟产品。”Siegel补充。

开放、灵活性高 
    “第一代的视频管理系统大多是针对中小规模的应用设计的，基本上前端少于16部摄像机。而现在，IVS需要越来越多地应用到有几百部摄像机的大型系统中。”Siegel表示。

    对于那些设计影像处理和控制电路板或IVS应用的系统开发人员而言，拥有丰富外设功能的处理器能够解决许多集成问题。不过，同时还需要在开发套件中捆绑业界领先的应用软件和开发环境等。

    对于大型项目而言，系统的灵活性和可扩展性更为重要，数字式录影记录先驱的Dallemier公司项目应用开发总监Konrad Hechtbauer指出，用户的系统需求是没有止尽的，所以软件也应该是开放、灵活的，只有如此，系统才能顺利地添加新的摄像机和录像机，并可为第三方的产品提供多个接口。

用户采购标准 
    在了解IVS的以上优势后，用户在对产品进行采购时，可以从以下几点进行考量。

需要解决什么问题 
    这一点首当其冲。每一套IVS软件都各具优点且并非万能，用户应当针对自身的需要作出选择。“视频分析运算法则在不同的监控环境下有不同的应用。必先沟通，再确定用户需求和期望是什么，”加拿大智能视频软件公司Aimetis的 CEO Marc Holtenhoff强调。“如果在浓雾、大雪或暴雨等恶劣气候条件下使用目标跟踪功能，”以色列的多媒体录音公司Nice Systems的产品销售经理Eyal Zohar补充，“我们必须先采用过滤功能及其他程序来排除误报。但是，这些完全取决于客户的具体需求。”

    主要做智能化影像分析产品的iOminiscient公司营销渠道顾问Mathew Bertram也认为销售产品时沟通相当重要，他建议销售人员应该反客为主、先发制人地发现用户的需求。“这要求厂商完全明了用户的要求，并告知用户他们期待的功能中哪些是可以实现的，避免用户因不了解软件的实际能力而对厂商期望过高。”

注意摄像机的安装和配置 
    优质的图像是视频分析软件的基础单元。Bertram解释，“如果分析软件是整个监控系统的大脑，摄像机便是眼睛，连眼睛都无法看清东西的话，那么大脑更难以对所发生的事件进行精确分析。”为了确保画质，电子动力专业服务公司IntelliVision的市场副总监Sean Shankar建议使用一些基本的产品或技术来改善图像或视频质量。

    一旦画质有了保证，接下来要考虑的便是摄像机的摆放位置，如高度和角度，其次还应该考虑一些客观环境因素，包括室内或室外、光照条件、温度等。如果摄像机配置正确的话，是不受光线和气候变化影响的。在设置和调整参数时，也没有一成不变的规则，因为参数取决于运算法则、用户需求以及上述客观环境因素。“就人数统计功能而言，”Shankar说道， “摄像机可以装在天花板上，这样便于统计时不遗漏任何一个人。而要很好地进行面部识别的话，摄像机应装在墙壁上，如此定位才能提供优质的图像。有时候，用户希望摄像机固定而不喜欢改变位置或角度，但这样摄取的视频图像质量低下，从而导致IVS系统的识别精度不高。”

用户界面设计与硬件承受能力 
    友好的用户界面增加了系统价值，因为界面设计直接影响到用户熟悉系统输入和所花费的时间。所有功能都必须通过用户界面上的图标、按键及图形直观地展现出来，这样，即便没有专业IT技术的安装人员和最终用户都能轻松地操作，反之，如果界面设计得不合理、不科学或违背直觉的话，人机互动将变得相当困难。

    硬件问题也举足轻重。假如用户需要进行大量的视频运算和分析，计算机系统必须具备强大的承受能力。随着IVS软件的进步，硬件如CPU也应当不断升级。“有时，分析过程对计算机的能力要求甚高，”主要生产数位监控系统的Huperlab公司的销售经理Jamie Wu解释，“用户不情愿再去购买高价位的计算机，但是基于现有的硬件系统运行程序将难以达到预期效果。举例来说，一个具有16路视频通道的系统就需要有较高计算能力的计算机。这也是为何分析技术得不到大规模商业化的原因之一，即大众市场既有硬件系统，如CPU无法支持高端复杂的分析。”

集成方案更受欢迎 
    整个安全行业朝着集成化方向发展，分析软件也同样如此。分析法并非智能化解决方案的全部，其第二要素是数据管理。Wu认为，“用户应选择一套可将多个子系统集成到同一管理平台的整体解决方案，该方案由同一供应商提供，并且集成了图像捕捉、处理、分析，以及数据储存或远程播放的功能。然后，这些功能基于硬件可以正常工作。最后，如果系统需要增加新硬件，该平台应该具备足够的兼容性和开放性。”

    “视频分析解决方案应该基于开放式标准，并易于与其它安全防范系统集成，”提供国际领先的分析、网络化视频安全监控软件分析解决方案的Verint System公司的全球销售副总监Mariann McDonagh继续说，“对来自不同的报警系统、传感器或其他监控系统的数据能够快速响应，一旦系统接收到报警信息，预设的安全程序将同步启动，如锁门、升高障碍栏、调整摄像机角度或调节视频帧率和画质等级。

    集成方案使得系统维护更加轻松，那种苦苦寻觅与软件匹配的视频捕捉卡的抱怨也会随之减少。“基于计算机的集成化视频分析解决方案更受欢迎，因为硬件可以相互兼容和灵活配置，”智慧影像应用的先驱IoImage公司的市场副总监Dvir Doron表示，“假设从两个不同的供应商处购买解决方案之后，遇到问题的话，你该找谁解决呢？是IVS厂商？还是将分析功能嵌入到产品中的编码器厂商？为了避免这些维护问题，用户最好从同一供应商处购买解决方案。”

必须测试产品 
    在选择供应商之前查看其简介、财政是否稳定，并调查该供应商的历史记录，以及第一次发布产品的时间。通常情况下，发布时间更早的供应商有更多的用户使用过或使用着其产品。有人甚至推测，产品拥有越多的用户则意味着该产品更可靠。此外，还得考察该供应商在垂直市场是否有可作参考的用户。

    最后，零距离感受IVS产品功能特性的最好方法是实地测试。“我们鼓励用户在购买前进行测试。和其它PC产品一样，我们公司的网站上有免费试用的软件，”Holtenhoff说道。

    “用户也可以从将视频剪辑提供给厂商，请厂商运行软件并验证结果，”Shankar建议。致力提供安全解决方案及服务的Lenel Systems公司的首席视频图像科学家Arie Pikaz则建议，测试时至少让系统运行24小时，在已知时间内测试某些功能，以便估算出误报率，同时用户也应该检查报警记录。”

国外产品进驻中国市场 国产品仍具备发展空间 
    据笔者采访了解，目前在国内市场上使用较多的IVS产品以国际品牌居多，为Ov、Dallmeier、Vidient、Ivbox、Mate、Nice以及Viseowave等，并且这些产品都在质量、功能等方面具备一定优势。例如，DALLMEIRE就给产品系列提供相应的管理软件，满足在质量、可靠性和用户友好性方面最苛刻的要求，该软件模块可以与Dallmeier的硬盘录像机、视频运动探测器和摄像机兼容，共同构成完全智能化的系统，并且确保所有独立部件都可以按用户喜好简单地配置，还可以对一个或多个录像机的视频图像进行控制、分析，从而为操作员提供了一套多功能、人性化的解决方案。

    此外，在06年9月A&S举办的GDSF全球数字监控论坛上，国际著名厂商Milertone开始进入人们的视野，此外，为2004年雅典奥运会提供网络视频安防系统的Videobridge公司2005年也在中国建立了分支机构。从视频管理软件本身的技术水平来看，这些国际一流的厂商要比国内厂商的软件水平高出一截，而且相当成熟。

    与这些国外品牌相比，国内的IVS市场还有很大的差距，目前基本上还处于起步状态。不过对于这些情况，中国IVS厂商们仍然对未来市场的发展信心十足。蓝色之星网络监控部经理陈岩峰认为，从目前国内的市场上看，用户对网络视频监控的应用还在摸索阶段，定制开发很多，国外厂商在建立一个相当规模的本地开发队伍之前很难适应中国的市场，所以从现在的情况看，留给国内IVS的发展空间还是比较大的。

平安城市中的新星 
    从美国的9.11到美国的爆炸事件，都给世界敲响了警钟：平安城市安防监控的真正意义是什么？仅仅是事后取证，还是事前预警？无数事实证明，事后追查是必须的，但事前的预警却更为重要，它可以把一些重大事件隐患抑制于萌芽之中。因此，城市安防监控必须能事前预警，“防患于未然”才是最有效的措施。所以，IVS作为监控系统智能化的得力助手，将担起重要角色。

    兆维泰奇的技术总监沈宇枫认为，“平安城市”是一项规模浩大的工程，在部分大城市的规划中摄像头使用数量达到了20万以上。然而，如此大量的摄像头，纵然采取了分级管理的模式，人们还是无法依靠单纯的监看来完成如此繁杂的任务，及时发现制止犯罪更沦为空谈，事后取证几乎变成了大规模视频监控系统的唯一功能。在“平安城市”工程完成大规模的布点之后，这个问题将变得更加突出，而使用IVS则会让“平安城市”工程能够真正预防犯罪事件，避免事后取证，所以在“平安城市”建设中，其商机更多。

    先进视讯的总经理助理吕世光也认为，要具备这种事前预警的功能，安防监控系统必须要实现让计算机代替人去监看视频图像。通过运用IVS，来提高监控系统的预警能力。可以看出，IVS在事前预警方面将是监控技术发展的方向。

未来 
    新产品从开发到发展需要多年时间。纵览智能化的历史，IVS朝着IT网络视频监视系统方向发展的趋势日益明朗。如同其它许许多多的技术，未来IVS有望与其他技术合并为一。“兼具自动监测和识别两种功能的产品便是一个极好的例子，系统可以自动监测某些行为，例如逗留或违章泊车，镜头将自动地移近人或车辆，以便进行面部识别或车牌识别。”Bertram举例道。

    不过，IVS技术不是万能的，它只是人的感官的延伸，可以代替人处理所不擅长的事，但前提条件是，这种技术必须得到正确的应用，并且其具体功能仍然取决于操作者。要想IVS的功效最大化，用户必须根据自身的需要选择合适的解决方案，遵循各种标准协议，但同时，又不能完全依赖IVS，因为它仅仅是一种服务工具，真正的决策者、行动者是用户自己。归根结底，人类才是智能化的创造者，惟独人才能主导整个监控系统。
