40+最佳GIMP插件

GIMP无可置疑的是Linux平台下图像处理的王者,并且在人气上仅次于Windows 和 Mac平台下的Photoshop。它拥有一个巨大的开发者社区和甚至更大量的使用者，GIMP很流行就并不奇怪了。

就像Firefox一样，GIMP的强处在于它的插件，这些插件都是开源社区带动开发的。因为插件是用户自己开发的，用户了解所有的需要并且构思好了插件的一切（当然，除了为你准备一杯咖啡;) ）。

那还有什么方式比一系列好插件更能让我认可GIMP呢？它们大多数是针对网页设计师和摄影师的，这是GIMP的主要用户。还有其他一些是修复GIMP的普通问题的。

网页设计

  • Snow： 产生逼真的3D雪花。注意：需要大量的内存和交换分区。
  • Anti-aliasing： 平滑和固定直线。
  • Adaptive contrast enhancement： 智能调整对比度。
  • Fourier Transformation： 傅里叶变换的GIMP插件版本。
  • Refract/Reflect：仿真滴落水滴的折射效应。
  • ShapeIt:：根据映像层信息弯曲文字或图像。
  • Borders： 给图像添加边框。
  • SoapBubble：产生泡泡。包含针对GIMP v2.2 和 v2.4的两个版本。
  • Aqua Bou：轻松的制作‘Aqua Bou’风格的网页按钮。
  • Chrominium： 产生带光辉和火花的铬合金logo。
  • Cloud Generation： 轻松制作云彩。更新后可适用于最新版本GIMP 2.4。
  • Gothic Glow： GIMP版本的哥特辉光效果，源自Photoshop。
  • Liquid Rescale：智能图片剪裁算法。当沿单方向剪裁图像时保留图像特性。 
  • Particle Deposition： 在图像表面仿真粒子沉积效果。它产生随机大小和速度的小粒子，让它们落到图像表面上，使图像产生表面高度小小渐增的效果。
  • Stampify：使你的图片看起来像一张邮票。
  • Plasma2： GIMP原来等离子插件的新版本。
  • SaveForWeb：快速在最小文件尺寸和可接受图像质量之间找到最优折衷。对设计师很有用。
  • Steel Text： 创建带阴影的闪亮钢铁（或黄金）文字效果。

摄影

  • Pandora：组合一组图片生成一幅全景图。 
  • Film Grain：添加经典的电影纹理感觉，使图像成为黑白电影片风格。
  • Black Ink：把图像转换成墨水画的黑白电影风格。
  • Copyright Text：为图像添加版权信息。
  • DeNoise： 使用高斯模糊算法的降噪插件。
  • DustCleaner： 检测并移除数码图像上的尘点。
  • Astro-plugins：天文图象处理的一组插件。
  • FocusBlur：通过亮度和深度使图像失焦。
  • Lomo： 使照片清晰、锐利和简洁。对户外照片尤其有用。
  • ISO Noise Reduction：实现ISO降噪功能，使用不同的方法（ 屏蔽边界，然后单独模糊单色通道或亮度通道）。
  • Photo Effects：打包到一个script-fu文件里的一串脚本。包括卡通、彩色铅笔、彩色粉笔、切块工具、制图、墨水笔、笔记本、油漆点像、调色刀、彩色蜡笔、图章、水性漆效果、 包装效果、成角的线条、交叉排线阴影、铬图像、交叉光线、搞笑色、软聚焦、过度曝光、砖墙、拼缀图、彩色玻璃效果、组织成型、高通、卷轴效果。
  • RedEye：快速消除照相机闪光产生的红眼效应。
  • ShadowRecovery：恢复图像阴暗区域的信息。
  • Shadows/Highlights：消除照片上的阴影和高光。
  • Refocus： FocusBlur(上列)的反作用, 此功能重新聚焦图像。提供比GIMP内置的锐化掩模更好的效果。
  • PhotoToolbox：立即在图片上执行散焦、降低饱和度（多层仿真）、 调色、增加纹理等功能。
  • Light Filter：通过调整每个像素的色彩层次提供标准滤光片功能。
  • Gradient Along Path： 对路径应用梯度正交产生矢量图。
  • Landscape painter：在风景画上产生看起来不错的有趣油漆效果。
  • EZ Improviser：改善阴暗、色深的图像。
  • Edgy：使图像看起来锐利。
  • Digital IR：仿真黑白的红外图像。
  • Colour saturation:：改变图像的色饱和度。
  • Blot： GIMP污点刷的具体化。基于污点算法，可以很好的处理溅污或有云纹的图像。

其他

  • GIMP Dewierdifyer：（Windows专用） 摆脱GIMP普通窗口的烦恼。它在GIMP之下放置一个背景窗口。把其他GIMP窗口结合为一组，那么当你按ALT+TAB键切换到其他程序时，GIMP符号只出现一次。
  • Turing Pattern：呈现一种图灵模式。
  • Yin-Yang+：一个表面看起来没有用的插件，但是非常有趣。添加阴、阳符号。也会随机的添加阴影。

更多资源

  1. Writing a GIMP plugin：PDF, HTML, 存档文件 和 XML格式的插件指导。
  2. GIMP plugin template：一个有基本结构的空白插件模板，可以大大减少开发一个插件的时间。直接链接在这里 ，但是以后可能会改变（如果它改变了，直接点击“GIMP plugin template”）。
  3. Writing plugins：编写插件的另一个指导，它在GIMP.org开发者分站上。
  4. Writing Plugin：GIMP.org 主站上的另一个指导。
