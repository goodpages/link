投资银行Digi-Capital近日发布了一份最新报告，称至2020年，全球增强现实(AR)与虚拟现实(VR)市场规模将达到1500亿美元。其中AR市场规模为1200亿美元，VR市场规模则仅300亿美元。

谷歌公司推出谷歌眼镜(Google Glass)、Facebook收购Oculus，微软推出HoloLens都是AR与VR技术迅速增长的强力信号，但梅内尔相信，在AR与VR市场，真正具有变革意义的大事件尚未发生。

“2015年的VR/AR，给我的感觉有点像是iPhone问世前的智能手机市场。我们期待有公司推出真正能够改变行业的产品。”

马上手机要上4K屏了，最受益的要数VR和AR了，三星的GEAR2代据说今年年底上市，好像也不贵的样子

这样的硬件现在好多家在做，我常见的就有30几款了，现在都在等谁家的硬件平台稳定出货量大，

然后CP都可以去那个平台开发相关内容了，同事买了个Oculus 放公司给我们玩~··还有人买了99元的暴风影音眼镜

楼主体验了一下，觉得非常有意思！


接下来是楼主对未来市场VR和AR产品方向的一些探索总结报告：

1、儿童涂鸦墙---可记录的电光笔，麻麻再也不用担心宝宝把墙壁弄脏了，还可以自己和自己玩得开心


2、远程修水管辅导---即便男友出差了，家里的水管灯泡坏了也可以远程辅导了


3、反恐演习---实弹演习好危险，警察叔叔可以用虚拟实境来演练如何制服歹徒了


4、虚拟试穿试戴---脱衣穿衣好麻烦现在一秒搞定了


5、家具试摆---买来的家具不晓得放客厅合适不沙发颜色配不配呢先摆摆看鲁，看房不用出门



6、医疗建模---脏器哪里出问题了扫描建模后全息研究研究鲁


7、带介绍的广告单和入场券---电影票看预告片，还有各种立体产品说明书


8、显示增强地图导航---还可以用在旅游电子导游上，这下爱逛哪里就哪里了


9、交通导航增强---除了用在车子GPS上，以后逛街找厕所也很方便鲁，步行街广告还可以用搜索功能的鲁


10、看IMAX电影和大屏幕上网都是小CASE啦，最主要可以打虚拟实境游戏了互动感棒棒哒


11、立体百科全书---以后小朋友看书恐龙都会走来走去啦，哪里不会指哪里，黑板上老师的中文都自动翻译英语双语教学的啦


12、约妹几吃饭，立马就能看到对方愿不愿意和你交朋友或者一些其它联系方式蛮适合大龄青年找对象的


13、电视节目数字舞台---各种摄影增强，以后可以和虚拟偶像一起互动啦宅男福利哦



14、以后户外广告也会变得很奇葩和立体哦


15、工业设计---随着工信部工业4.0的出台，以后虚空中设计好，3D打印的个性产品会越来越多


16、虚拟宠物和厨房提醒饼干不会烤糊了，外加超市食物卡路里提醒，钥匙信息


17、绘画辅助---帮助像楼主这样的重度手残及懒癌晚期患者重拾绘画自信


18、立体游戏棋、角度测算、摄影增强、VR AR云存储、脑波操控、立体照片带回忆可以见已故亲人

照片增强，视角代入（球员身上的摄像头，让直接第一视角看球），增强现实博物馆，盲人导航，虚拟军事指挥系统……
