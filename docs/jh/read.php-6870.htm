你觉得“下嫁”那一刻，婚姻已走向死亡

孔雀女为何钟爱凤凰男

见过许多家境不错，单纯善良的女孩子在成人后择偶的时候，偏偏钟爱与自己成长环境家庭背景完全迥异的“凤凰男”。询问下来也不难理解，“孔雀女”见多了和自己一样，从小娇生惯养备受呵护长大的男孩子，他们温室长大，不曾经历苦难风雨，不懂生活的疾苦，任性自我不谙社会生存之道；而“凤凰男”则恰恰相反，生在中国的农村，早早锻炼了他们通达人情世故的能力，在从农村走向城市一路艰难的求学与打拼中，无一不练就了成熟稳重、做事心思缜密深谋远虑并且忍耐坚韧的性格，贫寒的家境激发了他们比城市男孩子更强的上进心与事业心，加上朴实无华的言语，健硕的体魄，怎么能不深深地吸引生活成长环境单纯、从未曾为生活忧虑、一路只是在象牙塔里读书、对生活充满幻想的“孔雀女”呢？

况且只要入的了“孔雀女”眼目的“凤凰男”，多是青年创业小有成就，姑娘们深信他们有着宏伟的目标大好的未来；要么就是名校毕业在事业、科研、机关单位或企业中年轻有为，这两者也都在一定程度上满足了“孔雀女”的虚荣心和好胜心。以自己这样“优越”的条件，算是“下嫁”给了你这穷小子，想必定会终生感激，对自己好一辈子的吧。

因此“孔雀女”们常常不顾家人的反对，坚贞如王宝钏一般，不惜为此与父母断绝关系，也要坚定地与“凤凰男”在一起，并深信他们的爱情最真挚，最纯粹；在父母亲友都不看好的境况下，更加决意要以婚后生活的美满，男人最终事业经济上的成功来向众人证明自己眼光的长远和选择的正确。

理想中的“凤凰”婚姻，现实中的“麻雀”境况

有言道：男女之间常常因为彼此的差异而相互吸引，然而婚姻却往往是因为相同才走得更加长远。当“孔雀女”们冒着众叛亲离的风险坚定地走进自己设想的、美好的婚姻城堡时，你可曾想到过，你以为你以为的真的就是你以为的那般美好吗？理想中的“凤凰”婚姻，最后在现实中却常常败走麦城，遭遇着“麻雀”般的现实境况。来听听“孔雀女”们的婚后自述吧！

安打来电话哭诉：“结婚的时候，我没有要他家出一分钱彩礼钱，北京的房价高我体贴他家境贫寒事业刚刚起步，说服自己父母出资买房，还将他父母兄弟接来新房居住，我们自己租房在外面，这么多年，我哪一点对不起他？我只不过希望他能将公司做好，将来证明给我父母看，我的选择是对的。我对他这样好，可在他心里，我永远要排在他的事业哥们兄弟父母后面，他怎么就不能顾忌一下我的感受，与我多沟通，安慰安慰我，陪我喝杯咖啡吃个下午茶，出差的时候偶尔也帮我带个名牌的包包呢？我这样的要求难道很过分吗？我的朋友们都是这样生活的呀，这在我的朋友中间都是再平常不过的事情，怎么到他这里就这么难？！我做了那么多牺牲，这么一点要求他怎么就不肯妥协一下呢？！我顶着家庭的压力下嫁给了他，而现在先开口提离婚的居然是他！”

乔在诉说时欲哭无泪：“他的家里三天两头来人，七大姨八大姑轮流来，今天弟弟找工作要我托人帮忙，明天父母来城里看病也要我托人找关系，可是我自己看病也要排队呀！这么多年，从来没有陪我出国旅游过没有给我买过一件像样的首饰，更可怕的是每年过年回公婆家没有暖气空调，不能洗热水澡，厕所在室外冻的屁股都生疼。可是他一点也不心疼我，还觉得我好娇气，他不知道我为他做了多大的牺牲嘛！恋爱时觉得他那么奋进也讲究生活质量和情趣，可是现在发现他回到自己的村庄居然依然喜欢和众人聚在一起喝酒抽烟打牌，烟雾缭绕中的他对我来说特别不真实，他不是应该和村里的其他人不一样的吗？可是那一刻的他是如此放松真实，这才是真正的他真实的他，也许在城市中打拼的他戴了太多的面具，我突然觉得一起生活了多年的他是如此陌生而遥远。最近，他对我说，我们是两个世界的人，他永远融入不了我的世界，他外面，已经有别的女人了……”

反哺，凤凰男无法逃避的宿命

我们常说，恋爱是上半身的，彼此气质样貌谈吐的吸引让人浮想联翩，对生活充满假设，而婚姻是下半身的，彼此最隐私最丑陋的部位相互袒露，为生活养家不停地奔波；褪去光鲜的外表和对生活的想象，原来婚姻是如此真实，想象中的他又是如此不同。听过了许多“孔雀女”们的控诉，内容都是相似的，而静下来思考，这些相似问题的表象背后又是什么呢？

首先，作为独生子女，养尊处优长大的你，也许习惯了一切以自我为中心的生活。恋爱时的你也许不曾想过，身旁这位令你都骄傲万分的优秀男人，更是他父母的骄傲，是他整个家乡的骄傲，在他的身后有着庞大的家族成员和支持者，他，绝不只属于你一个人，你们的小家庭，也绝不会是从此只有你和他的幸福生活。农村和城市的差距在当下依然巨大，而农耕社会的一些狭隘之处就在于，很多人把养孩子当作了投资，因为缺乏资源，靠着父母及姐妹兄弟的牺牲和血泪成长起来的凤凰男，承载了太多人的太多期望。反哺，是凤凰男无法逃避的宿命。因此，嫁给了“凤凰男”也基本意味着你嫁给了他的全家，至少要接受他的全家，他成长背后的所有人。不同于城里人的“关起门来过日子”，“养儿防老”、“一人得道，全家沾光”这样一荣俱荣的思想今天依然存在于大多数的农村家庭中，你的丈夫现在有了出息来到城里，如果不帮三表舅找个看门的工作，二堂弟租个房子，四婶子进城在家住几天，那不是“忘恩负义”“没有良心”吗？！

孔雀女，你真的长大成人了吗

其次，“孔雀女”也要问问自己是否真的长大“成人”。也许刚刚步入婚姻的时候，他满足了你对未来的想象，你满足了他从小的愿望，看上去一切都是美好的。然而再优秀的“凤凰男”在二十几岁的年纪，凭着贫寒的家境和刚刚毕业几年的积蓄，去面对当今城市的房价时，依然是相形见拙的。当“孔雀女”们一旦接受了自己父母的出资帮助，而不是那个心中“优秀”的他为自己购买的住房，面对家徒四壁的公婆家时，请问又有几个可以真正心里不会产生落差感，不会觉得自己的“下嫁”将来应该得到点补偿？不会对自己的家庭付出产生优越感？又对看似“一无所得”的“裸婚”觉得有点委屈？一旦这样的感受产生，必然对男方开始有各种要求和期望，开始产生“补偿”心理，而男人往往恰恰不喜欢被要求和改变，喜欢反其道而行之，于是双方的“拉锯战”正式开始。

在这里，也想请“孔雀女”们可以换角度想一下，自己对丈夫的“优越感”真的来自于自己能力的体现吗，还是只是建立在父母为自己打造的经济基础上“俯视”丈夫呢？如果丈夫与原生家庭成员们之间的各种纠缠不清，各种“边界不清”让你愤怒的话，请问在生活与经济能力上，你是否也依旧依赖自己的父母呢，当你接受父母为自己出资建立家庭的时候，是否也是“边界不清”的另一种表现形式呢？对丈夫的“期待”背后其实是另一种“依赖”的表现形式，从小习惯了“依赖”父母的“孔雀女”，如今开始寄希望于继续”依赖“丈夫了。所以，“孔雀女”们应察省婚后的自己是否内在还依然是个小女孩儿。找到内在的那个成年人的自己，学会为自己的生活来承担责任，才是幸福的开始。

“凤凰男“既然被称为鸟中之凰，自然是难以驾驭的，在从农村走向城市的过程中，他们难免会有自卑，但由自卑产生的自信往往膨胀为过度自我，有的自认为高人一等且脾气古怪，难以沟通。在资源稀缺的环境下长大的他们，对资源的利用也有着天生的敏感，娶一位城里的富家小姐也许是他们从小的梦想和证明自己的方式，能拥有今天这样的生活是自己能力的体现和应得的一切，而“孔雀女”因为自己的“下嫁”要给他们戴上一顶“永世不得翻身”的帽子，他们怎么会干？！网上著名的凤凰男征婚帖子也说明了该现象：“征婚男30岁，研究生毕业，月入八千，要求高学历收入相当的当地女孩，合伙买房子必须加上自己父母名字，如果对方已买房子他出家电，加上父母名字，必须与他父母同住，大姐会经常来住；女方家境要好，不能有兄弟姐妹，要保证生儿子”……这则征婚，正是凤凰男用最真实的心声告诉“孔雀女”：——“不要觉得你女方的家境好，买了房子我们住了这是你的莫大牺牲，我们一生都该感恩戴德，在我们眼里，找了我这么优秀的男人才是你们一家的福气，这些根本不叫牺牲，所以也不要指望婚后对我增加要求和期待。”——看到了吧，婚前的你以为下嫁了“凤凰男”，他会捧着你一辈子，你在他的“上面”，所以可以“控制”他一辈子，没有想到，婚后的他完全“不可控”，而且你也一点都“不受宠”。所以不要觉得条件如此优越的你“下嫁”“牺牲”，最后反而被先抛弃是件特别不可思议的事情，无论霍小玉，赵五娘还是王宝钏，也都曾经折在凤凰男的手里，寒窑18年丈夫考验羞辱你一番还能给你一个名分，已经算阿弥陀佛皆大欢喜了，连对男性宽容到令人发指的古代都对凤凰男的行径出离愤怒，所以你的痛苦真的是“前有古人，后有来者”，不必如此心伤，怪只怪选择时没有好好了解一下“凤凰男”这个物种呀。

最后，生活习惯价值观念的不同，消费观金钱观的差异，在婚后统统暴露出来。受农村中男性主导地位与本位主义的影响，使凤凰男在婚后家庭位序的排列中，妻子永远难以排在首位，这更是令“孔雀女”们委屈到极致。丈夫的“冷落”也许是“无意”为之，可是“孔雀女”们是否意识到，你的很多“无意”为之也深深伤害了丈夫呢？比如，丈夫在娶你之后辛辛苦苦费劲气力打拼的，在他眼里弥足珍贵的生活，满以为会令你欢欣赞赏的一切，在你这里却是稀松平常，你无意中平淡而不屑的态度是否也曾深深刺伤了他的心，让他觉得无论如何努力都难以追上你的脚步让你满意？你又可曾想到，也许在他少年的成长中，一枚鸡蛋一块糕点都是奢侈的，他所有的梦想就是读书出人头地，他成长的土壤注定了生活的“无情趣”，你让这样长大的他怎么具备像你一样从小衣食无忧的“小资情怀”呢？怎样去喜欢陪你喝咖啡吃下午茶，逛街就喜欢买哭泣和驴包给你呢？你可曾想到，因为出身的不同，决定了在他多年的奋斗中要历经比别人更多自卑与自尊的考验，剥开他们在人前自信强大的外衣，里面其实是一颗脆弱冰冷甚至自卑的心？而这颗真实的内心却只有你能看到，只有你去温暖，我知道这对于从小没有受过那么多磨练的你也很难，但是没有办法，婚姻就是这样让我们从小女孩儿变成强大温暖的女人，让我们成熟长大，带上了“妻子”的称号，就注定要有更多的承担。

所以，小资的“孔雀女”们也许永远理解不了，一份平淡温馨的小日子，在与“凤凰男”的婚姻中怎么就变得如此之难，为什么事业、父母、兄弟的情谊永远比你更重要，为什么你的小资情怀在“凤凰男”这里永远难以满足。当然，这不是你的错，更不是他的错。是因为，他所经历的，是你不曾经历的，你理解不了从小出身贫寒受中国传统价值观终于“鱼跃龙门”成功的他，正如他理解不了从小吃着蛋糕汉堡读着安徒生童话对生活充满幻想的你，因为，这就是老人们常讲的，为什么婚姻要“门当户对”。（未完待续，明天推送《凤凰男与孔雀女的婚姻，如何破局》）
