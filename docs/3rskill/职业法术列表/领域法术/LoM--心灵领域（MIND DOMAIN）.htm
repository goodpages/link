心灵领域（MIND DOMAIN）

神授力量（Granted Power）：你在唬骗，交涉和察言观色检定上获得+2加值。

心灵领域法术（Mind Domain Spells）

1通晓语言（Comprehend Languages）：懂得所有说出和写出的语言。

2侦测思想（Detect Thoughts）：施法者可以“聆听”到他人的表面想法。

3 次级心灵连线（Lesser Telepathic Bond）：与受术者在30尺内获得心灵交流，持续10分钟/每级。

4辨识谎言（Discern Lies）：识破蓄意谎言。

5心灵连线（Telepathic Bond）：一条心灵连线可让盟友互相交流。

6 思想探针（Probe Thoughts）：阅读受术者的记忆，每轮一个问题。

7脑蛛术（BrainSpider）：窃听最多其他8个生物的思想。

8心灵屏障（Mind Blank）：受术者对精神性/情绪性魔法和探知免疫。

9怪影杀手（Weird）：与魅影杀手相同，但是影响30尺范围内的所有目标。
